# agent-runtime/outputs — финальные результаты

Сюда попадают готовые результаты работы агентов: отчёты, стратегии, планы, документы.

## Форматы

- `.md` — стандартные документы
- `.html` — готовые страницы
- `.pdf` — PDF-отчёты (если генерируем)

## Соглашение по именам

```
<дата>-<тема>-<клиент или «inna»>.md
```

Примеры:
- `2026-04-29-competitor-analysis-inna.md`
- `2026-04-29-vk-strategy-inna.md`
- `2026-04-29-content-plan-may-inna.md`

## Важно

Эта папка в `.gitignore` — результаты НЕ попадают в git автоматически.
Если результат нужно сохранить постоянно → скопируй в `inna/`, `knowledge/` или `clients/`.

## Фаза 3 — Создание скиллов для агентов

### Алгоритм для каждого скилла

1. Читаю `agents/имя.md` (все уже в каноническом формате ✅)
2. Определяю 3 триггер-сценария ("что скажет Инна чтобы это включилось?")
3. Определяю 3 анти-триггера ("что похоже, но это НЕ этот агент?")
4. Пишу `.claude/skills/имя.md`:
   - `frontmatter`: name + description (триггеры + анти-триггеры, без шагов процесса)
   - `body`: протокол работы + KPI + рекомендуемая модель
5. VALIDATE через skill-conductor (11 чекпойнтов + trigger test 6/6)
6. Правлю если нужно → повторный VALIDATE

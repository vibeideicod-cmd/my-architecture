## Прототипы (из клиента: Детский санаторий Южнобережный)

### MINI-1: Интерактивный чеклист
- **Пример:** "Собери рюкзак" — родители отмечают что взять ребёнку в лагерь
- **Шаблон:** чеклист-подготовка для любой услуги или события
- **Кому ещё:** детские лагеря, туристические базы, курсы, медцентры

### MINI-2: Квиз-квалификатор
- **Пример:** "Готов ли ребёнок к смене?" — проверка перед покупкой путёвки
- **Шаблон:** квиз для квалификации лида, отвечает на вопрос "подходит ли мне это"
- **Кому ещё:** любой сервис с подбором: клиники, курсы, фитнес, консультации

### MINI-3: Виртуальный тур / онбординг
- **Пример:** "Экскурсия по ЮБ" — знакомство с территорией санатория
- **Шаблон:** интерактивное знакомство с местом или продуктом до покупки
- **Кому ещё:** отели, клиники, рестораны, шоурумы, офисы

### MINI-4: Анимация и оживление сайта
- **Пример:** "Оживить сайт ЮБ" — добавить движение, интерактивность, живость
- **Шаблон:** модуль анимации для существующего статичного сайта
- **Кому ещё:** любой оффлайн-бизнес с устаревшим или "мёртвым" сайтом

### MINI-5: Юнит-экономика для бизнеса (фреймворк + услуга)
- **Пример:** Финансовая модель агентства / SaaS / оффлайн-бизнеса — расчёт CAC, LTV, runway, break-even
- **Шаблон:** юнит-экономика как услуга — клиент даёт цифры, мы выдаём отчёт «делать продукт или нет»
- **Зачем нам самим (приоритет!):** Инне нужен фильтр для решений «какие проекты делать, от каких отказаться»
- **Зачем клиентам:** оффлайн-бизнес обычно не считает unit economics, для них это будет «вау»
- **Инструменты:** три внешних Claude-скилла: `startup-financial-modeling`, `startup-metrics-framework`, `market-sizing-analysis` (см. [knowledge/tools/skills-catalog.md](knowledge/tools/skills-catalog.md))
- **Применять для:** Beauty (перед запуском платных подписок), всех будущих SaaS-продуктов, выбора что делать из этого файла

### MINI-6: Партнёрский кабинет
- **Пример:** Веб-инструмент для управления партнёрской программой (учёт сделок, выплаты, кабинет партнёра)
- **Шаблон:** SaaS «партнёрка под ключ» для агентств и микробизнесов
- **Сначала для себя:** у Инны уже работает партнёрка вручную (10% / 6 мес / + последующие покупки) — упаковать как инструмент
- **Потом продукт:** продаём агентствам, тренерам, экспертам, у которых тоже есть рекомендации без учёта
- **Связано с:** [PLAN.md](PLAN.md) — стратегические инициативы

## 👤 Потенциальные клиенты (предложить продукт)

### Финансист из мастер-группы Инны
- **Кто:** Преподаёт финансовую грамотность, в мастер-группе Инны, хочет монетизироваться, не получается
- **Что предложить:** Продукт под её экспертизу — бот / приложение / мини-курс / квиз для квалификации клиентов
- **Бонус:** Возможна партнёрка — её экспертиза + наша разработка
- **Когда:** После того как Инна обсудит с ней напрямую. Если согласится — заводим папку `clients/finansist/`.

# RESPONSIBILITY-MATRIX — владельцы живых документов

Дата обновления: 2026-05-15

Этот файл отвечает на вопрос: **кто отвечает за то, чтобы документ не сгнил**.

Владелец не обязательно один всё пишет руками. Владелец:
- следит за актуальностью;
- решает, когда обновить;
- убирает противоречия;
- переводит устаревшее в архивный статус;
- эскалирует Инне только стратегические решения.

## Правило живого справочника

Если документ используется как справочник и на него ссылаются агенты:
- у него должен быть владелец;
- в начале желательно указывать дату обновления;
- если он устарел — это должно быть написано явно;
- если появился новый источник правды — старый документ должен ссылаться на новый.

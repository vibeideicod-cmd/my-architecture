# Раздел A — Визуал и нейрофотосессия

## A.1. Кто такой нейрокреатор
<!-- LIBRARIAN-PRINCIPLE: neurocreator-role -->

**Суть:** нейрокреаторы создают через ИИ:
- ИИ-модели (своих и чужих)
- ИИ-аватары
- ИИ-блогеров
- Виральный фото- и видео-контент

**Спрос на рынке:** растёт ежедневно — брендам нужны аватары, экспертам — упаковка, видео — всем.

**Экономика:**

| Что | Старое | С ИИ |
|---|---|---|
| 1 тематическая фотосессия | от 3000₽ | 1000₽/мес подписка на нейросеть = любое количество |
| Время | поиск студии, фотографа, дни ожидания | 30 минут от референса до результата |

---

## A.2. Три шага создания нейрофотосессии (KREA)
<!-- LIBRARIAN-PRINCIPLE: three-steps-neurophotosession -->

**Шаг 1. Референс в Pinterest**
- Регистрация на ru.pinterest.com
- Поиск: «стильные образы для фотосессии в стиле [предпочтение]»
- Лайфхак: добавить «эстетика» / «aesthetic» — даёт ещё более вдохновляющие примеры
- Создать «доску» (папку) для сохранения
- Кнопка «другие идеи» в доске — даёт сотни похожих
- Скачать выбранные на компьютер

**Шаг 2. Обучение модели в KREA**
- krea.ai → раздел Train
- Загрузить 10-50 своих фотографий (идеально 30-50)
- Разное качество, разные ракурсы (чем больше, тем качественнее модель)
- Train steps — на максимум
- Нажать Train → ждать обучения

**Шаг 3. Генерация по запросу**
- Загружаем готовый промпт (есть GPT-помощник для генерации промптов под KREA)
- Насыщенность модели = 0.8 (для максимальной реалистичности)
- Выбираем разрешение → генерируем

**GPT-помощник для промптов:**
> «создай промпт для KREA на основе референса, определи стиль, чтобы максимально соответствовало прикрепленному примеру»

## B.3. Базовая формула промпта для NanoBanana — 7 элементов
<!-- LIBRARIAN-PRINCIPLE: nanobanana-7-elements -->

```
[1. Главный объект] +
[2. Внешность и эмоция] +
[3. Поза / действие] +
[4. Одежда / стиль] +
[5. Фон / окружение] +
[6. Свет и атмосфера] +
[7. Формат и стиль изображения]
```

| Элемент | Примеры формулировок |
|---|---|
| 1. Главный объект | молодая женщина / бизнесмен / художница / подросток |
| 2. Внешность + эмоция | с длинными русыми волосами, зелёными глазами, лёгкой улыбкой |
| 3. Поза / действие | сидит / стоит / повернулась боком / держит чашку |
| 4. Одежда | в белом льняном платье / чёрном костюме / стиль кэжуал |
| 5. Фон | на фоне светлой стены / в парке / минималистичный фон |
| 6. Свет | мягкий дневной свет / тёплое вечернее освещение / контрастный |
| 7. Формат | портретный кадр / 16:9 / кинематографичный / в стиле Studio Ghibli |

---

## B.4. Секреты эффективного использования NanoBanana
<!-- LIBRARIAN-PRINCIPLE: nanobanana-secrets -->

1. **Качество исходника** — фото с равномерным освещением, минимум теней, чёткие, не размытые
2. **Понятный промпт = 3 уровня:**
   - Что изменить (конкретный объект/область)
   - На что заменить (детальное описание)
   - Стиль и атмосфера
3. **Многоэтапное редактирование** — не всё сразу. Сначала фон → потом одежда → потом аксессуары. Поэтапно даёт качественнее результат.
4. **Эксперименты со стилями:** добавлять «кинематографичный», «фотореалистичный», «в стиле Studio Ghibli» — наблюдать как меняется
5. **Русский язык** — NanoBanana отлично понимает русский. Pro-версия пишет на картинках на русском почти без ошибок

# Раздел C — Посты с ИИ

## C.1. Проблема шаблонных постов от AI
<!-- LIBRARIAN-PRINCIPLE: ai-posts-pattern-problem -->

**Статистика:**
- 3+ часа на создание поста
- 80% постов собирают менее 10 реакций
- 90% звучат шаблонно
- 70% авторов выгорают

**Решение:** AI всё равно может писать как живой человек — **разница в том, КАК просим**.

---

## C.2. 3 типа постов
<!-- LIBRARIAN-PRINCIPLE: three-post-types -->

| Тип | Содержание | Когда выбирать |
|---|---|---|
| **Экспертный** | Обучающие, аналитические, разборы кейсов | Есть уникальное знание + цифры подтверждают + ЦА ищет решение |
| **Вовлекающий** | Истории, провокации, инсайты | Есть яркая история + нужно построить доверие + хотим эмоций |
| **Продающий** | Прямая продажа, soft-продажа, прогревы | Есть чёткий оффер + ЦА готова к покупке + есть соц.доказательства |

---

## C.3. Структура промпта по типу поста
<!-- LIBRARIAN-PRINCIPLE: post-prompt-structure -->

**Экспертный пост:**

```
Роль: Эксперт по [ниша] с 10+ летним опытом
Контекст: ЦА, уровень подготовки, ключевая проблема
Элементы:
- Инсайт / открытие
- Разбор ошибок
- Практические шаги
- Измеримые результаты
Формат: 2000 знаков, структура «проблема → решение → применение → результат»
Тон: уверенный профессионал, делящийся проверенными методиками
```

**Вовлекающий пост:**

```
Роль: Опытный сторителлер
Структура:
- Завязка (неожиданная ситуация / противоречие / интригующий вопрос)
- Развитие (конфликт → попытки решения → поворотный момент)
- Кульминация (инсайт + решение + результат)
Фишка: техника «Прерванного рассказа» — разбить историю на части с интригующими подзаголовками
Тон: искренний рассказчик, делящийся личным опытом
```

**Продающий пост:**

```
Роль: Опытный копирайтер / маркетолог-практик / эксперт по воронкам
Контекст: продукт, ЦА, основные боли, уникальное преимущество
Элементы:
- Цепляющий оффер
- Описание проблемы
- Решение
- Доказательства
- Призыв к действию
Усиливающие: соц.доказательства / ограничения (время, количество) / гарантии / бонусы
Структура: «боль → решение → доказательства → оффер»
Тон: эмпатичный эксперт
```

## C.4. Лайфхаки для постов
<!-- LIBRARIAN-PRINCIPLE: post-lifehacks -->

| Тип | Фишки |
|---|---|
| Экспертные | «Малоизвестный факт:» / цифры и статистика / «Вопреки распространённому мнению...» |
| Вовлекающие | «Никогда не думал, что...» / диалоги / техника «Было / Стало» |

## C.5. Чек-лист проверки качества поста
<!-- LIBRARIAN-PRINCIPLE: post-quality-check -->

**1. Начало цепляет?**
- Первые 2 строки вызывают интерес
- Есть неожиданный элемент
- Понятна ценность для читателя

**2. Структура работает?**
- Логическая последовательность
- Удобная подача
- Нет «воды»

**3. Контент релевантен?**
- Соответствует цели
- Решает проблему ЦА
- Ведёт к нужному действию

# Раздел D — Контент-план через ИИ

## D.1. Проблемы хаотичного контент-плана
<!-- LIBRARIAN-PRINCIPLE: content-plan-problems -->

- Контент-план занимает 2-3 дня в месяц
- Половина запланированного не публикуется
- Темы повторяются, идеи кончаются
- Выгорание от постоянной генерации
- Нерегулярность убивает систему привлечения клиентов

**Цель урока:** создавать **месяц контента за 20 минут**.

---

## D.2. Формула 40/30/30 — анатомия идеального контент-плана
<!-- LIBRARIAN-PRINCIPLE: content-plan-40-30-30 -->

| % | Тип | Что внутри |
|---|---|---|
| **40%** | Образовательный | Решения проблем, инсайты, лайфхаки, разборы кейсов |
| **30%** | Развлекательный | Истории и метафоры, закулисье процессов, человеческие моменты |
| **30%** | Продающий | Кейсы и результаты, триггеры и боли, призывы к действию |

**Почему многие не получают продаж из контента:** нарушают эту пропорцию (например, 80% продающего → выглядит как реклама).

---

## D.3. Психология контента
<!-- LIBRARIAN-PRINCIPLE: content-psychology -->

**Психологические триггеры:**
- Социальное доказательство («у нас 1000+ клиентов»)
- Дефицит («осталось 3 места»)
- Взаимный обмен («я даю — ты получаешь, потом мне взаимность»)
- Последовательность (микро-обязательства, ведущие к большому)
- Авторитет (экспертиза, медийность)

**Эмоциональные крючки:**
- Любопытство
- FOMO (страх упущенной выгоды)
- Желание принадлежности
- Стремление к совершенству
- Поиск безопасности

**Копирайтинг-формулы:**

| Формула | Расшифровка | Когда |
|---|---|---|
| **PAS** | Проблема → Усиление → Решение | Боль ярко выражена |
| **AIDA** | Внимание → Интерес → Желание → Действие | Классическая воронка продаж |
| **4U** | Полезно → Срочно → Уникально → Ультраспецифично | Заголовки, офферы |

## E.1. Контент-маркетинг — стратегия привлечения клиентов (Алексей Южанинов, Урок 14)

### Главная идея
<!-- LIBRARIAN-PRINCIPLE: content-marketing-strategy -->

**Контент-маркетинг — это НЕ реклама и НЕ продажи.** Это способ делиться знаниями и опытом так, чтобы люди сами тянулись к вам как к эксперту.

«Ваша аудитория уже ищет то, что вы умеете делать. Контент — это способ быть там, где они ищут».

Это **длинная игра**: от незнакомца до доверяющего клиента может пройти неделя, месяц или три месяца. Но клиенты, которые приходят через контент, уже **тёплые**.

## E.2. Золотое правило 80/20 — баланс контента
<!-- LIBRARIAN-PRINCIPLE: content-80-20-rule -->

**Формула: 4 экспертных/личных + 1 продающий пост.**

- Если писать только «купи» — люди отписываются
- Если только умничать — уважают, но не платят
- Личный контент создаёт связь с аудиторией — он генерирует доверие
- **Баланс: 80% экспертизы + личности, 20% продажи**

## E.3. 3 вида контента
<!-- LIBRARIAN-PRINCIPLE: 3-content-types -->

| Вид | Что | Эффект |
|---|---|---|
| **Экспертный** | Ваш взгляд на рынок, тренды, профессию. Показываете, что думаете о происходящем | Люди сохраняют и цитируют |
| **Личный** | Истории, ошибки, процессы, закулисье | Люди любят и доверяют |
| **Продающий** | Предложение, оффер, призыв к действию | Конвертирует в сделки |

**Принцип:** нужны все три. Без личного контента нет эмоциональной связи; без продающего — некуда вести аудиторию; без экспертного — нет позиционирования.

## E.4. 5-шаговая контент-воронка
<!-- LIBRARIAN-PRINCIPLE: 5-step-content-funnel -->

| Шаг | Что происходит |
|---|---|
| 1. **Увидел контент** | Натянулся на ваш пост |
| 2. **Подписался** | Стало интересно, хочет видеть больше |
| 3. **Доверяет** | Видит ваши кейсы, результаты, процессы. Думает: «Этот человек понимает» |
| 4. **Пишет сам** | «А вы можете сделать такое для меня?» |
| 5. **Платит** | И с удовольствием, потому что **сам выбрал** вас |

**Важный момент:** контент-маркетинг — не «написал пост, получил клиента». От шага 1 до шага 5 может пройти неделя, месяц или три месяца. Но зато клиенты уже тёплые.

## E.5. Психологические барьеры новичка (и как их преодолеть)
<!-- LIBRARIAN-PRINCIPLE: content-psychology-barriers -->

Большинство начинает со ступора:
- «Я не умею продавать»
- «Не люблю рекламу»
- «Мне неловко писать о себе»

**Контент-маркетинг решает это:** вы делитесь знаниями и опытом так, чтобы люди сами тянулись к вам. Это **не маска** — ваш настоящий стиль общения, применённый к текстам.

**Преодоление барьеров:**
- Контент — не рекламный трюк, а способ быть там, где ищут клиенты
- Вы уже помогаете людям (есть портфолио, навыки, ценность для рынка)
- Нужно просто рассказать об этом честно, без фильтров

## E.6. Структура контент-плана 8/8/4 на месяц
<!-- LIBRARIAN-PRINCIPLE: content-plan-8-8-4 -->

20 тем на месяц, разбитых по 3 блокам:

| Блок | Кол-во | Что |
|---|---|---|
| **Образовательные** | 8 | Объясняют, учат, разбирают |
| **Вдохновляющие** | 8 | Кейсы, процессы, результаты, истории |
| **Продающие** | 4 | Оффер, место в работе, кейс с ценой |

Для каждой темы указать:
- Заголовок/тему поста
- Тип (образовательный / вдохновляющий / продающий)
- Боль аудитории, которую закрывает
- Крючок: первое предложение, которое остановит скроллинг

**Частота публикаций:** 5-7 постов в неделю достаточно. Качество &gt; количество.

> **Связь с формулой 40/30/30 из секции D.2:** обе формулы дают баланс. 40/30/30 — образовательный/развлекательный/продающий. 8/8/4 — образовательный/вдохновляющий/продающий. Главное — три блока, не один.

## E.7. Tone of Voice — как найти свой голос
<!-- LIBRARIAN-PRINCIPLE: tone-of-voice -->

«Голос — это не маска. Это ваш настоящий стиль общения, просто **осознанный** и применённый к текстам».

«Когда вы знаете свой голос — писать становится в 10 раз легче».

**5 вопросов для определения голоса (5 минут):**
1. Как вы объясняете что-то сложное другу? Через аналогии? Цифры? Истории?
2. Используете юмор? Какой — мягкий самоиронический или острый?
3. Вы больше вдохновляете или информируете?
4. Вы матёрый эксперт или честный практик, который учится вместе с читателем?
5. Какие три слова описывают вас как автора? (например: конкретный, живой, честный)

Ответы записываются в `tone-of-voice.md`. Дальше Claude использует этот файл при генерации постов — стиль остаётся вашим.

## E.8. Хорошие vs плохие посты
<!-- LIBRARIAN-PRINCIPLE: post-good-vs-bad -->

**❌ Слабый вариант (про инструмент):**
«Купите мой сайт за 30 000 рублей. Быстро и качественно. Звоните».

Холодно, неудобно, вызывает отторжение. 97% людей это игнорируют.

**✅ Хороший вариант (про клиента):**
«Вот как я сделал лендинг для фотографа за один вечер — и вот промпты, которые использовал».

Полезно, интересно, строит доверие.

**Метафора Левитта:** «Бурильщик продаёт не дрель — он продаёт дырку в стене. Покупателю не важно, как устроена дрель. Важно — попадёт ли картина туда, куда он хочет».

**Принцип:** «Пишите о дырках, не о дрелях» — о результатах и проблемах, не о техническом инструменте.

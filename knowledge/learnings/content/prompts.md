# Готовые промпты: создание контента через ИИ

> **Назначение:** все шаблоны промтов по созданию контента, готовые к копированию. Сгруппированы по 4 направлениям: визуал в KREA, NanoBanana, посты, контент-план.
> **Источники:** см. `sources/` в этой же папке.

# Раздел A — Визуал и нейрофотосессия (KREA)

## A.1. Помощник для генерации промптов KREA
<!-- LIBRARIAN-PROMPT: krea-prompt-helper -->

**Когда использовать:** есть референс из Pinterest, нужен промпт под него.

**В ChatGPT прикрепляешь референс и пишешь:**

```
Создай промпт для KREA на основе референса.
Определи стиль, чтобы максимально соответствовало прикрепленному примеру.
Промпт должен быть на английском, детализированный, с указанием:
- одежды
- освещения
- ракурса
- стиля (cinematic / fashion / portrait)
- настроения
```

---

## A.2. Промпт для генерации в KREA (на английском)
<!-- LIBRARIAN-PROMPT: krea-generation -->

**Структура (на английском, KREA лучше работает):**

```
[type of shot: cinematic / fashion / portrait]
of a [subject: young woman / man / etc.],
wearing [clothes detailed],
in [setting / location detailed],
illuminated by [lighting type and atmosphere],
[camera movement / angle if relevant],
[mood and aesthetic: minimalist / luxury / dynamic],
high-detail, realistic texture
```

**Пример:**

```
A cinematic portrait of a confident woman with short blonde bob haircut
wearing an elegant black blazer with large gold floral brooches,
arms crossed, clean grey studio background,
soft diffused studio light,
modern fashion portrait, minimalist aesthetic,
high-detail, realistic texture
```

# Раздел B — NanoBanana (Gemini 2.5 Flash Image)

## B.1. Базовый шаблон промпта (7 элементов)
<!-- LIBRARIAN-PROMPT: nanobanana-7-elements-template -->

```
[Главный объект: кто/что] +
[Внешность и эмоция: цвет волос, выражение лица] +
[Поза / действие: что делает, куда смотрит] +
[Одежда / стиль: что на нем, какого цвета] +
[Фон / окружение: где происходит] +
[Свет и атмосфера: тип освещения, время суток] +
[Формат и стиль: 16:9 / портрет / кинематографичный / в стиле X]
```

**Пример:**

```
Молодая женщина с длинными русыми волосами, зелёными глазами,
лёгкой улыбкой, нейтральным лицом, сидит за столом в кафе,
держит чашку кофе, в белом льняном платье, стиль кэжуал,
на фоне светлой стены с растениями, мягкий дневной свет,
тёплая атмосфера, портретный кадр, фотореалистично, высокая детализация.
```

## B.2. 10 промптов для 10 фишек NanoBanana
<!-- LIBRARIAN-PROMPT: nanobanana-10-prompts -->

### 1. Создание разных поз из одного фото

**Шаг 1 (в ChatGPT с фото-референсом):**

```
Опиши позу человека на этом фото подробно — положение рук, ног, головы,
направление взгляда, общее настроение позы.
На основе описания создай промпт для NanoBanana,
чтобы воспроизвести эту позу с другим человеком (моим фото).
```

**Шаг 2 (в NanoBanana):** загрузить своё фото + полученный промпт.

### 2. Макросъёмка деталей

```
Мне нужна макросъёмка [браслета на руке девушки / текстуры ткани одежды /
украшения / часов]. Профессиональный свет, высокая детализация, премиум-стиль.
```

### 3. Смена одежды

```
Надень на [объект слева / девушку / мужчину] [конкретный предмет одежды].
Сохрани лицо, позу, фон. Идеально подгони под фигуру.
```

### 4. Объединение нескольких картинок

```
Помести [объект] на [фон]. Камера снимает [сверху / сбоку / снизу].
Профессиональный свет в кадре, [блики / тени / эффекты] играют на [объекте].
Минималистичный стильный кадр.
```

### 5. Замена объектов на изображении

```
Создай фото [объект 1] с [объект 2 с другой картинки]. [Опиши взаимодействие].
Сохрани общий стиль и атмосферу первой картинки.
```

### 6. Расширение изображения / смена формата

```
Расширь это фото в формате [16:9 / 9:16 / квадрат], чтобы оно стало идеальным
для [баннера / сторис / поста]. Добавь по бокам естественное продолжение
[пейзажа / интерьера / фона], сохраняя освещение и детализацию.
Сделай так, будто фото было снято с таким соотношением сторон изначально.
```

### 7. Реставрация и улучшение качества

```
Улучши качество старого снимка. Убери все шумы и дефекты,
сделай более чётким и детализированным, раскрась в реалистичные цвета.
Сохрани при этом атмосферу той эпохи.
```

### 8. Замена текста по референсу

```
Напиши на этом баннере [новый текст] в этом же стиле, шрифте, цвете.
Сохрани композицию баннера.
```

### 9. Визуализация и макеты

```
На основе этого эскиза создай фотореалистичный 3D-макет [квартиры / интерьера].
Добавь текстуры, освещение, мебель, чтобы выглядело как реальная комната.
Стиль: [минимализм / лофт / классика / японский].
```

### 10. Создание стикеров и аватаров

```
На основе этого снимка создай 3D-аватар в стиле [кукольной анимации /
Pixar / аниме / фигурки funko pop]. Сделай с большими глазами, доброй улыбкой,
[опционально: детали — например, в костюме самурая, с книгой].
```

## B.3. Универсальный промпт для нейрофотосессии без фотографа
<!-- LIBRARIAN-PROMPT: nanobanana-photoshoot-without-photographer -->

**Из домашнего задания урока — серия из 3 фото:**

**Фото 1 — новый фон:**

```
Замени фон на [пляж с пальмами и мягким закатным светом /
кабинет с книжными полками / городскую улицу вечером].
Сохрани мою позу, одежду, выражение лица.
Свет от фона должен естественно ложиться на меня.
```

**Фото 2 — новый образ:**

```
Одень меня в [чёрный классический костюм / летнее платье / спортивный комплект].
Сделай студийное освещение. Сохрани лицо, причёску, общий ракурс.
```

**Фото 3 — новая эмоция:**

```
Сохрани моё лицо без изменений. Добавь [лёгкую улыбку / уверенное выражение глаз /
задумчивый взгляд вдаль]. Поправь свет под новое выражение.
```

# Раздел C — Посты с ИИ

## C.1. Промпт для экспертного поста
<!-- LIBRARIAN-PROMPT: expert-post -->

```
Роль: Эксперт по [ниша] с 10+ летним опытом, специализация [конкретная зона],
автор [методики/исследования], успешные кейсы [результаты].

Задача: Создай обучающий пост о [тема], который:
- Раскрывает одну конкретную технику
- Показывает её применение на практике
- Включает распространённые ошибки
- Даёт измеримые результаты

Контекст:
- ЦА: [портрет с 6 якорями: кто/когда/где/проблема/состояние/формат]
- Уровень подготовки: [новички / средний / продвинутые]
- Ключевая боль: [что мучает в моменте чтения]

Структура: проблема → решение → применение → результат
Объём: до 2000 знаков
Стиль: экспертный, но доступный, без жаргона
Тон: уверенный профессионал, делящийся проверенными методиками

Фишки усиления:
- Используй фразу «Малоизвестный факт:»
- Добавь конкретные цифры и статистику
- Включи «Вопреки распространённому мнению...»
```

---

## C.2. Промпт для вовлекающего поста (история)
<!-- LIBRARIAN-PROMPT: engaging-post -->

```
Роль: Опытный сторителлер, мастер историй с эмоциональным эффектом.

Задача: Создай вовлекающий пост-историю о [тема трансформации], который:
- Начинается с момента напряжения
- Имеет яркий поворотный момент
- Заканчивается значимым результатом
- Содержит обучающий элемент

Контекст:
- ЦА: [портрет]
- Ключевая боль: [что чувствует ЦА]
- Что должна понять ЦА после прочтения

Структура: завязка → конфликт → решение → результат
Стиль: эмоциональный нарратив с диалогами и деталями
Тон: искренний рассказчик, делящийся личным опытом

Фишки усиления:
- Начни с «Никогда не думал, что...»
- Используй технику «Прерванного рассказа» — разбей на части с интригующими подзаголовками
- Добавь диалоги между героями
- Используй технику «Было / Стало»
```

---

## C.3. Промпт для продающего поста
<!-- LIBRARIAN-PROMPT: selling-post -->

```
Роль: Опытный копирайтер, маркетолог-практик, эксперт по воронкам продаж.

Задача: Создай продающий пост о [продукт/услуга], который:
- Начинается с главной боли ЦА
- Показывает понимание проблемы
- Представляет решение через результаты учеников/клиентов
- Ведёт к действию

Контекст:
- Продукт/услуга: [описание]
- ЦА: [портрет]
- Основные боли: [список]
- Уникальное преимущество: [что отличает]
- Социальные доказательства: [кейсы, отзывы, цифры]

Структура: боль → решение → доказательства → оффер → призыв
Объём: до 2000 знаков
Тон: эмпатичный эксперт, предлагающий проверенное решение

Усиливающие элементы:
- Социальные доказательства
- Ограничения (время / количество)
- Гарантии
- Бонусы

Фишки:
- Начни с главной боли
- Добавь истории клиентов
- Включи «Бонус в конце поста»
- Используй цифры и факты
```

# Раздел D — Контент-план

## D.1. Базовый промпт контент-плана
<!-- LIBRARIAN-PROMPT: basic-content-plan -->

**Когда использовать:** быстрый старт, без углубления.

```
Создай контент-план на неделю для [эксперта / бизнеса в нише].

Структура (формула 40/30/30):
- 3 обучающих поста (40%)
- 2 развлекательных (30%)
- 2 продающих (30%)

Для каждого поста укажи:
1. День публикации
2. Тему
3. Основной посыл
4. Структуру подачи

Пришли в виде таблицы.
```

---

## D.2. Каскадный промпт контент-плана (3 шага)
<!-- LIBRARIAN-PROMPT: cascade-content-plan -->

**Когда использовать:** когда нужен глубокий, продающий план.

**Шаг 1 — Стратегическая основа:**

```
Проанализируй нишу [пример: онлайн-фитнес / нутрициология / бьюти-салон] и создай:

1. Ключевые темы контента:
   - Топ-5 болей аудитории
   - Топ-5 желаний аудитории
   - Топ-5 возражений

2. Форматы подачи:
   - 3 образовательных формата
   - 3 развлекательных формата
   - 3 продающих формата

3. Триггеры вовлечения:
   - 3 главных мотиватора аудитории
   - 3 основных страха
   - 3 точки желаний

Используй психологические триггеры (соц.доказательство, дефицит, авторитет)
и копирайтинг-формулы (PAS, AIDA, 4U).
```

**Шаг 2 — Структура месяца (каркас):**

```
На основе результатов шага 1 создай структуру месяца контента (4 недели):

НЕДЕЛЯ 1: Активация интереса
- 3 поста про боли
- 2 поста с решениями
- 2 поста с результатами

НЕДЕЛЯ 2: Углубление доверия
- 3 поста с инсайтами
- 2 кейса с трансформациями
- 2 поста с экспертными разборами

НЕДЕЛЯ 3: Продуктовая неделя
- 3 поста про преимущества
- 2 поста с возражениями
- 2 поста с призывами

НЕДЕЛЯ 4: Закрытие сомнений
- 3 поста с кейсами
- 2 поста с гарантиями/бонусами
- 2 финальных продающих

Для каждого поста укажи день публикации.
```

**Шаг 3 — Детализация каждого поста:**

```
Для каждого из постов в плане шага 2 распиши:

1. День публикации
2. Тема поста (1 предложение)
3. Основной посыл
4. Тип поста (экспертный / вовлекающий / продающий)
5. Структура подачи (PAS / AIDA / 4U / своя)
6. Психологический триггер
7. Эмоциональный крючок
8. Призыв к действию (если уместно)

Пришли в виде таблицы.
```

## D.3. Промпт «1 идея = 10 постов» (масштабирование)
<!-- LIBRARIAN-PROMPT: scale-one-idea-ten-posts -->

**Когда использовать:** есть одна сильная тема, нужно растянуть на серию.

```
У меня есть тема [название темы].

Создай из этой одной темы 10 разных постов, используя:

1. Разные углы подачи:
   - История из жизни
   - Кейс клиента
   - Разбор ошибки
   - Экспертный инсайт
   - Развенчание мифа
   - Сравнение «было/стало»
   - Чек-лист
   - Мини-гайд
   - Вопрос аудитории
   - Прямая продажа (мягко)

2. Разные типы (по 3 постам экспертный/вовлекающий/продающий)

3. Разные триггеры (минимум 5 разных психологических триггеров)

Каждый пост — самостоятельный, но все вместе создают цельную картину темы.
```

# Связь с темой `prompting/`

Все промпты этого раздела используют принципы из `learnings/prompting/`:
- **CRAFT** — лежит в основе каждого шаблона (контекст + роль + действие + формат + тон)
- **Контекстные якоря** — обязательно для постов и контент-плана
- **Few-shot** — рекомендуется давать примеры успешных постов нишей
- **Эксперт наоборот** — для финальной полировки результата

> **Стандартная связка:** сначала собрать промпт по шаблону отсюда → применить технику CRAFT/якоря → полученный результат прогнать через «Эксперт наоборот».

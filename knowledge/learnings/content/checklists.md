# Раздел B — NanoBanana

## Чек-лист «Готов ли мой промпт для NanoBanana» (7 элементов)
<!-- LIBRARIAN-CHECKLIST: nanobanana-prompt-readiness -->

- [ ] **1. Главный объект** — конкретно, кто или что
- [ ] **2. Внешность и эмоция** — цвет волос, выражение лица, настроение
- [ ] **3. Поза / действие** — что делает, куда смотрит
- [ ] **4. Одежда / стиль** — что надето, какого цвета, в каком стиле
- [ ] **5. Фон / окружение** — где происходит
- [ ] **6. Свет и атмосфера** — тип освещения, время суток
- [ ] **7. Формат и стиль** — портрет / 16:9 / кинематографичный / Studio Ghibli

**Если хотя бы 1 элемент пропущен** — результат может «уехать в случайность».

---

## Чек-лист «Хорошее исходное фото для NanoBanana»
<!-- LIBRARIAN-CHECKLIST: nanobanana-good-source -->

- [ ] Освещение **равномерное** (без жёстких теней на лице)
- [ ] Минимум теней на главном объекте
- [ ] Изображение чёткое, не размытое
- [ ] Нет фильтров (Instagram-фильтры портят результат)
- [ ] Лицо хорошо видно, не закрыто волосами/руками
- [ ] Высокое разрешение (минимум 1024px по короткой стороне)

---

## Чек-лист «Многоэтапное редактирование в NanoBanana»
<!-- LIBRARIAN-CHECKLIST: nanobanana-multi-step -->

**Не пытаюсь изменить всё одним промптом.** Иду шагами:

- [ ] Шаг 1: смена фона (если нужен)
- [ ] Шаг 2: смена одежды (на новом фоне)
- [ ] Шаг 3: смена позы/выражения
- [ ] Шаг 4: добавление аксессуаров
- [ ] Шаг 5: финальная правка света/цветокоррекции

# Раздел C — Посты

## Чек-лист «Качество поста» (проверка перед публикацией)
<!-- LIBRARIAN-CHECKLIST: post-quality -->

**1. Начало цепляет:**
- [ ] Первые 2 строки вызывают интерес
- [ ] Есть неожиданный элемент / интрига
- [ ] Ценность для читателя ясна с первой строки

**2. Структура работает:**
- [ ] Логическая последовательность
- [ ] Удобная подача (абзацы, списки, эмодзи в меру)
- [ ] Нет «воды» и общих фраз

**3. Контент релевантен:**
- [ ] Соответствует цели поста
- [ ] Решает проблему ЦА
- [ ] Ведёт к нужному действию (комментарий / переход / покупка)

**Если хотя бы 2 пункта не выполнены** — переделать через итерации.

---

## Чек-лист «Какой тип поста выбрать»
<!-- LIBRARIAN-CHECKLIST: post-type-choice -->

| Сценарий | Какой тип |
|---|---|
| Есть уникальное знание + цифры подтверждают + ЦА ищет решение | **Экспертный** |
| Есть яркая история + нужно построить доверие + хочу вызвать эмоцию | **Вовлекающий** |
| Есть чёткий оффер + ЦА готова к покупке + есть соц.доказательства | **Продающий** |
| Не знаю — берем экспертный (универсально, низкий риск) | **Экспертный** |

# Раздел D — Контент-план

## Фреймворк «Формула 40/30/30»
<!-- LIBRARIAN-FRAMEWORK: formula-40-30-30 -->

**Назначение:** проверить пропорции в контент-плане.

```
40% — Образовательный контент
   ├─ Решения проблем
   ├─ Инсайты и лайфхаки
   └─ Разборы кейсов

30% — Развлекательный контент
   ├─ Истории и метафоры
   ├─ Закулисье процессов
   └─ Человеческие моменты

30% — Продающий контент
   ├─ Кейсы и результаты
   ├─ Триггеры и боли
   └─ Призывы к действию
```

**На месяц 20 постов:**
- 8 образовательных
- 6 развлекательных
- 6 продающих

**Проверка:** если в плане за месяц 80% продающих — это реклама, не контент. Перебалансировать.

## Фреймворк «Каскадный промпт контент-плана» (3 шага)
<!-- LIBRARIAN-FRAMEWORK: cascade-content-plan -->

```
Шаг 1. СТРАТЕГИЧЕСКАЯ ОСНОВА
   ├─ Топ-5 болей / желаний / возражений ЦА
   ├─ 9 форматов подачи (3 образ. / 3 развл. / 3 прод.)
   └─ Триггеры (3 мотиватора / 3 страха / 3 желания)

Шаг 2. КАРКАС МЕСЯЦА (4 недели)
   ├─ Неделя 1: Активация интереса (боли + решения + результаты)
   ├─ Неделя 2: Углубление доверия (инсайты + кейсы + разборы)
   ├─ Неделя 3: Продуктовая неделя (преимущества + возражения + призывы)
   └─ Неделя 4: Закрытие сомнений (кейсы + гарантии + финальные продающие)

Шаг 3. ДЕТАЛИЗАЦИЯ КАЖДОГО ПОСТА
   ├─ День публикации
   ├─ Тема (1 предложение)
   ├─ Основной посыл
   ├─ Тип (экспертный / вовлекающий / продающий)
   ├─ Структура подачи (PAS / AIDA / 4U)
   ├─ Психологический триггер
   ├─ Эмоциональный крючок
   └─ Призыв к действию
```

## Фреймворк «5 психологических триггеров»
<!-- LIBRARIAN-FRAMEWORK: 5-psychological-triggers -->

| Триггер | Когда применять | Пример |
|---|---|---|
| **Социальное доказательство** | Когда нужно построить доверие | «1000+ выпускников нашли работу за 3 месяца» |
| **Дефицит** | Когда нужна срочность | «Осталось 3 места в группе» |
| **Взаимный обмен** | Перед прямой продажей | «Бесплатный гайд → потом продажа» |
| **Последовательность** | Долгие воронки | Микро-обязательства (лайк → комментарий → бесплатник → покупка) |
| **Авторитет** | Премиум-сегмент | Регалии, медийность, экспертность |

---

## Фреймворк «5 эмоциональных крючков»
<!-- LIBRARIAN-FRAMEWORK: 5-emotional-hooks -->

- **Любопытство** — «Ты не поверишь, что произошло...»
- **FOMO (страх упущенной выгоды)** — «Те, кто пропустил это в 2020-м, до сих пор жалеют»
- **Желание принадлежности** — «Все умные предприниматели уже это делают»
- **Стремление к совершенству** — «Хочешь идеальный результат? Вот как»
- **Поиск безопасности** — «Простой способ не потерять деньги»

## Фреймворк «Копирайтинг-формулы» (выбор по задаче)
<!-- LIBRARIAN-FRAMEWORK: copywriting-formulas -->

| Формула | Расшифровка | Когда применять |
|---|---|---|
| **PAS** | Problem → Agitation → Solution (Проблема → Усиление → Решение) | Боль ярко выражена, нужно «прокопать» больнее → дать решение |
| **AIDA** | Attention → Interest → Desire → Action | Классическая воронка продаж — холодная аудитория |
| **4U** | Useful + Urgent + Unique + Ultra-specific | Заголовки, офферы — каждое слово на счёт |

**Шаблон 4U для заголовков:**

```
[Polezno: что получишь] + [Срочно: к какой дате] +
[Уникально: почему только мы] + [Ультраспецифично: конкретные цифры]

Пример:
"Получи 30 готовых постов под свою нишу за вечер —
с авторскими промптами, которых нет в открытых базах,
гарантировано закроют 5 главных болей твоей ЦА"
```

## Шаблон «План масштабирования: 1 идея = 10 постов»
<!-- LIBRARIAN-TEMPLATE: scale-one-idea -->

**Когда нужен:** есть одна сильная тема, надо растянуть.

**Шаблон 10 углов подачи одной темы:**

| # | Угол | Тип |
|---|---|---|
| 1 | История из жизни | Вовлекающий |
| 2 | Кейс клиента | Продающий |
| 3 | Разбор ошибки | Экспертный |
| 4 | Экспертный инсайт | Экспертный |
| 5 | Развенчание мифа | Экспертный |
| 6 | Сравнение «было / стало» | Вовлекающий |
| 7 | Чек-лист | Экспертный |
| 8 | Мини-гайд | Экспертный |
| 9 | Вопрос аудитории | Вовлекающий |
| 10 | Прямая продажа (мягко) | Продающий |

**Распределение по типам:** 5 экспертных + 3 вовлекающих + 2 продающих ≈ соответствует 50/30/20 (близко к 40/30/30 формуле).

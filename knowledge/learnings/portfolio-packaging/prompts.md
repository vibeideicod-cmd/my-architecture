# Готовые промпты: упаковка портфолио

> **Назначение:** 3 промпта из урока, готовые к копированию и применению. Используются последовательно: 1 → 2 → 3.
> **Принцип:** промпты — рабочий инструмент. Команда может использовать их как есть, адаптировать под клиента, или предлагать свои варианты.

---

## Как пользоваться

1. Промпты применяются **последовательно** — результат первого используется во втором, потом в третьем.
2. Заменить переменные `{в фигурных скобках}` на конкретные значения под клиента.
3. Использовать в нужной нейросети (в обучении — Claude, у нас можно DeepSeek или Claude).
4. Результаты каждого промпта **сохранять** — они нужны на следующих шагах.

---

## Последовательность применения

```
Промпт 1: Исследование рынка
  ↓ (получили: боли клиентов + позиционирование по нише)
Промпт 2: Интервью для позиционирования
  ↓ (получили: 3 варианта позиционирования через ценность)
Промпт 3: Создание HTML-страницы портфолио
  ↓ (получили: готовый лендинг под публикацию)
```

## Промпты

### 1. ПРОМПТ — ИССЛЕДОВАНИЕ РЫНКА
<!-- LIBRARIAN-PROMPT: market-research -->

**Когда использовать:** на старте — когда специалист определился с нишей и хочет понять рынок снаружи перед тем как формулировать своё позиционирование.

**Целевая нейросеть:** Claude (по обучению) — у нас можно использовать DeepSeek.

**Промпт:**

```
Я разработчик: создаю сайты, боты и автоматизации с помощью ИИ-инструментов
(Claude Code, Cursor, Lovable). Хочу специализироваться на нише: {НИША}

Например: автоматизация для салонов красоты / приложения для маркетплейсов /
боты для онлайн-школ.

Проведи исследование этого рынка:

1. БОЛИ КЛИЕНТОВ
   - Какие типичные проблемы испытывают владельцы бизнеса в нише {НИША}?
   - Что они делают вручную и ненавидят делать?

2. СУЩЕСТВУЮЩИЕ РЕШЕНИЯ
   - Какие решения уже есть? Что у них есть? Чего не хватает?

3. МОЁ ПОЗИЦИОНИРОВАНИЕ
   - Как я могу позиционировать себя — не через инструменты, а через результат,
     который получает клиент?

4. ВАРИАНТЫ ПОЗИЦИОНИРОВАНИЯ
   - Напиши 3 варианта (1-2 предложения каждый): кто я и какую конкретную
     проблему решаю.
   - Важно: без слов «разработчик», «программист», «вайбкодер» — только
     ценность для клиента.
```

**Адаптация под Ирину (дизайнер):**

Заменить «разработчик: создаю сайты, боты и автоматизации» на:
> «Я дизайнер: изготавливаю брендбуки, лендинги, мини-приложения и иллюстрации с помощью ИИ-инструментов и vibe coding»

Запрещённые слова к замене: вместо «разработчик/программист/вайбкодер» — без «дизайнер», «специалист по графике», названий инструментов.

**Что должно получиться на выходе:** отчёт по 4 разделам — боли клиентов в нише, что есть на рынке, варианты позиционирования через ценность, 3 варианта формулировки.

---

### 2. ПРОМПТ — ИНТЕРВЬЮ ДЛЯ ПОЗИЦИОНИРОВАНИЯ
<!-- LIBRARIAN-PROMPT: positioning-interview -->

**Когда использовать:** после Промпта 1 — когда есть понимание рынка снаружи. Цель: понять себя изнутри + найти пересечение «боли рынка ↔ свои сильные стороны».

**Целевая нейросеть:** Claude / DeepSeek.

**Промпт:**

```
Я хочу разработать своё позиционирование как специалист, который создаёт
сайты, боты и автоматизации с помощью ИИ-инструментов разработки.

Проинтервьюируй меня — задай 10 вопросов, чтобы понять:
- Какой у меня профессиональный опыт и экспертиза в конкретных нишах
- Какие проблемы бизнеса мне понятны лучше среднего
- Какие проекты я уже сделал и чем горжусь
- С какими клиентами мне комфортнее работать

После моих ответов — предложи 3 варианта позиционирования. Формулируй через
результат и боль клиента, а не через технологии или инструменты.

Объясни, почему каждый вариант может работать именно для меня.
```

**Адаптация под Ирину (дизайнер):**

Заменить «специалист, который создаёт сайты, боты и автоматизации» на:
> «Я дизайнер, который изготавливает дизайн-системы, брендбуки, лендинги и интерактивные продукты»

**Что должно получиться:** 10 вопросов от нейросети → ваши ответы → 3 варианта позиционирования с обоснованием.

### 3. ПРОМПТ — СОЗДАТЬ ПОРТФОЛИО-СТРАНИЦУ
<!-- LIBRARIAN-PROMPT: create-portfolio-page -->

**Когда использовать:** финальный шаг — когда есть позиционирование (из Промпта 2) и описанные проекты. На выходе — HTML-страница портфолио.

**Целевая нейросеть:** Claude Code (для генерации HTML) — лучший вариант. Можно DeepSeek или Claude.ai.

**Промпт:**

```
Помоги мне создать портфолио-страницу специалиста по автоматизации и
разработке под конкретную нишу.

МОЁ ПОЗИЦИОНИРОВАНИЕ:
{Вставь результат из Промпта 2 — формулировку через ценность для клиента,
без слова «вайбкодер»}

МОИ ПРОЕКТЫ (опиши каждый):
1. {Проект 1}: задача — {описание}, что сделал — {описание},
   результат — {описание}
2. {Проект 2}: ...
3. {Проект 3}: ...

БОЛИ КЛИЕНТОВ В МОЕЙ НИШЕ:
{Вставь из исследования рынка}

Создай портфолио-страницу в HTML:
- Заголовок — конкретно, что я делаю и для кого (без слов «разработчик»,
  «вайбкодер», «специалист»)
- Блок «Что я решаю» — 3-4 конкретные боли клиентов с примерами
- Блок «Проекты» — карточки: боль → решение → результат
- Призыв к действию — как связаться

Стиль: современный, тёмный фон, профессионально. Тон: как у человека
с экспертизой в нише, а не технарь.
```

**Адаптация под Ирину:**

Заменить «специалист по автоматизации и разработке» на «дизайнер, изготавливающий брендбуки, лендинги, мини-аппы и иллюстрации».

Стиль изменить с «современный, тёмный фон» на:
> «Стиль: палитра Хвоя `#306654` + Ваниль `#FCFAE1` + Коралл `#FF935E`. Шрифты: Playfair Display (заголовки) + Montserrat (текст). Тон: тёплый, личностный, через метафору цвета.»

Это согласуется с нашим брендбуком Ирины (palette.md + typography.md + voice.md).

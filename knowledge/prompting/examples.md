# Банк примеров промтов

Реальные примеры из практики. Используй как шаблоны — адаптируй под задачу.

---

## Шкала качества промта

```
Плохой → Обычный → Хороший → Отличный
```

**Плохой** — нет контекста, нет структуры, AI догадывается сам
**Обычный** — есть задача и аудитория, но нет роли/формата/тона
**Хороший** — есть 3-4 элемента CRAFT
**Отличный** — все 5 элементов CRAFT + конкретные критерии

---

## Пример 1: Продающий пост о курсе

**Задача:** написать продающий пост о курсе по фотографии

### Плохой промт
```
напиши продающий текст о курсе по фотографии
```
Проблема: AI не знает ничего — ни аудиторию, ни платформу, ни задачу поста.

---

### Обычный промт
```
Напиши продающий текст о курсе по фотографии для Instagram,
целевая аудитория — начинающие фотографы
```
Лучше, но нет роли, нет формата, нет тона.

---

### Отличный промт
```
Представь, что ты фотограф-преподаватель с 10-летним опытом,
автор методики «Первые красивые фото за 3 дня»,
превратил 1000+ новичков в успешных фотографов.

Контекст:
- Аудитория: новички 25-35 лет, мечтают о красивых фото для Instagram
- Главная боль: думают, нужна дорогая техника и долгое обучение
- Наше решение: результат с первого занятия на любой технике
- Подтверждение: 80% учеников делают продающие фото через 2 месяца

Создай продающий пост, который:
1. Разрушает миф о дорогой технике
2. Показывает истории успеха учеников
3. Ведёт к регистрации на бесплатный мастер-класс

Формат:
- Instagram-пост, 1500 знаков
- Структура: боль → миф → решение → доказательства → призыв
- Обязательно: фото до/после, эмодзи, призыв к действию

Тон:
- Вдохновляющий, но экспертный
- Без сложных терминов
- Акцент на простоту и достижимость результата
```

**Что изменилось:** роль даёт экспертизу, контекст даёт понимание аудитории, структура действия даёт конкретику, формат исключает догадки, тон задаёт голос.

## Пример 2: Текст для лендинга (санаторий)

**Задача:** написать текст первого экрана для санатория

### Плохой промт
```
Напиши текст для сайта санатория
```

### Отличный промт
```
Ты опытный копирайтер для сферы оздоровления и туризма,
специализируешься на конвертирующих лендингах для санаториев,
написал тексты для 30+ объектов на черноморском побережье.

Контекст:
- Санаторий «Южнобережный» в Крыму, работает 40 лет
- Аудитория: семьи с детьми 35-55 лет, ищут отдых + лечение
- Боли: дорогие курорты за рубежом закрыты, хотят качественный отдых в России
- Конкуренты делают упор на советское прошлое, мы — на современное оборудование

Создай текст первого экрана (hero-блок):
- Заголовок: цепляет за боль или мечту, до 8 слов
- Подзаголовок: раскрывает главное УТП, 1-2 предложения
- Кнопка CTA: конкретный призыв, 3-5 слов

Формат:
- Три отдельных текстовых блока: H1 / H2 / CTA
- Объём: H1 до 50 знаков, H2 до 120 знаков
- Дай 3 варианта на выбор

Тон:
- Тёплый, доверительный, без пафоса
- Как будто советует хороший друг, который там бывал
- Никаких слов «уникальный», «лучший», «незабываемый»
```

## Пример 3: Пост для соцсети (малый бизнес)

**Задача:** пост для Instagram автошколы

### Отличный промт
```
Ты SMM-специалист с опытом работы с автошколами и курсами,
умеешь писать тексты которые вовлекают и продают без агрессии.

Контекст:
- Автошкола «Первый руль», Краснодар
- Аудитория: молодёжь 18-25 лет, получают права впервые
- Боль: боятся не сдать экзамен, боятся трассы, стыдно не уметь
- Фишка: 97% наших учеников сдают с первого раза (реальная цифра)

Создай пост-развенчание мифа:
1. Начало: цепляющее утверждение которое читатель сам себе говорил
2. Середина: развенчание мифа + почему именно у нас это решено
3. Конец: мягкий призыв записаться на пробное занятие (бесплатно)

Формат:
- Instagram-пост, 800-1000 знаков
- 3-4 коротких абзаца, не монолит
- 2-3 эмодзи максимум, только в конце абзацев
- Хэштеги: 5 штук в конце

Тон:
- Дружеский, как старший друг который уже умеет водить
- Без менторства и поучений
- Немного юмора — один лёгкий момент
```

## Пример 4: Email-рассылка

**Задача:** письмо после записи на консультацию

### Отличный промт
```
Ты специалист по email-маркетингу для малого бизнеса,
понимаешь как важно первое впечатление после записи клиента.

Контекст:
- Клиент только что записался на бесплатную консультацию
- Он нервничает — правильно ли сделал, не потратит ли время впустую
- Консультация через 2 дня
- Цель письма: снять тревогу, подготовить к встрече, создать ожидание

Создай welcome-письмо:
1. Тема письма: снимает тревогу + создаёт интригу
2. Тело: подтверждение записи + что будет на встрече + как подготовиться
3. Постскриптум: один небольшой бонус (ссылка на полезный материал)

Формат:
- Объём: 1500-2000 знаков
- Структура: приветствие → подтверждение → что ждёт → подготовка → PS
- Кнопка: «Добавить в календарь»
- Подпись: от имени реального человека, не компании

Тон:
- Тёплый, немного личный
- Как будто пишет живой человек, не автоматическая система
- Уверенный, но без самовосхваления
```

## Шаблон для быстрого старта

Скопируй и заполни квадратные скобки:

```
Ты [должность/роль] с [опыт] лет, специализируешься на [область],
[конкретное достижение которое доказывает экспертизу].

Контекст:
- Продукт/услуга: [что именно]
- Аудитория: [кто, возраст, характеристики]
- Главная боль: [что мучает аудиторию]
- Наше решение/фишка: [как мы решаем боль]

Создай [тип контента], который:
1. [цель 1]
2. [цель 2]
3. [цель 3]

Формат:
- Тип: [пост/статья/письмо]
- Объём: [знаки или слова]
- Структура: [разделы]
- Обязательно: [что должно быть]

Тон:
- [тональность]
- Пиши как [метафора персонажа]
- Избегай: [что нельзя]
```

# Работа со стилем: копирование и создание цифровой копии

Техники для воспроизведения чужого стиля и создания собственного шаблона голоса. Применяй при создании контента под конкретного клиента или автора.

---

## Зачем это нужно

- Клиент хочет, чтобы тексты звучали «в его голосе»
- Нужно адаптировать контент под стиль известного эксперта для клиента
- Клиент не ведёт соцсети, но хочет начать — нужно найти его стиль

---

## Метод 1: Копирование стиля известного автора

Четыре шага. Каждый — отдельный запрос к AI.

### Шаг 1 — Анализ стиля
```
Проанализируй стиль автора [ИМЯ] на основе следующих постов:
[ВСТАВИТЬ 5–10 ОРИГИНАЛЬНЫХ ПОСТОВ]

Выдели:
1. Структуру мышления:
   - Как строит аргументацию
   - Какие приёмы убеждения использует

2. Языковые паттерны:
   - Типичные обороты речи
   - Длину предложений
   - Характерные слова и фразы

3. Эмоциональные триггеры:
   - Как вызывает эмоции
   - Какие приёмы вовлечения использует
```

### Шаг 2 — Создание шаблона стиля
```
На основе проведённого анализа создай шаблон поста в стиле [АВТОР].

Структура шаблона:
1. Открывающая фраза (как автор начинает пост)
2. Основная часть (как развивает мысль)
3. Заключение (как заканчивает)
4. Характерные элементы:
   - Типичные обороты речи
   - Эмодзи и форматирование
```

### Шаг 3 — Трансформация текста
```
Трансформируй следующий пост в стиль [АВТОР], используя его характерные приёмы:

Исходный пост: [ВСТАВИТЬ ПОСТ]

Требования:
1. Сохрани ключевое сообщение и факты
2. Используй характерные обороты речи автора
3. Сохрани структуру аргументации автора
4. Добавь типичные для автора эмоциональные триггеры
```

### Шаг 4 — Проверка качества стилизации
```
Оцени качество стилизации поста по критериям:
1. Сохранение ключевого сообщения
2. Соответствие стилю автора (структура, языковые паттерны, триггеры)
3. Естественность текста
4. Уместность стилизации

Исходный пост: [ВСТАВИТЬ]
Стилизованный пост: [ВСТАВИТЬ]

Дай оценку по каждому критерию и предложи улучшения.
```

## Примеры стилей из практики

### Стиль Лебедева
Для клиентов, которым нужна жёсткая экспертная позиция.

| Элемент | Как проявляется |
|---|---|
| Тон | Прямолинейный, без смягчений и дипломатии |
| Оценки | Категоричные, с позиции эксперта |
| Ирония | Сарказм как инструмент критики |
| Структура | Разбор через конкретные примеры «как есть / как надо» |
| Фишки | «Объясняю на пальцах...», короткие рубящие фразы, «полная ерунда» |

### Стиль Курпатова
Для клиентов, которым нужна экспертность без давления.

| Элемент | Как проявляется |
|---|---|
| Тон | Спокойный, научный, без агрессии |
| Структура | Логическая, разрушение иллюзий через объяснение механизмов |
| Язык | Сложное — простым, через жизненные примеры |
| Фишки | «Давайте разберёмся...», «На самом деле всё устроено так...» |
| Финал | Экзистенциальный инсайт о мышлении или природе сознания |

**Готовый промт для стиля Курпатова:**
```
Роль: Эксперт-психолог, объясняющий сложные вещи простым языком без давления

Задача: Написать пост о [тема] в стиле Курпатова:
- Объяснение через научные механизмы, без терминологии
- Разрушение типичного заблуждения по теме
- Логичная структура: иллюзия → реальность → механизм → вывод
- Финал: инсайт о природе мышления или выборе

Особенности:
- Начало: «Мы привыкли думать, что...»
- Разворот: «Но мозг устроен иначе. На самом деле...»
- Финал — всегда экзистенциальный, не прикладной
```

## Метод 2: Создание цифровой копии клиента (его голос)

Когда клиент хочет, чтобы тексты звучали как он сам. Три шага.

### Шаг 1 — Анализ существующих материалов
```
Проанализируй мои последние посты / переписки / заметки: [материалы]

Выдели:
1. Характерные обороты речи и слова-маркеры
2. Типичную структуру мысли
3. Любимые темы и типы примеров
4. Эмоциональные паттерны (юмор, серьёзность, ирония)
```

### Шаг 2 — Создание шаблона стиля
```
На основе анализа создай шаблон для написания текстов в моём стиле.

Включи:
1. Типичные открывающие фразы
2. Характерные связки между абзацами
3. Любимые обороты и словесные конструкции
4. Особенности форматирования
5. Часто используемые эмодзи и символы (если есть)
```

### Шаг 3 — Генерация в стиле клиента
```
Напиши пост о [тема] в стиле [имя клиента], используя:
1. Его шаблон стиля
2. Характерные обороты из анализа
3. Типичную структуру

Тема: [тема]
Длина: [знаки]
Цель: [продажа / экспертность / вовлечение]
```

## Метод 3: Tone of Voice из транскрипций (из Урока 14) — лучший способ

Настоящий голос — это то, как человек **говорит**, а не пишет. Его можно поймать.

**Источники:** голосовые из Telegram, записи созвонов, аудио-заметки — всё, где человек говорит вживую, объясняет, рассуждает, делится мыслями.

**Лайфхак от тренера:** ведите аудиодневники. Каждое утро записывайте планы на день, каждый вечер — ретроспективу: что произошло, какие инсайты, что не получилось. Даже если не публикуете — транскрибируйте и отправьте в Claude для анализа ToV. Это самый чистый ваш голос — без фильтров и редактуры.

**Процесс:**
1. Собрать 5-10 аудио-фрагментов
2. Транскрибировать (любой ИИ справится за минуту)
3. Загрузить в Claude с промптом ниже

**Промпт:**
```
Ниже — транскрипция моих голосовых сообщений / записей созвонов / аудио-заметок.
Это мой живой разговорный стиль — как я говорю по-настоящему, без редактуры.

[ВСТАВИТЬ ТРАНСКРИПЦИЮ]

Проанализируй мой Tone of Voice на основе этого текста:

1. ЛЕКСИКА — какие слова использую чаще всего? Есть ли слова-паразиты, которые делают меня узнаваемым? Какие обороты повторяются?
2. РИТМ РЕЧИ — длинные предложения или короткие? Говорю плавно или рублено? Делаю паузы через «ну», «значит», «вот»?
3. СПОСОБ ОБЪЯСНЯТЬ — через аналогии? Через примеры? Через цифры? Через истории?
4. ЭМОЦИОНАЛЬНЫЙ ТОНУС — насколько эмоционален? Сдержан или экспрессивен? Уверен или сомневается вслух?
5. ОТНОШЕНИЕ К ЧИТАТЕЛЮ — как обращаюсь? Учу? Делюсь? Рассуждаю вместе?

На основе анализа создай:
- Описание моего Tone of Voice (5–7 предложений)
- 5 правил написания текстов в моём стиле
- 3 пары примеров: «я бы НЕ написал так» vs «я бы написал вот так»

Сохрани в tone-of-voice.md.
Этот документ я буду прикреплять к каждому запросу на написание постов.
```

> **Почему это лучший способ:** ИИ вытащит настоящие паттерны — любимые слова, ритм фраз, способ аргументировать, интонации. Это настоящий Tone of Voice — без фильтров.

## Как найти стиль клиента (если нет соцсетей)

Три варианта — от простого к глубокому:

**Вариант 1 — Переписки:**
Попроси клиента сделать 10 скриншотов деловых переписок. Загрузи в ChatGPT или Claude. Попроси проанализировать стиль общения, характерные фразы и интонацию.

**Вариант 2 — Разговор на диктофон:**
Запиши любой рабочий разговор или объяснение своего дела. Транскрибируй (через Whisper, Notion AI или ручной сервис). Дай AI на анализ.

**Вариант 3 — AI-интервью:**
```
Сыграй роль опытного интервьюера. Задай мне 10 нестандартных, глубоких и провокационных вопросов, чтобы раскрыть мой стиль мышления, подачи информации и способ выражения мыслей.

Формат — один вопрос за другим, без оценок и комментариев.
```
Ответы записать на диктофон → транскрибировать → анализ.

## Универсальный промт для глубокого анализа стиля

Используй когда нужен полный «стиль-профиль» клиента.

```
Ты — эксперт по анализу речевых и письменных коммуникаций с 10-летним стажем. Умеешь по любому текстовому источнику (переписки, аудио, видео, заметки) выявить стиль общения, характерные обороты, метафоры и тональность. Твоя суперсила — превращать неосознанный стиль в структурированный и воспроизводимый речевой образ.

Проанализируй предоставленный источник: [тип источника — скриншоты / транскрибация / заметки / документ]

Задачи:

1. Определи особенности стиля:
   - Структура подачи мыслей
   - Манера общения (формально, дружелюбно, с юмором, наставнически)
   - Повторяющиеся фразы, вводные слова, клише
   - Слова-связки и логические переходы
   - Эмоциональный тон и характер подачи

2. Выдели метафоры и образные элементы:
   - Типы образов (визуальные, бытовые, философские)
   - Насколько устойчивы и узнаваемы
   - Какие смыслы через них передаются

3. Сформируй «стиль-профиль»:
   - 1–2 абзаца с описанием стиля
   - Список характерных выражений и речевых конструкций
   - Перечень метафор и аналогий
   - Рекомендации по воспроизведению стиля

4. Создай шаблон генерации текста в этом стиле:
   - Поле для темы
   - Поле для примера/кейса
   - Поле для метафоры
   - Поле для финального вывода

Формат результата:
- Аналитический отчёт по пунктам
- Таблицы/списки фраз и речевых паттернов
- Стиль-профиль
- Шаблон для генерации поста или сценария

Прежде чем начать — запроси у меня все необходимые материалы.
```

## Метод 4: Анализ стиля за 5 шагов (из методички «Нейроклон»)

Более структурированный метод чем Метод 1. Даёт готовый гайд «Как писать в стиле [имя]».

```
Ты — эксперт по анализу стиля письма с более чем 10-летним опытом работы с известными писателями и копирайтерами.

Задача: Проанализируй мои тексты и создай подробный профиль моего авторского стиля.

Вот примеры моих текстов:
[ВСТАВЬ 3-5 ЛУЧШИХ ТЕКСТОВ РАЗНЫХ ФОРМАТОВ]

Шаг 1: Проанализируй структуру моих текстов (длина предложений, абзацев, использование списков).
Шаг 2: Выдели характерные обороты речи, словесные конструкции и фразы.
Шаг 3: Определи мой эмоциональный тон и стиль аргументации.
Шаг 4: Опиши 5-7 ключевых особенностей, которые делают мой стиль уникальным.
Шаг 5: Создай краткий гайд «Как писать в стиле [ИМЯ]».

Результат представь в структурированном виде с разделами и примерами.
```

> **Важно:** сохрани результат анализа как файл `tone-of-voice.md` в проект — тогда не нужно обучать заново в каждом чате.

## Бонус: Переделать чужой текст в свой стиль

Берёшь чужой текст (статью, пост конкурента, новость) и превращаешь в свой голос. Идеи и информация сохраняются, звучание — твоё.

```
Ты — редактор с 15-летним опытом. Переделай этот текст, чтобы он звучал так, как будто его написал я, сохраняя все идеи и информацию:

[ВСТАВЬ ТЕКСТ]

Используй мои характерные фразы, стиль построения предложений и эмоциональный тон из tone-of-voice.md.
```

## Проверка уникальности контента

Когда текст готов — проверяй по четырём критериям:

```
Оцени уникальность этого поста:
1. Оригинальность подачи (1–10)
2. Специфичность примеров (1–10)
3. Яркость метафор (1–10)
4. Запоминаемость (1–10)

Предложи конкретные улучшения по каждому пункту.
```

Если средний балл ниже 7 — прогони через технику инверсии из [techniques.md](./techniques.md): «Что делает этот текст банальным?»

# Типичные ошибки при работе с ИИ

Список ошибок из практики — что мешает получить хороший результат и как это исправить.

---

## Ошибка 1: Многозначность промта

**Что происходит:** Запрос можно интерпретировать по-разному — AI выбирает один из вариантов, не тот что нужен.

**Пример плохого промта:**
```
Напиши про наш продукт что-нибудь интересное для клиентов
```

Что значит «интересное»? Для каких клиентов? Какой продукт? AI додумывает сам.

**Как исправить:** Конкретизируй каждое расплывчатое слово.
```
Напиши пост-анонс нового курса по SMM для малого бизнеса.
Аудитория: владельцы кафе и магазинов, 30-45 лет.
Акцент: можно учиться без опыта, первый результат за 2 недели.
```

---

## Ошибка 2: Одно окно — разные задачи

**Что происходит:** Пользователь в одном чате ставит разнородные задачи. Контекст предыдущих сообщений путает AI и снижает качество.

**Пример:**
```
[Раньше в чате] Напиши рецепт борща
[Потом] Напиши коммерческое предложение для клиента
```

AI может «помнить» предыдущий контекст и это влияет на результат.

**Как исправить:** Для каждой новой задачи — новый чат. Или используй системный промт в начале чата, который задаёт роль и контекст для всей сессии.

---

## Ошибка 3: Первый результат сразу в работу

**Что происходит:** Пользователь берёт первый ответ AI без проверки и правок. Ошибки, неточности и несоответствия попадают в финальный материал.

**Правило:** Первый результат — это черновик. Всегда.

**Как исправить:**
1. Проверь по 4 критериям (соответствие задаче, полнота, формат, тон)
2. Сделай хотя бы одну итерацию уточнения
3. Перечитай финальный текст глазами читателя, не автора

---

## Ошибка 4: Нет роли — нет экспертизы

**Что происходит:** Без указания роли AI даёт общий, поверхностный ответ. Не хватает глубины и специфики.

**Пример:**
```
Напиши текст о пользе йоги
```

**Как исправить:** Дать конкретную роль с опытом и специализацией.
```
Ты инструктор по йоге с 12-летним опытом, работаешь с людьми после 40,
специализируешься на восстановлении после травм позвоночника...
```

---

## Ошибка 5: Слишком длинный промт без структуры

**Что происходит:** Пользователь пишет всё в одном абзаце. AI теряет часть требований или игнорирует менее заметные пункты.

**Пример:**
```
Напиши пост для инстаграм о нашем курсе по дизайну для начинающих которые хотят работать
удалённо пиши вдохновляюще и с примерами успеха и чтобы было не длинно и призыв к действию
и чтобы были хэштеги и не забудь упомянуть что курс онлайн и начинается в мае
```

**Как исправить:** Структурируй промт по CRAFT — отдельными блоками с заголовками.

---

## Ошибка 6: Нет формата — AI выбирает сам

**Что происходит:** Без указания формата AI даёт результат в том виде, который считает подходящим. Может быть слишком длинно, слишком коротко, не та структура.

**Как исправить:** Всегда указывай:
- Тип контента (пост / статья / письмо)
- Объём (знаки, слова, абзацы)
- Структуру (какие разделы и в каком порядке)

---

## Ошибка 7: Противоречивые требования

**Что происходит:** В промте есть взаимоисключающие условия. AI пытается угодить всем сразу — результат получается компромиссным и слабым.

**Примеры противоречий:**
- «Напиши подробно, но коротко»
- «Формальный тон, но дружелюбный»
- «Продающий текст без призыва к покупке»

**Как исправить:** Выбери приоритет и укажи его явно.
```
Тон: дружелюбный (важнее, чем формальность)
Объём: до 800 знаков (краткость важнее полноты)
```

---

## Ошибка 8: Нет критериев успеха

**Что происходит:** AI не знает, что считается хорошим результатом. Угадывает.

**Как исправить:** Добавь критерии прямо в промт:
```
Хороший результат — если читатель понимает главное за 10 секунд
и хочет нажать на ссылку.
```

---

## Быстрая диагностика

Если результат плохой — задай себе вопросы:

| Проблема | Вероятная причина |
|----------|-------------------|
| Слишком общо, нет глубины | Нет роли |
| Не то, что просил | Многозначный промт |
| Не тот формат | Не указан формат |
| Не тот тон | Не указан тон |
| Упустил часть требований | Всё в одном абзаце |
| Странный результат | Накопленный контекст чата |

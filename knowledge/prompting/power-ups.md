# Фишки усиления промтов

Добавки к основному промту, которые повышают качество результата. Работают поверх CRAFT — используй когда нужен результат выше среднего.

---

## 1. Шкала качества

**Принцип:** Задаёшь конкретный числовой стандарт. AI понимает, что черновик — это не финал.

**Базовый шаблон:**
```
Оцени каждый [абзац / компонент / блок] по шкале 1–10.
Переписывай пока не будет минимум 9/10.
```

**Для контента:**
```
Оцени каждый абзац по шкале 1–10 по критериям:
конкретика, вовлечённость, соответствие аудитории.
Всё что ниже 8 — перепиши.
```

**Для сайтов и лендингов:**
```
Оцени каждый блок лендинга по шкале 1–10:
насколько он закрывает возражение и ведёт к следующему шагу.
Переработай блоки с оценкой ниже 8.
```

**Для кода (вайб-кодинг):**
```
Оцени этот код по шкале 1–10 по трём критериям:
читаемость, надёжность, простота поддержки.
Перепиши части с оценкой ниже 8 и объясни что изменил.
```

## 2. Угроза / Ставки

**Принцип:** Добавляешь контекст важности. AI переключается в режим "это должно быть хорошо".

**Базовый шаблон:**
```
Этот [текст / код / дизайн] идёт на [важный контекст].
Проверь так, будто от этого зависит успех всего проекта.
```

**Для контента:**
```
Этот пост — главный для запуска продукта. Он увидит 50 000 человек.
Найди все слабые места прежде чем я его опубликую.
```

**Для сайтов:**
```
Этот лендинг запускается с рекламным бюджетом 100 000 рублей.
Каждый слабый блок — слитые деньги. Найди все слабые места.
```

**Для кода:**
```
Этот код пойдёт в продакшн и будет использоваться 1000 пользователями в день.
Найди все потенциальные баги и узкие места прежде чем я деплою.
```

## 3. Денежная мотивация

**Принцип:** Создаёт образ высококвалифицированного специалиста с высокой ставкой.

**Базовый шаблон:**
```
You're a $1000/hour [роль]. [Задача].
```

**Для контента:**
```
You're a $1000/hour copywriter. Analyze this text and rewrite it
to maximize conversion. Explain every change you make.
```

**Для сайтов:**
```
You're a $1000/hour UX designer. Review this landing page structure
and identify the top 3 conversion killers.
```

**Для кода (вайб-кодинг):**
```
You're a $1000/hour senior developer doing a code review.
Find every place where this code could be improved: performance,
readability, edge cases. Be brutal.
```

**Для стратегии:**
```
You're a $1000/hour marketing consultant. I'll give you my business.
Tell me the 3 highest-leverage things I should do this month.
```

## 4. Английский язык

**Принцип:** На английском AI имеет более богатую базу обучения. Для технических задач, аналитики и кода — часто даёт более точные и глубокие результаты.

**Когда переключаться на английский:**
- Технические задачи (код, архитектура, баги)
- Аналитика и стратегия
- Когда нужна максимальная детализация

**Когда оставаться на русском:**
- Тексты для русскоязычной аудитории
- Контент для соцсетей и сайтов
- Работа с клиентскими материалами

**Пример — разработка:**
```
Write a JavaScript function that [описание].
Requirements:
- Handle edge cases: [список]
- Return: [что возвращает]
- Error handling: [как обрабатывать ошибки]
Add comments only where logic is non-obvious.
```

**Пример — анализ:**
```
Analyze this landing page copy and identify:
1. Top 3 conversion killers
2. Missing trust signals
3. Unclear value proposition
Be specific, not general.
```

## 5. Поиск актуальных данных

**Принцип:** Просишь AI найти свежие данные перед созданием контента. Добавляет конкретику и актуальность.

**Базовый шаблон:**
```
Перед [задача] найди актуальные данные: [что искать].
Используй эти данные в результате — с цифрами и источниками.
```

**Для контента:**
```
Перед написанием статьи найди:
- Последние тренды в [ниша] за 2024–2025 год
- 2–3 интересных статистических факта
- Актуальные боли аудитории по теме
Используй это в тексте — с конкретными цифрами.
```

**Для лендингов:**
```
Найди данные о рынке [ниша]:
- Средние показатели конверсии
- Типичные возражения аудитории
- Что сейчас работает у конкурентов
Учти это при создании структуры лендинга.
```

**Для продуктовых решений:**
```
Перед тем как предложить архитектуру [инструмент]:
найди, как похожие инструменты решают эту задачу.
Что работает хорошо? Что является типичной проблемой?
```

## Готовые комбинации

### Комбинация 1: Шкала + Угроза
Для финальной проверки любого результата перед публикацией:
```
Этот [текст/код/страница] идёт в публикацию завтра.
Оцени каждый элемент по шкале 1–10. Всё ниже 8 — перепиши.
```

### Комбинация 2: Деньги + Английский
Для глубокого code review или аналитики:
```
You're a $1000/hour senior developer.
Review this code. Find every issue. Prioritize by impact.
```

### Комбинация 3: Данные + Шкала + Инверсия (финальный микс)
Для создания сильного контента с нуля:
```
Найди свежие данные по рынку [ниша].
Затем: почему контент конкурентов в этой теме не работает?
Напиши пост, который закрывает все эти проблемы.
Оцени каждый абзац 1–10 и доведи до минимум 9.
```

---

## Когда какую фишку применять

| Задача | Рекомендованные фишки |
|---|---|
| Пишешь важный продающий текст | Угроза + Шкала |
| Делаешь code review | Деньги + Английский |
| Создаёшь контент в новой нише | Данные + Шкала |
| Нужна глубокая аналитика | Деньги + Английский |
| Финальная проверка перед публикацией | Угроза + Шкала |
| Технические задачи (код, архитектура) | Английский + Деньги |

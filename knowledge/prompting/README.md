## Как пользоваться

**Для агентов при создании текстов и контента:**
1. Прочитай [craft-formula.md](./craft-formula.md) — применяй структуру к задаче
2. Если задача сложная — используй [techniques.md](./techniques.md): Chain Prompting или Инверсию
3. Для постов — выбери тип по [../standards/post-types.md](../standards/post-types.md)
4. Если нужен стиль клиента — используй [style-copy.md](./style-copy.md)
5. Посмотри [examples.md](./examples.md) — возьми подходящий пример как основу
6. Усиль по [power-ups.md](./power-ups.md) — добавь одну-две фишки
7. Улучши по [improvement.md](./improvement.md) — прогони через итерации если нужно

**Для Инны при добавлении новых материалов:**
1. Добавь урок/источник в [lesson-log.md](./lesson-log.md)
2. Если нужен новый файл — создай и обнови этот README
3. Если знание вписывается в существующий файл — дополни его

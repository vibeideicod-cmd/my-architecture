# База знаний: Промтинг

Раздел с накопленными знаниями о том, как составлять эффективные запросы к ИИ.

## Файлы раздела

| Файл | Что внутри |
|------|-----------|
| [craft-formula.md](./craft-formula.md) | Универсальная формула промта CRAFT — 5 элементов с примерами |
| [examples.md](./examples.md) | Банк примеров: плохой / обычный / отличный промт |
| [improvement.md](./improvement.md) | Техники улучшения: итерации, критерии оценки |
| [mistakes.md](./mistakes.md) | Типичные ошибки при работе с ИИ |
| [lesson-log.md](./lesson-log.md) | Лог обработанных обучающих материалов |

## Как пользоваться

**Для агентов при создании текстов и контента:**
1. Прочитай [craft-formula.md](./craft-formula.md) — применяй структуру к задаче
2. Посмотри [examples.md](./examples.md) — возьми подходящий пример как основу
3. Улучши по [improvement.md](./improvement.md) — прогони через итерации если нужно

**Для Инны при добавлении новых материалов:**
1. Добавь урок/источник в [lesson-log.md](./lesson-log.md)
2. Если нужен новый файл — создай и обнови этот README
3. Если знание вписывается в существующий файл — дополни его

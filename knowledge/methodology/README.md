# Методология СССР

**Структура · Стратегия · Свобода · Результат**

Полный цикл работы с клиентом — от первого контакта до масштабирования. 7 этапов, каждый является фундаментом для следующего.

---

## Ключевые принципы

1. **Последовательность** — нельзя перескакивать этапы
2. **Документирование** — все решения обоснованы данными
3. **Итеративность** — система циклична, постоянно улучшается
4. **Комплексность** — всё связано, ничего фрагментарно
5. **Data-driven** — решения на основе данных, не догадок

## Чего мы принципиально не делаем

- Не начинаем с тактики — сначала стратегия
- Не работаем без данных — сначала аналитика
- Не обещаем быстрых результатов — это система
- Не делаем "просто контент" — всегда с целью
- Не игнорируем обратную связь — всегда анализируем

## Связи между этапами

```
Этап 0 (Вход)
    ↓ квалифицированный лид
Этап 1 (Диагностика)
    ↓ структурированная база фактов
Этап 2 (Бренд-база)
    ↓ цифровой паспорт бизнеса
Этап 3 (Аналитика)
    ↓ инсайты по рынку и аудитории
Этап 4 (Стратегии)
    ↓ план действий + KPI
Этап 5 (Реализация)
    ↓ контент + кампании
Этап 6 (Публикация → Реакция → Обработка)
    ↓ данные для оптимизации
    ↩ возврат на Этап 3, 4 или 5 — новый цикл
```

---

## Артефакты системы

На каждом этапе создаются:
- **Документы** — стратегии, отчёты, планы
- **Базы данных** — клиенты, контент, реакции
- **Шаблоны** — брифы, ТЗ, скрипты
- **Библиотеки** — визуал, тексты, видео

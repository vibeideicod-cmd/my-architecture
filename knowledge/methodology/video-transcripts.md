# Получение текста из видео — иерархия инструментов

Когда нужен транскрипт видео (YouTube-референс конкурента, видео клиента для аналитики, обучающий ролик для `learnings/`) — иди по этой иерархии от бесплатного к платному.

## Уровень 1 — YouTube субтитры через WebFetch (бесплатно)

Большинство YouTube-видео имеют автоматические субтитры (auto-generated). Их можно вытащить без API.

**Когда подходит:**
- Видео с YouTube
- Есть субтитры (проверь иконку «CC» в плеере)
- Качество автосубтитров устраивает (для русского — в целом нормально)

**Как:**
1. Через WebFetch — попроси Claude вытащить субтитры по URL видео
2. Альтернатива — `yt-dlp --write-sub --skip-download <url>` если есть на маке

**Когда не подходит:**
- Субтитров нет (короткие видео часто без них)
- Субтитры на неподдерживаемом языке без перевода
- Нужна точность по таймкодам

## Уровень 2 — Через Claude 20 в браузере (`inbox-cl20`)

У нас есть процесс — Claude 20 в браузере умеет смотреть YouTube и пересказывать. Кладёт MD-итог в `inbox-cl20/`.

**Когда подходит:**
- Короткие видео (до 30 минут)
- Нужен пересказ, не точная транскрипция
- Быстро, без точности по словам
- Для общего понимания контекста

**Как:**
1. Дай Claude 20 ссылку с задачей «посмотри это видео и сделай конспект»
2. Через `git pull` забираешь результат из `inbox-cl20/`

**Когда не подходит:**
- Нужна точная транскрипция слово-в-слово
- Длинные видео (> 30 минут — Claude 20 может пропустить детали)
- Видео не на YouTube

## Уровень 3 — Deepgram REST API / Supadata MCP (платно, для регулярной работы)

**Сейчас в холодильнике плана Жорика** — в `zhorik-decomposition-plan.md` раздел «Регулярная обработка видео». Готовые curl-команды Deepgram уже там, разморозим при сборке скилла `/client-onboarding-audio` (после Этапа 6 разбора Жорика).

**Триггер разморозки:**
- Появился пайплайн обработки аудиозаписей нового клиента (см. раздел «Работа с новым клиентом Инны» в [agents/director.md](../../agents/director.md))
- Регулярная задача транскрибации видео клиентов
- Нужна точность по таймкодам или диаризация (кто что говорит)

**Стоимость когда разморозим:**
- Deepgram — Free $200 кредитов (без банковской карты), $0.0043/мин для Nova-2 модели
- Supadata MCP — Free 100 credits/мес, $5+/мес платный

**Подключение:** ключ — в Связку ключей macOS под именем `deepgram-api-key` (по правилам [agents/director.md](../../agents/director.md) раздел «Работа с секретами»).

## Принцип выбора

```
Видео на YouTube + есть субтитры → Уровень 1 (WebFetch)
Видео на YouTube + нет субтитров + короткое → Уровень 2 (Claude 20)
Видео не на YouTube + длинное + регулярно → разморозить Уровень 3 (Deepgram/Supadata)
```

## Длинные транскрипты

Если результат > 5000 слов — сохраняй в `clients/<имя>/transcripts/<дата>-<тема>.md` (для клиента) или `knowledge/learnings/<тема>/transcript-<имя>.md` (для обучения).

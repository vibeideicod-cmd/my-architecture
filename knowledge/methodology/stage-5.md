# Этап 5: Реализация и производство

**Назначение:** Создание контента и реализация стратегий

---

## Контент-производство

**Процесс (всегда этот порядок):**
1. **Идея** — из контент-плана
2. **Бриф** — ТЗ для создания
3. **Создание** — написание / дизайн / съёмка
4. **Редактура** — проверка качества
5. **Согласование** — утверждение клиентом
6. **Публикация** — размещение

**Типы контента:**
- Тексты (посты, статьи, рассылки)
- Визуал (фото, графика, видео)
- Аудио (подкасты, голосовые)
- Интерактив (опросы, тесты, квизы)

## Инструменты производства

| Задача | Инструменты |
|---|---|
| Дизайн | Canva, Figma |
| Видео | CapCut, Premiere |
| Тексты | Claude, ChatGPT |
| Управление | Trello, Notion |
| Генерация иллюстраций | Midjourney, Leonardo, Dalle, Kling |
| Автопостинг | Make (Integromat) |

## Реализация по направлениям

Для каждого направления — настройка, запуск, оптимизация, отчётность:

- Контент-маркетинг
- Таргетированная реклама
- Контекстная реклама
- Email-маркетинг
- SMM
- SEO

## Дистрибуция контента

- По расписанию из контент-плана
- Адаптация под формат каждого канала
- Органический охват + платное продвижение
- Кросс-постинг (где уместно)

## Взаимодействие с аудиторией

- Ответы на комментарии
- Работа с возражениями
- Поддержка диалога
- Сбор обратной связи

## Контроль качества на этапе

- Соответствие стратегии
- Качество контента
- Технические ошибки
- Соответствие брендбуку

## Результаты этапа

- **Созданный контент** (посты, видео, статьи)
- **Запущенные кампании** (реклама, рассылки)
- **Отчёты** (что сделали, какие результаты)
- **База контента** (библиотека материалов)

## Чего НЕ делаем

- ❌ Не отклоняемся от стратегии без обоснования
- ❌ Не публикуем без проверки
- ❌ Не игнорируем метрики

## Метрика успеха

Объём и качество созданного контента, соответствие контент-плану.

# Этап 0: Вход клиента (Pre-Sales)

**Назначение:** Первичный контакт и квалификация лида

---

## Источники лида

- **Органика** — сарафан, соцсети, рекомендации
- **Платные каналы** — контекст, таргет, influencers
- **Партнёрства** — кросс-промо
- **Контент-маркетинг** — блог, YouTube, подкасты

## Точки входа

- Личное сообщение (прямое обращение)
- Заявка на сайте / лендинге
- Отклик на пост / сторис
- Звонок по рекомендации
- Email после контента

## Типы входящих

| Тип | Суть |
|---|---|
| "Хочу то, не знаю что" | Нужна диагностика |
| "Нужен SMM / контекст" | Запрос на тактику |
| "Хочу стратегию" | Осознанная потребность |
| "Рекомендовали" | Тёплый лид |

## Результаты этапа

- **Квалификация лида** (подходит / не подходит)
- **Первичный бриф** (понимание задачи)
- **Назначение Zoom** (встреча для диагностики)
- **Отказ** (не наш клиент)

## Важно

На этом этапе **НЕ проводится** глубокая аналитика. Только сбор первичной информации и квалификация.

## Метрика успеха

Конверсия из входящего обращения в квалифицированный лид (назначенный Zoom).

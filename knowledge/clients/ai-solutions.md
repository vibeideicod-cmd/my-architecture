# AI-решения для клиентов

Практическая карта: какую проблему клиента можно закрыть с помощью AI. Используй при брифинге и формировании предложения.

---

## Карта: проблема → AI-решение

| Проблема клиента | AI-решение | Инструмент |
|---|---|---|
| Нет времени вести соцсети | AI пишет посты и контент-план | Claude + шаблоны |
| Заявки теряются, отвечают медленно | AI-саппорт в Telegram 24/7 | Telegram Bot + GPT API |
| Не знают свою аудиторию | AI-анализ ЦА и конкурентов | Claude / Perplexity |
| Нет рекламных материалов | AI генерирует картинки и тексты | Midjourney + Claude |
| Нет времени снимать контент | AI создаёт визуал без фотосессии | DALL-E / Midjourney |
| Хотят продавать без звонков | AI-продавец квалифицирует лиды | Telegram Bot + AI |
| Не понимают, что делают конкуренты | AI-анализ конкурентов | Claude + Perplexity |
| Нет сил на рутинные ответы | AI отвечает на типовые вопросы | Бот + FAQ-база |

## По типам клиентов

### Санаторий / оздоровительный комплекс

**Что предлагать:**
- **AI-саппорт** — бот отвечает на вопросы о процедурах, ценах, расписании 24/7
- **AI-контент** — еженедельные посты о здоровье, процедурах, историях гостей
- **AI-анализ** — исследование ЦА: кто едет, зачем, что ищет
- **AI-визуал** — атмосферные картинки для соцсетей без фотографа

**Типичный первый шаг:** AI-бот для записи + ответов на FAQ

---

### Магазин (розница)

**Что предлагать:**
- **AI-продавец** — бот рассказывает о товарах, принимает заявки в Telegram
- **AI-контент** — посты с описаниями товаров, акциями, сравнениями
- **AI-визуал** — красивые карточки товаров без профессиональной съёмки
- **AI-анализ конкурентов** — что продают, по каким ценам, как позиционируют

**Типичный первый шаг:** AI-контент для Telegram-канала + карточки товаров

---

### Сервисный бизнес (ремонт, beauty, клининг, авто и др.)

**Что предлагать:**
- **AI-саппорт** — первичный ответ клиенту, квалификация заявки, передача мастеру
- **AI-отзывы** — помощь в сборе и обработке отзывов, ответы на негатив
- **AI-контент** — кейсы "до/после", советы, ответы на типовые вопросы
- **AI-воронка** — от первого сообщения до записи без участия человека

**Типичный первый шаг:** AI-бот для приёма заявок с передачей в CRM/таблицу

## Готовые AI-продукты

### AI-саппорт
**Что это:** Telegram-бот который отвечает на вопросы клиентов, используя базу знаний о бизнесе.
**Когда предлагать:** клиент теряет заявки, долго отвечает, устаёт от одних и тех же вопросов.
**Что нужно от клиента:** список частых вопросов + ответы, описание услуг/продуктов.

### AI-маркетолог
**Что это:** регулярная генерация контента (посты, идеи, тексты) под нишу клиента.
**Когда предлагать:** нет времени/идей для соцсетей, контент выходит редко и хаотично.
**Что нужно от клиента:** описание бизнеса, аудитории, 2-3 примера того, что нравится.

### AI-аналитик
**Что это:** разовое или регулярное исследование: ЦА, конкуренты, тренды в нише.
**Когда предлагать:** клиент не понимает свою аудиторию или хочет выйти в новый сегмент.
**Что нужно от клиента:** описание бизнеса и вопросы на которые ищет ответы.

### AI-дизайнер (базовый)
**Что это:** генерация изображений для постов, рекламы, оформления каналов.
**Когда предлагать:** нет бюджета на фотографа или дизайнера для рутинного контента.
**Что нужно от клиента:** стиль, цвета бренда, примеры что нравится.

## Что AI не может — для честного разговора с клиентом

| Запрос клиента | Реальность |
|---|---|
| «AI сам будет вести весь маркетинг» | AI помогает, но нужен человек для контроля и стратегии |
| «AI примет решение за меня» | AI даёт варианты и анализ — решение за человеком |
| «AI сделает фото наших товаров» | AI генерирует изображения, но не реальные фото продукта |
| «Бот будет продавать сам» | Бот квалифицирует и отвечает — сложные продажи закрывает человек |
| «AI обучит другой AI» | Нет — AI создаёт промпты, но не обучает модели |

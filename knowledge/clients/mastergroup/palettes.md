# МГ-платформа — цветовая система

> Это **не** брендбук «Нейро Бабки СССР». МГ-платформа — продукт-конструктор для мастеров мастер-групп. У неё собственный визуальный язык, подобранный под аватарку сообщества «Рабочая мастер-группа» и тестирование Ирины. Внутренний брендбук команды здесь **не применяется** (см. `memory/feedback_brand_scope.md`).

Источник палитр — подбор Ирины через сервис AWSMCOLOR (три файла в `inbox-inna/archive/` от 2026-04-19).
Источник основной палитры — аватарка сообщества «Рабочая мастер-группа» (розово-фиолетовая с радужным градиентом).

---

## Базовая палитра платформы — «Сакура» (palette #3)

Применяется **на всех страницах МГ-платформы** кроме `admin.html` и `guide-admin.html`:
- `index.html` (лендинг)
- `apply.html` (анкета «Стать мастером»)
- `build.html` (конструктор)
- `guide-master.html` (гайд мастера)

По умолчанию также на `master-page.html` (если мастер не выбрал другую палитру).

**Почему именно Сакура:** доминанта аватарки сообщества — малиново-розовый (~`#D62A8E`) и фиолетовый градиент. Сакура совпадает по температуре и оттенку, создаёт единое визуальное поле с самой мастер-группой. Океан и Красно-жёлтая рядом с аватаркой конфликтуют.

### CSS-переменные

```css
:root {
  --mg-deep:      #2E2249;  /* глубокий фиолет — заголовки, тёмные акценты */
  --mg-accent:    #F64A8A;  /* розовый — CTA, ссылки, активные состояния */
  --mg-bg:        #FAF8F0;  /* почти-белый тёплый — основной фон */
  --mg-soft:      #DFCBDB;  /* сиреневый — карточки, секции, мягкие фоны */

  /* Функциональные */
  --mg-text:       #2E2249;  /* основной текст — глубокий фиолет */
  --mg-text-soft:  rgba(46, 34, 73, 0.65);
  --mg-text-hint:  rgba(46, 34, 73, 0.45);
  --mg-white:      #FFFFFF;
}
```

### Роли цветов

| Цвет | Где используется |
|---|---|
| `#2E2249` | заголовки, основной текст, бордеры CTA |
| `#F64A8A` | главный CTA (кнопка публикации, «отправить», «связаться»), ссылки, активные поля |
| `#FAF8F0` | фон страницы |
| `#DFCBDB` | фон карточек, блоков, вторичные секции |
| `#FFFFFF` | фон форм, input-полей, контрастные поверхности |

### CTA-кнопка (пример)

```css
.mg-cta {
  background: var(--mg-accent);
  color: #fff;
  border: 0;
  border-radius: 50px;
  padding: 16px 32px;
  font-weight: 700;
  letter-spacing: 1.2px;
  text-transform: uppercase;
  box-shadow: 0 6px 22px rgba(246, 74, 138, 0.35);
  transition: transform .15s, box-shadow .2s;
}
.mg-cta:hover {
  transform: translateY(-2px);
  box-shadow: 0 10px 30px rgba(246, 74, 138, 0.5);
}
```

## Палитры-пресеты для мастера (на странице `master-page.html`)

В конструкторе ([build.html](../../../clients/mastergroup/mvp/build.html)) мастер выбирает один из **3 пресетов** — эта палитра применяется на его публичной странице. Все три — из подбора Ирины.

### Пресет 1 — «Океан»

```css
[data-palette="ocean"] {
  --page-deep:   #2F404F;  /* тёмно-синий графит */
  --page-accent: #3894A1;  /* бирюзовый — CTA */
  --page-bg:     #F0F1EE;  /* почти белый */
  --page-soft:   #C7DAD3;  /* мятный — карточки */
}
```

Подходит для: спокойных, профессиональных ниш — финансы, образование, консультирование, коучинг.

### Пресет 2 — «Красно-жёлтая»

```css
[data-palette="red-yellow"] {
  --page-deep:   #CF0A10;  /* красный — заголовки/акценты */
  --page-accent: #F7BC13;  /* жёлтый — CTA */
  --page-bg:     #F6EFE3;  /* кремовый */
  --page-soft:   #88D9CF;  /* мятный — карточки */
}
```

Подходит для: энергичных, эмоциональных ниш — творчество, обучение, мотивация, продажи.

### Пресет 3 — «Сакура»

```css
[data-palette="sakura"] {
  --page-deep:   #2E2249;  /* глубокий фиолет */
  --page-accent: #F64A8A;  /* розовый — CTA */
  --page-bg:     #FAF8F0;  /* почти белый */
  --page-soft:   #DFCBDB;  /* сиреневый — карточки */
}
```

Подходит для: эстетика, стиль, красота, дизайн, лайфстайл. **Совпадает с базовой палитрой платформы** — это дефолт для новых страниц.

## Хранение выбора в БД

Поле `palette TEXT` в таблице `mg_master_pages`:
- возможные значения: `ocean` · `red-yellow` · `sakura`
- дефолт при создании страницы: `sakura`
- применяется через атрибут `data-palette="<value>"` на `<body>` страницы `master-page.html`

---

## Правила использования палитр

1. **Базовая палитра платформы (Сакура) фиксирована** — её не меняет мастер, это идентичность самого продукта МГ.
2. **Палитры-пресеты применяются только к публичной странице мастера** (`master-page.html`). Лендинг, анкета, конструктор, гайд — всегда в Сакуре.
3. **Логотипы мессенджеров на кнопках связи** не перекрашиваются под палитру — см. правило в корневом `CLAUDE.md` про узнаваемые бренды.
4. **При добавлении новых палитр** в будущем — формат тот же: 4 цвета (deep/accent/bg/soft), добавляется как `[data-palette="<name>"]` + новое значение в БД.

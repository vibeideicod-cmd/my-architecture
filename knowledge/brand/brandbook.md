### 2.3 Фоны — только два варианта

```
ТЁМНЫЙ  #103206  — Премиум, баннеры, посты, обложки, сайт
СВЕТЛЫЙ #FFF2E9  — Документы, методички, печать, презентации для чтения
```

Никаких других фонов. Чистый белый `#FFFFFF` — запрещён.

### 2.4 Дерево решений: какой цвет выбрать

```
ШАГ 1: Фон
  Премиум / серьёзно / на продажу?      → #103206 тёмный
  Документ для чтения / печать?          → #FFF2E9 кремовый

ШАГ 2: Основной текст
  На тёмном (#103206):
    Заголовки                             → #D4AF37 gold
    Основной текст                        → #FFF2E9 cream
    Вторичный текст / цифры               → #4B7850 sage
  На светлом (#FFF2E9):
    Всё основное                          → #103206 green-ink
    Подзаголовки                          → #4B7850 sage

ШАГ 3: Живой акцент (обязательно один!)
  Куда поставить #13F740 neon?
    → Тонкая линия / подчёркивание под важным словом
    → Свечение вокруг печати / логотипа
    → Точки, штрихи, декоративные элементы в углах
    → Индикатор прогресса в UI
    → Бейдж «NEW» или маркер
    → Рамка активной карточки

ШАГ 4: Эмоциональный акцент (опционально)
  Нужна энергия / выделение блока?      → #FFDD4C yellow (точечно, не фон)

ШАГ 5: CTA или активное действие
  Главный CTA / кнопка действия         → #FF2EC4 magenta
  Нажатое/hover состояние               → #4DAA02 acid
  Ссылки                                → #FF2EC4 magenta
```

### 2.5 Запрещённые цвета

| Цвет | Причина |
|---|---|
| Чёрный | «Дёшево», не из нашего мира. Тёмная основа — зелёный `#103206`. |
| Чистый белый `#FFFFFF` | Холодный, клинический. Для светлой основы — кремовый `#FFF2E9`. |
| Фиолетовые | Ломают целостность зелёно-золотой базы. |
| Красный | Кричит, агрессивен. Все сигналы внимания — через `#FF2EC4`. |
| Серый | Обезличивает. Служебные задачи решает `#FFF2E9` или sage с прозрачностью. |
| Оранжевый `#e86c3a` | Только старые страницы до ребрендинга. Не использовать в новых. |

### 2.6 Промпты цвета для AI-генерации

### 3.4 Правила типографики

- Заголовки на тёмном фоне → `--gold` (`#D4AF37`)
- Заголовки на светлом фоне → `--green-ink` (`#103206`)
- Подзаголовки → `--sage` (`#4B7850`)
- Основной текст на тёмном → `--cream` (`#FFF2E9`)
- Основной текст на светлом → `--green-ink` (`#103206`)
- Акцентное слово → `--gold` или `--neon` подчёркивание (тонкая линия)
- Letter-spacing для CAPS-меток: `0.08em`

## 4. Логотип · Монограмма · Печать С.С.С.Р.

### 4.1 Описание печати (для воссоздания)

**Форма:** круглая печать

**Фон:** радиальный градиент `#4B7850` (sage, центр) → `#103206` (green-ink, края)
> Не фиолетовый, не чёрный, не однотонный.

**Обводка:** `#D4AF37` (gold), 1–2px

**В центре:** монограмма «**ИА**»
- Шрифт: Unbounded 800, цвет: `#D4AF37`
- Размер: ~50% от диаметра

**Над монограммой (внутри):** «АРХИТЕКТОР»
- Шрифт: JetBrains Mono, цвет: `#D4AF37`, маленький

**Вокруг печати дугами (НЕ внутри круга):**
- Верхняя дуга: **С · С · С · Р** — Unbounded 700, letter-spacing 14–20, цвет `#D4AF37`
- Нижняя дуга: СТРУКТУРА · СТРАТЕГИЯ · СВОБОДА · РЕЗУЛЬТАТ — Manrope 500, цвет `#D4AF37`

**Сигнатурная деталь:** две точки-сепаратора `#FF2EC4` (magenta) на 3 и 9 часах

**Живой акцент (опц.):** тонкий `#13F740` ореол/свечение вокруг

### 4.2 Запреты для логотипа

- С.С.С.Р. внутри круга (только дугой снаружи)
- Фиолетовый или чёрный фон печати
- Градиент к фиолетовому или в сторону серого
- Перекрашивать монограмму в не-золотой цвет
- Искажать пропорции

### 4.3 Safe zone

### 7.1 Характер фото

**Настроение:** уверенность без пафоса, живость без суеты.
**Свет:** мягкий направленный. Натуральный или студийный тёплый. Без жёсткой вспышки.
**Цветовая тональность:** тёплая, насыщенная. Тёмно-зелёные тона в фоне или одежде. Никаких холодных голубых фильтров.
**Поза:** активная, вовлечённая — в процессе работы или разговора. Не постановочные «смотрю в камеру со скрещёнными руками».
**Фон:** нейтральный тёмно-зелёный или тёпло-кремовый. Аккуратный офис — ок.

**Не использовать:** стоковые фото с улыбающимися людьми в пиджаках, перенасыщенные фильтры, холодный синий/серый тон.

| архитектор, эксперт | Claude Code, n8n, ChatGPT |
| рекомендации, стратегия | MVP, SaaS-подписка |
| методология, внедрение | обучение, курс |
| консультация | лечение, терапия |

### 8.3 Иерархия формулировок

| Контекст | Формулировка |
|---|---|
| Главные продукты и офферы | **Аудит** ⭐ |
| Платные глубокие разборы | Экспертный аудит |
| Бесплатные первые касания | Консультация / Мини-аудит |
| Оферты и чеки | Экспертное сопровождение |
| Позиционирование роли | Архитектор / Эксперт |

### 8.4 Тон по ситуации

| Ситуация | Тон |
|---|---|
| Пост о методологии | Уверенный, структурный |
| Кейс клиента | Спокойный, конкретный с результатами |
| Ответ на вопрос | Дружеский, без снобизма |
| CTA / призыв | Прямой, без давления |
| Ошибка клиента | Без осуждения, с поддержкой |

### 8.5 Примеры до/после

**До (запрещено):**
> «Я создаю AI-ботов и автоматизирую ваш бизнес с помощью Claude Code и n8n. MVP за 2 недели.»

**После (правильно):**
> «Провожу аудит бизнес-процессов и сопровождаю внедрение систем автоматизации. Методология С.С.С.Р. — от структуры к результату.»

---

**До (запрещено):**
> «Вы устали работать на износ? Вот курс который изменит вашу жизнь!!!»

**После (правильно):**
> «Для тех, кто тянет бизнес на себе 24/7 — есть другой путь. Сначала разбираемся со структурой.»

### 8.6 Стиль

- Простой язык — как объясняешь знакомому
- Конкретные цифры и примеры вместо абстракций
- Структура: тезис → доказательство → вывод
- Без «уникальный», канцелярита и трёх восклицательных знаков

## 10. Чек-лист выдачи

### Цвет и фон
- [ ] Фон только `#103206` или `#FFF2E9`
- [ ] Нет чёрного, белого `#FFFFFF`, фиолетового, красного, серого
- [ ] Не больше 2 акцентных цветов рядом
- [ ] `#13F740` и `#FF2EC4` НЕ вплотную
- [ ] `#D4AF37` и `#FF2EC4` НЕ вплотную

### Композиция
- [ ] Есть один главный якорь (главная мысль выделена)
- [ ] Есть один живой акцент (`#13F740`) — маленький, точечный
- [ ] Пропорции 60/30/10 соблюдены
- [ ] Минимум 32px padding (воздух)

### Брендинг
- [ ] Печать С.С.С.Р. или монограмма «ИА» присутствует
- [ ] С.С.С.Р. дугой снаружи круга — НЕ внутри
- [ ] Фон печати — зелёный градиент (НЕ фиолетовый)
- [ ] Шрифты: Unbounded / Manrope / JetBrains Mono

### Текст
- [ ] НЕТ: vibe coding, Claude Code, MVP, разработка, создаю
- [ ] ЕСТЬ: аудит, сопровождение, экспертиза (где нужно)
- [ ] Нет эмодзи в брендовых композициях

# Технологии и инструменты

Что используем в команде и когда.

---

## Сайты и лендинги

| Инструмент | Когда |
|---|---|
| Чистый HTML + CSS | Всегда — для всех страниц проекта |
| Google Fonts | Шрифты Montserrat и Playfair Display |
| GitHub Pages | Хостинг статических страниц |

---

## Боты и автоматизация

| Инструмент | Когда |
|---|---|
| Telegram Bot API | Боты для приёма заявок, FAQ, рассылок |
| n8n | Автоматизация процессов (self-hosted) |
| Make (Integromat) | Автоматизация без сервера |
| Google Forms + Sheets | Простые формы и учёт заявок |

---

## CRM и управление клиентами

| Инструмент | Когда |
|---|---|
| AmoCRM | Средний бизнес, нужна воронка |
| Битрикс24 (free) | Если нужно много функций бесплатно |
| Notion | Простой учёт + база знаний клиента |
| Google Sheets | Самый простой старт |

---

## Принцип выбора

1. Простота прежде всего — не перегружаем клиента сложными инструментами
2. Учитываем, есть ли у клиента технический человек
3. Предпочитаем инструменты с русским интерфейсом для оффлайн-бизнеса
4. Стараемся минимизировать ежемесячные расходы клиента на старте

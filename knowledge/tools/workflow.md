# Рабочий процесс команды

---

## Как устроена работа с клиентом

```
1. Первый контакт
   → Инна или Ирина общается с клиентом
   → Понимаем задачу и бюджет

2. Брифинг
   → Аналитик задаёт вопросы
   → Формируется бриф

3. Предложение
   → Директор определяет, кто и что делает
   → Инна согласовывает с клиентом

4. Исполнение
   → Специалист выполняет работу
   → QA проверяет результат

5. Сдача
   → Инна передаёт клиенту
   → Правки при необходимости
```

---

## Роли в команде

| Кто | Роль |
|---|---|
| Инна | Руководитель, общение с клиентами, принятие решений |
| Ирина | Дизайн — логотипы, фирменный стиль, визуальные материалы |
| Директор (ИИ) | Управление задачами, распределение по специалистам |
| Специалисты (ИИ) | Исполнение в своём направлении |
| QA (ИИ) | Проверка качества перед сдачей |

---

## Принципы работы

- Сначала описываем, потом делаем (структура перед кодом, бриф перед исполнением)
- Каждый результат проходит QA
- Инна принимает финальное решение по всем задачам
- Детали о клиенте берём только из брифа — не придумываем

# ИИ-инструменты для работы команды

Классификация AI-инструментов по задачам. Дополняет `tech-stack.md` — там стандартные инструменты, здесь ИИ-специфика.

---

## Для текста и контента

| Инструмент | Что делает | Когда использовать |
|---|---|---|
| Claude (этот ИИ) | Тексты, стратегия, анализ, код, агенты | Основной инструмент команды — всё |
| ChatGPT | Тексты, идеи, переводы | Альтернатива, когда нужен второй взгляд |
| Gemini | Работа с Google-документами, поиск | Когда нужна интеграция с Google Workspace |

**Задачи:**
- Посты для соцсетей, контент-планы, рубрики
- Тексты для лендингов и сайтов
- Сценарии для Reels и коротких видео
- Email-рассылки и прогревы
- Анализ и переработка клиентских материалов

---

## Для картинок и визуала

| Инструмент | Что делает | Когда использовать |
|---|---|---|
| Midjourney | Художественные и стилизованные изображения | Рекламные креативы, атмосферные картинки |
| DALL-E (через ChatGPT) | Изображения по описанию | Быстрые варианты, иллюстрации |
| Ideogram | Изображения с текстом внутри | Когда нужен текст на картинке |
| Adobe Firefly | Редактирование фото, генерация фонов | Работа с реальными фотографиями клиента |
| Canva AI | Шаблоны + AI-генерация | Быстрые материалы для соцсетей |

**Задачи:**
- Рекламные креативы для таргета
- Обложки для постов и Telegram-каналов
- Визуальные концепции для презентаций клиенту
- Фоны и дополнение фотографий

## Для видео и голоса

| Инструмент | Что делает | Когда использовать |
|---|---|---|
| ElevenLabs | Генерация голоса / озвучка текста | Аудио для обучения, прогревы, подкасты |
| Runway | AI-видео, анимация изображений | Анимированные креативы |
| HeyGen | AI-аватар говорит на камеру | Видеопрезентации без съёмки |
| CapCut AI | Монтаж + субтитры + эффекты | Быстрый монтаж Reels для клиентов |

**Задачи:**
- Озвучка сценариев без диктора
- Анимация статичных изображений
- Видеоконтент для клиентов без съёмочной группы

## Для аналитики и исследований

| Инструмент | Что делает | Когда использовать |
|---|---|---|
| Claude / ChatGPT | Анализ данных, поиск инсайтов | Анализ брифа, ЦА, конкурентов |
| Perplexity | Поиск с источниками | Когда нужны актуальные данные с ссылками |
| NotebookLM | Работа с большими документами | Анализ объёмных материалов клиента |

**Задачи:**
- Исследование целевой аудитории
- Анализ конкурентов по описанию / сайту
- Поиск трендов в нише клиента
- Обработка больших объёмов клиентских данных

## Для автоматизации и ботов

| Инструмент | Что делает | Когда использовать |
|---|---|---|
| n8n + AI-нода | Автоматизация + AI в одном потоке | Когда клиенту нужен умный бот или авто-ответчик |
| Make + AI-модули | То же, облачный вариант | Когда нет своего сервера |
| Telegram Bot + GPT API | Чат-бот с AI-ответами | AI-саппорт, AI-продавец в Telegram |
| **OpenRouter** | Агрегатор моделей — один API-ключ для Claude, GPT-4.1, Gemini, Llama и др. | Мультиагентные боты, где разные агенты используют разные модели. Лимит $5-10 |

**Задачи:**
- AI-бот для приёма заявок с умными ответами
- Авто-обработка входящих сообщений
- Рассылки с персонализацией через AI
- **Мультиагентные боты** — бот-копирайтер, бот-аналитик, бот с командой специалистов

## Выбор модели по задаче (для мультиагентных ботов, из Урока 15)

| Модель | Для чего | Температура |
|---|---|---|
| **claude-opus-4-6** | Архитектура, сложный код, ключевые решения. Топовое качество — медленнее и дороже | 0.7 |
| **claude-sonnet-4-6** | Творческие задачи, тексты, копирайтинг, основной код. Лучший выбор для бота-копирайтера | 0.3–0.8 |
| **claude-haiku-4-5** | Быстрые и простые задачи: проверки, маршрутизация, тестирование. Дёшево и быстро | 0.1–0.2 |

**Принцип:** ИИ плохо понимает многосоставные задачи. Когда каждый агент отвечает за одну задачу — результат точнее. Декомпозитор разбивает сложное на мелкое, Оркестратор собирает обратно.

## Принцип выбора AI-инструмента

1. **Начни с Claude** — для большинства задач этого достаточно
2. **Нужна картинка?** → Midjourney (стиль) или DALL-E (скорость)
3. **Нужно видео/голос?** → ElevenLabs (голос) или CapCut AI (монтаж)
4. **Нужны актуальные данные?** → Perplexity, не ChatGPT
5. **Нужна автоматизация?** → n8n + AI, не просто бот
6. **Мультиагентный бот?** → OpenRouter (один ключ, разные модели по ролям)
7. **Технический барьер клиента** — всегда учитывай. Если клиент не разберётся сам — не предлагай.

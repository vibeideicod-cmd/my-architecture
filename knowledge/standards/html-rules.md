# Правила вёрстки

Обязательные требования для всех HTML-страниц проекта.

---

## Структура файла

```html
<!DOCTYPE html>
<html lang="ru">
<head>
  <meta charset="UTF-8" />
  <meta name="viewport" content="width=device-width, initial-scale=1.0" />
  <title>...</title>
  <!-- Только Google Fonts, если нужны -->
  <style>
    /* Все стили здесь */
  </style>
</head>
<body>
  <!-- Контент -->
</body>
</html>
```

---

## Запрещено

- Внешние CSS-файлы
- CDN: Bootstrap, Tailwind, Animate.css и подобные
- Внешние JS-библиотеки (jQuery, GSAP и т.д.)
- Inline-стили в HTML (`style="..."`) — только в крайнем случае

---

## Обязательно

- Все стили в `<style>` внутри `<head>`
- `box-sizing: border-box` для всех элементов
- CSS-переменные для цветов (`:root { --accent: #e86c3a; }`)
- Семантические теги: `<header>`, `<main>`, `<section>`, `<footer>`, `<nav>`
- `alt` для всех изображений

---

## Адаптивность

Обязательные переломные точки:

```css
/* Мобильный (по умолчанию — пишем mobile-first) */

@media (min-width: 480px) { /* большой мобильный */ }
@media (min-width: 768px) { /* планшет */ }
@media (min-width: 1024px) { /* десктоп */ }
```

Размеры текста через `clamp()`:
```css
font-size: clamp(минимум, предпочтительное, максимум);
/* Пример: */
font-size: clamp(1.8rem, 5vw, 3.5rem);
```

Раскладка:
- `flexbox` для строчных элементов и навигации
- `grid` для сеток карточек и секций

---

## Производительность

- Шрифты подключать с `rel="preconnect"` и `crossorigin`
- Изображения указывать с атрибутами `width` и `height` (если возможно)
- Не добавлять JS без необходимости — большинство эффектов решается CSS

---

## Именование классов

- Строчные буквы, дефис как разделитель: `.card-title`, `.hero-btn`
- БЭМ не обязателен, но логика именования должна быть понятной
- Не использовать ID для стилей

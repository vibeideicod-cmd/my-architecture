## Запрещённые цвета

| Цвет | Причина |
|---|---|
| Чёрный | Не из нашего мира. Тёмная основа — зелёный `#103206`. |
| Белый `#FFFFFF` | Клинический. Светлый фон — только `#FFF2E9`. |
| Фиолетовые | Ломают зелёно-золотую базу. |
| Красный | Все сигналы внимания — через `#FF2EC4` magenta. |
| Серый | Обезличивает. Нейтраль решает `#FFF2E9` или sage с прозрачностью. |
| Оранжевый `#e86c3a` | Только старые страницы до ребрендинга. В новых — запрещён. |

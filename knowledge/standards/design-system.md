# Дизайн-система команды

Система строится вокруг **официальной палитры Инны Андрейченко** (8 цветов, согласована апрель 2026). Светлая нейтральная база. Тёмные темы допустимы только для клиентских проектов по их запросу.

---

## ⚠️ Важное правило

**Для всего, что делается для бренда Инны и внутренних продуктов** — только **светлая нейтральная палитра** (основа — Кремовая слоновая кость `#FFF2E9`). Никакого чёрного фона.

Чёрный/тёмный фон разрешён **только** когда:
- Клиент явно просит тёмную тему
- Или когда это подходит бренду конкретного клиента (не бренду Инны)

## Официальная палитра бренда — 8 цветов

> Согласована 26.04.2026. Это точный брендбук Инны. Никаких импровизаций, никаких «похожих» оттенков — только эти цвета.

| # | Название | HEX | RGB | CMYK | Число | Роль |
|---|---|---|---|---|---|---|
| 01 | Глубокий лес | `#103206` | 16, 50, 6 | C68 M0 Y88 K80 | 3 | Основной тёмный фон / заголовки |
| 02 | Шалфей | `#4B7850` | 75, 120, 80 | C38 M0 Y33 K53 | 6 | Поддерживающий зелёный / wellness |
| 03 | Золото | `#D4AF37` | 212, 175, 55 | C0 M17 Y74 K17 | 5 | Главный акцент / CTA / ценность |
| 04 | Яркий жёлтый | `#FFDD4C` | 255, 221, 76 | C0 M13 Y70 K0 | 8 | Энергия / акцент / эмоции |
| 05 | Кремовая слоновая кость | `#FFF2E9` | 255, 242, 233 | C0 M5 Y9 K0 | 2 | **Основной фон (60–70%)** |
| 06 | Неоновый зелёный | `#13F740` | 19, 247, 64 | C92 M0 Y74 K3 | 6 | Tech-акцент / wow / цифровые носители |
| 07 | Кислотно-зелёный | `#4DAA02` | 77, 170, 2 | C55 M0 Y99 K33 | 6 | CTA / кнопки действия / спорт/эко |
| 08 | Жгучая маджента | `#FF2EC4` | 255, 46, 196 | C0 M82 Y23 K0 | 6 | Wow-моменты / манифест / self-expression |

### Правило 60/30/10

На любом носителе — фон, основной цвет и акцент. **Не более 3 цветов на одном макете.**

| Слой | Доля | Цвет |
|---|---|---|
| Фон (60%) | базовый | `#FFF2E9` (кремовый) или `#103206` (лес) |
| Основной (30%) | по задаче | 01, 02, 03 — в зависимости от задачи |
| Акцент (10%) | точечно | 03, 06, 07 или 08 — один, не все |

### Контрастные пары для текста

- **Тёмный текст:** всегда `#103206`
- **Светлый текст:** всегда `#FFF2E9`
- Никогда не ставить жёлтый/неон/маджента как фон для длинного текста

### Применение по задаче

| Задача | Цвета |
|---|---|
| Премиум / наследие | 01 Глубокий лес + 03 Золото |
| Wellness / доверие | 02 Шалфей |
| CTA / действие | 07 Кислотно-зелёный |
| Wow / манифест | 08 Маджента |
| Tech / digital | 06 Неоновый зелёный (на тёмном фоне) |
| Базовый фон | 05 Кремовая слоновая кость |

### Нельзя

- Смешивать 06 Неоновый и 07 Кислотный — конфликтуют
- Жёлтый `#FFDD4C` или маджента `#FF2EC4` как фон длинного текста
- Яркие цвета (06–08) > 30% площади макета
- Кислотный зелёный без тёмной базы (`#103206`)

  --brand-gold:     #D4AF37;  /* 03 Золото — главный акцент, CTA */
  --brand-yellow:   #FFDD4C;  /* 04 Яркий жёлтый — эмоции, энергия */
  --brand-cream:    #FFF2E9;  /* 05 Кремовая слоновая кость — основной фон */
  --brand-neon:     #13F740;  /* 06 Неоновый зелёный — tech, digital */
  --brand-acid:     #4DAA02;  /* 07 Кислотно-зелёный — CTA-кнопки, действие */
  --brand-magenta:  #FF2EC4;  /* 08 Жгучая маджента — wow, манифест */

  /* Токены применения */

### CSS-переменные (бренд Инны)

```css
:root {
  /* Фоны — светлые нейтральные */
  --bg:           #ffffff;  /* основной фон — белый */
  --bg-warm:      #d3bfb1;  /* бежевый — тёплые секции */
  --bg-cool:      #cbcbd4;  /* светло-серый — нейтральные секции */

  /* Основные акценты */
  --gold:         #d4af37;  /* золотой — главный акцент, CTA */
  --mustard:      #c99700;  /* горчичный — вторичный акцент, hover */
  --green:        #4b7850;  /* тёмно-зелёный — поддержка, success */
  --green-deep:   #103206;  /* глубокий — для заголовков, кнопок depth */

  /* Текст */
  --text:         #4c4651;  /* серый — основной текст */
  --text-soft:    #cbcbd4;  /* светло-серый — вспомогательный */

  /* Эпизодически */
  --yellow:       #ffc403;  /* жёлтый — только эмоциональный акцент */
}
```

### Правила применения

- **Фон страницы** — белый `#ffffff` или бежевый `#d3bfb1`. Выбор по настроению. Никакого чёрного.
- **Основной текст** — серый `#4c4651`, не чистый чёрный.
- **Главный CTA и ссылки** — золотой `#d4af37`.
- **Hover / вторичный акцент** — горчичный `#c99700`.
- **Success / поддержка** — тёмно-зелёный `#4b7850`.
- **Deep акцент / заголовки глубины** — глубокий зелёный `#103206`.
- **Жёлтый** — только точечные эмоциональные всплески. Не кнопки, не большие области.

### ⛔ Чего НЕТ в бренде Инны

- Чёрный фон `#0a0a0a` — **нет**.
- Оранжевый `#e86c3a` — **нет**. Это остатки старых клиентских страниц.
- Самовыдуманные оттенки (кремовый, молочный, бежево-розовый и т.п.) — **нет**. Только цвета из списка.

## Визуальные принципы

### Общие
- Минимализм с характером — каждый элемент работает на идею
- Структура как эмоция — направляет, не перегружает
- Единые шаблоны с логикой отступов и сеткой
- Иконки: минималистичные, плоские, без теней и объёма

### Подбренд "Нейро Бабки СССР"
- Стиль подачи: **иронично-серьёзный** — визуальный юмор с глубиной смысла
- Персонаж: мультяшная зрелая женщина 55+, уверенная, с харизмой
- Одежда и контексты: от пижамы до бизнес-костюма, всегда через призму юмора
- **Фотографии не используются** — только аватар-персонажи
- Фоны: допускается зернистость, стилизация под бумагу, ретро-эффекты (70-е–80-е)
- Текстуры: рамки, зернистость, наложение бумаги

## 🚫 Чёрный список — что НИКОГДА не используем

Адаптация запретов из официального скила `frontend-design` от Anthropic + наш опыт. Эти вещи — индикатор «AI slop эстетики», именно их выдаёт нейросеть по дефолту, и именно они делают страницу безликой.

### Шрифты-табу

- ❌ **Inter** — самый узнаваемый «AI-default»
- ❌ **Roboto** — Google-default, выдаёт лень
- ❌ **Arial / Helvetica** — браузерный default, отсутствие выбора
- ❌ **System fonts** (`-apple-system`, `BlinkMacSystemFont`) как основной шрифт страницы — допустимо только в технических утилитах, не в продуктах
- ❌ **Open Sans** — был хорош в 2014, сейчас = «не думали»

**Что используем:** Montserrat + Playfair Display (наш дефолт). Для конкретных клиентов выбираем характерные шрифты с Google Fonts: Fraunces, Bricolage Grotesque, DM Serif Display, Cormorant, Space Mono, JetBrains Mono, Fredoka (как для ЮБ), Unbounded, Manrope, Spectral, Lora — и сотни других. Парный display-шрифт + refined body — это правило, а не опция.

### Цветовые штампы

- ❌ **Фиолетовые градиенты на белом** (`#a855f7 → #ec4899`) — флагманский AI slop, запрещено
- ❌ **Tailwind palette по умолчанию** (slate-500, blue-600, gray-700) — без адаптации к бренду
- ❌ **Чёрный `#000000` для текста** — слишком жёстко, используем `#4c4651` или `#103206`
- ❌ **Чистый белый `#ffffff` фон + чистый чёрный текст** — невыдержанный контраст без характера. Либо тёплый off-white, либо палитра клиента.
- ❌ **Самовыдуманные оттенки** «похожие на бренд» — только HEX-ы из палитры клиента или из брендбука Инны

### Структурные штампы

- ❌ **Симметричная 3-колоночная сетка** «карточка-карточка-карточка» одинакового размера — главный признак шаблонной страницы. Если 3 карточки нужны — пусть будут разной высоты, разного веса, с overlap или offset
- ❌ **Hero с центрированным заголовком + кнопкой по центру + плоским фоном** — дефолт лендинга 2015 года
- ❌ **Иконки из Heroicons / Feather по дефолту** без адаптации стиля под бренд — выдают AI-сборку
- ❌ **`box-shadow: 0 4px 6px rgba(0,0,0,0.1)`** — серая дефолтная тень. Тени должны быть с цветом из палитры (см. техника №5 в [visual-techniques.md](visual-techniques.md))
- ❌ **`border-radius: 8px`** на всём подряд — без визуального обоснования. Либо острые углы, либо большие 24-32px, либо асимметричные

### Эффекты и анимации-табу

- ❌ **Hover `opacity: 0.8`** — это не hover, это «забыли сделать hover». См. техника №9 в [visual-techniques.md](visual-techniques.md)
- ❌ **Десять разбросанных микро-анимаций** вместо одной оркестрованной page-load. Один сильный момент > десять слабых
- ❌ **`transition: all 0.3s ease`** — ленивая запись. Указывай конкретные свойства и характерные easing'и
- ❌ **Параллакс просто потому что можно** — без концептуальной связи с темой страницы

### Главный принцип

**Полумерные страницы запрещены.** Если сделал «немного градиента, немного теней, немного анимаций» без выбранного визуального курса — это AI slop. Либо коммитимся в курс полностью, либо убираем декор и идём в брутальный минимализм.

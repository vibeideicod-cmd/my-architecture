# Дизайн-система команды

Система включает два уровня: **базовая палитра команды** и **подбренд "Нейро Бабки СССР"** со своей акцентной системой.

---

## Базовая палитра команды

### Фоны
| Название | HEX | Когда использовать |
|---|---|---|
| Чёрный | `#0a0a0a` | Основной фон страницы |
| Тёмно-зелёный | `#103206` | Альтернативный фон, карточки, секции |
| Зелёный акцент | `#4b7850` | Поддержка, баланс, обводки, hover |

### Акценты
| Название | HEX | Когда использовать |
|---|---|---|
| Оранжевый | `#e86c3a` | Главный акцент — кнопки CTA, выделение, ссылки (новые страницы) |
| Золото | `#d4af37` | Ценность, успех — разделители, иконки, детали |
| Горчичный | `#c99700` | Сила, зрелость — hover, акцент второго уровня |

### Текст
| Название | HEX | Когда использовать |
|---|---|---|
| Светлый | `#e8e0d8` | Основной текст на тёмном фоне |
| Бежевый | `#d3bfb1` | Тепло, человечность — подзаголовки, описания |
| Приглушённый | `#a89e94` | Вспомогательный текст, подписи, метки |
| Серый | `#4c4651` | Структура, технология — нейтральные блоки |
| Светло-серый | `#cbcbd4` | Нейтральность, пространство |

### CSS-переменные (базовые)

```css
:root {
  --bg: #0a0a0a;
  --bg-green: #103206;
  --green: #4b7850;
  --accent: #e86c3a;
  --gold: #d4af37;
  --mustard: #c99700;
  --text: #e8e0d8;
  --beige: #d3bfb1;
  --text-muted: #a89e94;
  --gray: #4c4651;
  --gray-light: #cbcbd4;
}
```

## Подбренд "Нейро Бабки СССР"

Акцентная система подбренда смещается в сторону горчичного и бежевого. Зелёный становится вспомогательным.

### Акценты подбренда
| Название | HEX | Роль |
|---|---|---|
| Горчичный | `#c99700` | Главный акцент — сила, зрелость, деньги |
| Бежевый | `#d3bfb1` | Тепло, человечность, доверие |
| Золото | `#d4af37` | Ценность, успех, значимость |
| Жёлтый | `#ffc403` | Эпизодически — эмоциональный акцент, не основной |
| Зелёный | `#4b7850` | Вспомогательный |

### CSS-переменные подбренда

```css
:root {
  --nb-accent: #c99700;      /* горчичный — главный */
  --nb-warm: #d3bfb1;        /* бежевый */
  --nb-gold: #d4af37;        /* золото */
  --nb-highlight: #ffc403;   /* жёлтый — эпизодически */
  --nb-support: #4b7850;     /* зелёный — поддержка */
}
```

## Шрифты

| Шрифт | Назначение | Начертания |
|---|---|---|
| `Montserrat` | Основной — навигация, интерфейс, тело текста, посты | 300, 400, 500, 600, 700 |
| `Playfair Display` | Акцентный — заголовки лендингов, постеры, визуальные акценты | 400, 600, 700; italic |

**Стандарты:**
- Заголовки: Montserrat Bold или Playfair Display Bold
- Подзаголовки: Montserrat Medium
- Базовый текст: Montserrat Regular

### Подключение

```html
<link rel="preconnect" href="https://fonts.googleapis.com" />
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin />
<link href="https://fonts.googleapis.com/css2?family=Montserrat:wght@300;400;500;600;700&family=Playfair+Display:ital,wght@0,400;0,600;0,700;1,400&display=swap" rel="stylesheet" />
```

### Размеры текста (clamp)

```css
font-size: clamp(2rem, 6vw, 4rem);      /* главный заголовок */
font-size: clamp(1.3rem, 3.5vw, 2rem);  /* подзаголовок секции */
font-size: clamp(0.9rem, 2vw, 1rem);    /* основной текст */
font-size: clamp(0.75rem, 1.5vw, 0.875rem); /* метки */
```

## Визуальные принципы

### Общие
- Минимализм с характером — каждый элемент работает на идею
- Структура как эмоция — направляет, не перегружает
- Единые шаблоны с логикой отступов и сеткой
- Иконки: минималистичные, плоские, без теней и объёма

### Подбренд "Нейро Бабки СССР"
- Стиль подачи: **иронично-серьёзный** — визуальный юмор с глубиной смысла
- Персонаж: мультяшная зрелая женщина 55+, уверенная, с харизмой
- Одежда и контексты: от пижамы до бизнес-костюма, всегда через призму юмора
- **Фотографии не используются** — только аватар-персонажи
- Фоны: допускается зернистость, стилизация под бумагу, ретро-эффекты (70-е–80-е)
- Текстуры: рамки, зернистость, наложение бумаги

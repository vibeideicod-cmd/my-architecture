# Контент-план: стратегия и система создания

Полная система для планирования контента на месяц. Стратегические принципы + готовые промты + чеклист.

---

## Два подхода к балансу контента

### Подход 1: Формула 40/30/30 (из Урока 5)

Базовое соотношение для аккаунта, который продаёт:

| Тип | Доля | Что входит |
|---|---|---|
| Образовательный | 40% | Решения проблем, инсайты и лайфхаки, разборы кейсов |
| Развлекательный | 30% | Истории и метафоры, закулисье процессов, человеческие моменты |
| Продающий | 30% | Кейсы с результатами, триггеры и боли, призывы к действию |

**Практика:** на 10 постов в месяц → 4 обучающих, 3 вовлекающих, 3 продающих.

### Подход 2: Правило 4+1 (из Урока 14)

Для экспертного блога / канала вайбкодера / фрилансера:

| Тип | Доля | Что входит |
|---|---|---|
| Экспертный | 40% | Взгляд на рынок, тренды, профессию. Люди сохраняют и цитируют |
| Личный | 40% | Процессы, ошибки, закулисье, результаты. Строит доверие лучше рекламы |
| Продающий | 20% | Оффер, призыв к действию. Не чаще 1 из 5 |

**Золотое правило:** 4 экспертных/личных + 1 продающий. Если писать только «купи» — люди отписываются. Если только умничать — уважают, но не платят. Личный контент — самый недооценённый.

> **Когда какой подход:** Формула 40/30/30 — для коммерческих аккаунтов бизнеса (магазин, санаторий, сервис). Правило 4+1 — для экспертных блогов и личных каналов.

## Фреймворк БМПХС — 5 слоёв аудитории (из Урока 14)

Перед созданием контент-плана исследуй аудиторию по пяти слоям:

| Слой | Что это | Как использовать в контенте |
|---|---|---|
| **Б**оли | Что сейчас не работает, причиняет дискомфорт прямо сейчас | Начало поста: зацепить реальной болью |
| **М**ечты | Образ желаемого будущего | Показать результат: «вот как может быть» |
| **П**роблемы | Системные сложности, которые мешают расти | Экспертный контент: разобрать причину и дать решение |
| **Х**отелки | Конкретные желания, иногда невысказанные вслух | Оффер: предложить именно то, что хотят |
| **С**трахи | Что останавливает от действия | Закрытие возражений: снять тревогу, дать уверенность |

Подробный промпт для исследования ЦА по БМПХС — в [content-marketing.md](../playbooks/content-marketing.md#фаза-2--research-исследование-аудитории).

## Психологические триггеры

Встраивай в посты для повышения вовлечённости и конверсии:

**Триггеры доверия и действия:**
- **Социальное доказательство** — отзывы, цифры, количество клиентов
- **Дефицит** — ограниченное количество мест, дедлайн
- **Взаимный обмен** — даёшь ценность бесплатно → получаешь доверие
- **Последовательность** — маленькое «да» ведёт к большому
- **Авторитет** — экспертность, опыт, результаты

**Эмоциональные крючки:**
- **Любопытство** — незаконченная мысль, неожиданный тезис
- **Страх упустить** (FOMO) — то, что происходит без тебя
- **Принадлежность** — «такие как мы»
- **Стремление к совершенству** — стать лучше / умнее
- **Поиск безопасности** — снять тревогу, дать уверенность

## Копирайтинг-формулы

Используй как структуру поста или его части:

### PAS (Проблема → Усиление → Решение)
Называешь проблему → делаешь больнее (что будет если не решить) → даёшь выход.
Лучшая формула для экспертного и продающего контента.

### AIDA (Внимание → Интерес → Желание → Действие)
Цепляешь → удерживаешь → создаёшь желание → ведёшь к шагу.
Классика для продающих постов и серий.

### 4U (Полезно → Срочно → Уникально → Ультраспецифично)
Четыре критерия сильного заголовка или оффера.
Проверяй каждый заголовок по этим четырём — если проходит все → берём.

## 4-недельная структура месяца

Логика: сначала привлекаем → строим доверие → показываем продукт → закрываем сомнения.

| Неделя | Фокус | Содержание |
|---|---|---|
| **1 — Активация интереса** | Боли аудитории | 3 поста про боли, 2 поста с решениями, 2 поста с результатами |
| **2 — Углубление доверия** | Экспертность | 3 поста с инсайтами, 2 поста с историями, 2 поста с доказательствами |
| **3 — Продуктовая неделя** | Оффер | 3 поста про преимущества, 2 поста с закрытием возражений, 2 поста с призывами |
| **4 — Закрытие сомнений** | Финальный прогрев | 3 поста с кейсами, 2 поста с триггерами, 2 поста с призывами к действию |

## 6-шаговый каскадный промт для контент-плана

Полная система создания контент-плана на месяц. Каждый шаг — отдельный запрос, строится на предыдущем.

### Шаг 1 — Стратегическая основа
```
Проанализируй нишу [название ниши] и создай:

1. Ключевые темы контента:
   - Топ-5 болей аудитории
   - Топ-5 желаний аудитории
   - Топ-5 возражений

2. Форматы подачи:
   - 3 образовательных формата
   - 3 развлекательных формата
   - 3 продающих формата

3. Триггеры вовлечения:
   - 3 главных мотиватора аудитории
   - 3 основных страха
   - 3 точки желаний
```

### Шаг 2 — Каркас месяца
```
На основе стратегической основы создай структуру месяца по 4 неделям:
- Неделя 1: Активация интереса (3 боли + 2 решения + 2 результата)
- Неделя 2: Углубление доверия (3 инсайта + 2 истории + 2 доказательства)
- Неделя 3: Продуктовая (3 преимущества + 2 возражения + 2 призыва)
- Неделя 4: Закрытие сомнений (3 кейса + 2 триггера + 2 призыва)

Для каждого поста кратко укажи тему и тип контента.
```

### Шаг 3 — Детальный план в таблице
```
Создай детальный контент-план на месяц.
Для каждого поста укажи:
1. День недели
2. Оптимальное время публикации
3. Тему поста
4. Ключевой инсайт
5. Формат подачи
6. Ожидаемый результат

Оформи в виде таблицы.
```

### Шаг 4 — Генерация контента
```
Для каждого поста из плана создай:
1. Цепляющий заголовок
2. Структуру поста:
   - Открывающий хук
   - Основные блоки
   - Призыв к действию
3. Хештеги и ключевые слова

Используй техники: сторителлинг, эмоциональные триггеры, петли любопытства.
```

### Шаг 5 — Оптимизация и связки
```
Проанализируй план и:
1. Создай связки между постами (ссылки вперёд / назад)
2. Добавь перекрёстные отсылки внутри серий
3. Усиль призывы к действию
4. Предложи варианты для A/B-тестирования
```

### Шаг 6 — Финальная версия
```
С учётом оптимизации и связок пришли финальный контент-план.
Для каждого поста укажи:
1. День недели
2. Время публикации
3. Тему поста
4. Заголовок
5. Открывающий хук
6. Основные блоки
7. Призыв к действию
8. Ключевой инсайт
9. Формат подачи
10. Ожидаемый результат

Оформи в виде таблицы.
```

## Промт с технологиями воздействия (быстрый вариант на неделю)

Когда нужен план на одну неделю, не на месяц:

```
Создай контент-план на неделю для [ниша].

Психологические триггеры: социальное доказательство, дефицит, взаимный обмен.

Стратегии вовлечения: петля любопытства, сторителлинг, эмоциональные зацепки.

Для каждого поста:
- Цепляющий оффер / заголовок
- Описание проблемы
- Решение
- Доказательства
- Призыв к действию
- Усиливающие элементы: социальные доказательства, ограничения, бонусы
```

## Масштабирование: 1 идея = 10 постов

Одну тему можно развернуть в серию:
1. Проблема (почему это важно)
2. Миф об этой теме (что думают неправильно)
3. Механизм (как это работает на самом деле)
4. Пошаговое решение
5. Кейс клиента с результатом
6. Частая ошибка при применении
7. История из своей практики
8. FAQ — 3 главных вопроса по теме
9. Сравнение: было / стало
10. Призыв к действию с оффером

## Типичные ошибки контент-плана

### Слабые заголовки

✗ «Как похудеть»
✓ «Почему одни худеют на пицце, а другие толстеют на гречке: неочевидная причина»

Правило: заголовок должен проходить тест 4U — полезно, срочно, уникально, ультраспецифично.

### Нет системы в постах

✗ Случайные темы без связи между собой
✓ Серии с логикой: каждый пост ведёт к следующему

### Слабые призывы к действию

✗ «Пишите в комментариях»
✓ «Ответьте в комментариях "+", если узнали себя в пункте №3»

Правило: CTA должен быть конкретным, простым и с микро-обязательством.

## Чеклист проверки контент-плана

**Структура:**
- [ ] Соблюдена пропорция 40/30/30
- [ ] Все типы контента присутствуют
- [ ] Логичная последовательность по неделям

**Контент:**
- [ ] Заголовки проходят тест 4U
- [ ] У каждого поста чёткая структура
- [ ] Сильные конкретные призывы к действию

**Продажи:**
- [ ] Есть продающая серия (не отдельные посты)
- [ ] Встроены триггеры доверия
- [ ] Есть работа с возражениями

**Вовлечение:**
- [ ] Есть элементы интерактива
- [ ] Присутствуют эмоциональные крючки
- [ ] Есть связки между постами

# Типы постов и структуры промтов для них

Три основных типа контента для соцсетей. Для каждого — когда использовать, готовый промт-шаблон и лайфхаки усиления.

---

## Три типа постов

| Тип | Когда использовать | Цель |
|---|---|---|
| Экспертный | Есть уникальное знание, можно подтвердить цифрами | Авторитет, доверие |
| Вовлекающий | Есть яркая история, нужно построить эмоциональную связь | Вовлечённость, близость |
| Продающий | Есть чёткий оффер, аудитория готова к покупке | Заявки, продажи |

---

## Экспертный пост

**Когда использовать:**
- Есть конкретное знание, которое решает проблему аудитории
- Можно подкрепить цифрами или практическим примером
- Аудитория ищет решение — не развлечение

**Структура:** проблема → решение → применение → результат

**Промт-шаблон:**
```
Роль: Опытный эксперт по [ниша] с [X] лет опыта в [специализация]

Задача: Создай обучающий пост, который:
- Раскрывает одну конкретную технику [тема]
- Показывает её применение на практике с примером
- Включает 1-2 распространённые ошибки
- Даёт измеримый результат

Формат:
- Объём: до 2 000 знаков
- Структура: проблема → решение → применение → результат
- Тип: [Instagram / Telegram / VK]

Тон: Уверенный профессионал, делящийся проверенными методиками
```

**Лайфхаки:**
- Добавляй «Малоизвестный факт:» — поднимает интерес
- Используй конкретные цифры и статистику
- Фраза «Вопреки распространённому мнению...» — сразу цепляет

---

## Вовлекающий пост (история)

**Когда использовать:**
- Есть яркая история из практики или жизни клиента
- Нужно выстроить доверие через эмоцию
- Хочется вызвать отклик, комментарии, репосты

**Структура:** завязка → конфликт → решение → результат (+ обучающий элемент)

**Промт-шаблон:**
```
Роль: Опытный сторителлер с навыком вовлекающего нарратива

Задача: Создай пост-историю, который:
- Начинается с момента напряжения или неожиданного поворота
- Имеет яркий поворотный момент
- Заканчивается значимым результатом
- Содержит один обучающий инсайт

Формат:
- Структура: завязка → конфликт → решение → результат
- Стиль: эмоциональный нарратив с деталями и диалогами
- Объём: 1 500–2 000 знаков

Тон: Искренний рассказчик, делящийся личным опытом — без дидактики
```

**Лайфхаки:**
- Начинай с «Никогда не думал, что...» — сразу интрига
- Добавляй короткие диалоги в кавычках — делает живым
- Техника «Было / Стало» — простой и эффективный контраст
- Техника «Прерванный рассказ» — разбей на части с интригующими подзаголовками

---

## Продающий пост

**Когда использовать:**
- Есть чёткий оффер с понятной ценностью
- Аудитория уже прогрета или близка к решению
- Есть социальные доказательства (отзывы, цифры, кейсы)

**Структура:** боль → понимание → решение → доказательства → оффер + CTA

**Промт-шаблон:**
```
Роль: Опытный копирайтер со специализацией в продажах через текст

Контекст:
- Продукт/услуга: [что именно]
- Целевая аудитория: [кто, возраст, ситуация]
- Главная боль: [что мучает прямо сейчас]
- Уникальное преимущество: [чем отличаемся]
- Социальные доказательства: [цифры, отзывы, кейсы]

Задача: Создай продающий пост, который:
- Начинается с главной боли аудитории
- Показывает понимание проблемы (не сразу продаёт)
- Представляет решение через результаты клиентов
- Ведёт к конкретному действию

Формат:
- Структура: боль → решение → доказательства → оффер → CTA
- Включить конкретные цифры
- Объём: до 1 500 знаков

Тон: Эмпатичный эксперт, предлагающий проверенное решение — без агрессивного давления
```

**Элементы усиления:**
- Социальные доказательства (цифры, истории клиентов)
- Ограничение (время, количество мест) — добавляет срочность
- Бонус в конце поста — повышает ценность оффера

**Лайфхаки:**
- Начинай с главной боли, не с продукта
- Добавляй реальные истории клиентов с конкретикой
- Объявляй «Бонус в конце поста» — удерживает до конца

## Чеклист качества поста (перед публикацией)

**Начало:**
- [ ] Первые 2 строки вызывают интерес или интригу
- [ ] Есть неожиданный элемент или неочевидный тезис
- [ ] Понятна ценность для читателя

**Структура:**
- [ ] Логичное развитие — каждый абзац ведёт к следующему
- [ ] Чёткие переходы между смысловыми блоками
- [ ] Сильное завершение (инсайт, CTA или запоминающаяся мысль)

**Контент:**
- [ ] Соответствует цели (экспертный / вовлекающий / продающий)
- [ ] Решает конкретную проблему аудитории
- [ ] Ведёт к нужному действию

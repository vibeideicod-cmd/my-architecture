# Входящие материалы

Сюда кладутся файлы для обработки Директором.

## Как пользоваться

1. Положи MD-файл (или PDF) в эту папку
2. Скажи Директору: **«обработай входящие»** (или «что в inbox?», «проверь inbox»)
3. Директор прочитает, вытащит полезные знания, обновит базу
4. Обработанный файл переедет в `inbox/archive/` с датой в имени

## Формат файлов

- **MD файлы** — конспекты, транскрипции, заметки, статьи
- **PDF файлы** — презентации, документы (Claude прочитает напрямую)
- **Имя файла** — описательное, например:
  - `урок-15-бот-копирайтер.md`
  - `статья-промтинг-для-бизнеса.md`
  - `идея-квиз-для-санатория.md`

## Метаданные (необязательно, но помогает)

В начале MD-файла можно добавить:

```yaml
---
источник: Нейроуниверситет / статья / свои заметки
тема: промтинг / продукты / клиенты / контент
приоритет: высокий / средний / низкий
тип: урок / статья / идея / кейс
---
```

## Что НЕ класть сюда

- Файлы `.env`, токены, пароли, ключи API
- Огромные файлы (если транскрипция очень длинная — разбей на части)
- Файлы которые уже обработаны и лежат в knowledge/

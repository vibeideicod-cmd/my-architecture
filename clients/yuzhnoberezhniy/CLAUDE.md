# Проект: Санаторий Южнобережный (ЮБ)

## Контекст клиента

- **Название:** ГБУ РК СДДР «Южнобережный»
- **Сфера:** Детский санаторий, Крым, г. Алупка
- **Сайт:** ujnoberejniy.ru
- **Этап:** MVP — разработка интерактивных инструментов

## ВАЖНО: стиль и цвета

**НЕ использовать** цвета команды "Нейро Бабки СССР" (тёмный фон, оранжевый, горчичный).

Цвета ЮБ — светлые, морские:
- Фон: `#ffffff` (белый)
- Акцент: `#0693e3` (синий)
- Кнопки/тёмное: `#32373c`
- Текст: `#000000` / `#2c2c2c`

Полный бренд-стандарт: `brand.md`

## Специфика проекта

Аудитория двойная: **родители** (тревожные, принимают решение) и **дети** (пользуются).
Тон — тёплый, заботливый, без казённости.
Слоган: "Безопасно, полезно и с улыбкой каждый день"

## MVP (4 продукта)

1. **"Собери рюкзак"** — интерактивный чеклист сборов на смену
2. **Квиз "Готов ли ребёнок к смене?"** — снятие тревоги родителей
3. **Экскурсия по ЮБ** — страница-знакомство с санаторием
4. **Оживление сайта** — улучшения ujnoberejniy.ru

## Корневые правила

Вёрстка: один HTML-файл, стили внутри `<style>`, без внешних фреймворков.
Адаптивность обязательна.
Шрифт: Arial или системный sans-serif (как на сайте ЮБ).

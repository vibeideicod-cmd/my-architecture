## Промпт (копировать целиком)

````
Сделай мне одностраничную цифровую визитку дизайнера в стиле Awwwards-портфолио. Один HTML-файл с CSS+JS внутри, без фреймворков и CDN кроме Google Fonts.

КТО:
Ирина Цепаева. Дизайнер, 5+ лет опыта. Делает брендбуки, фирменный стиль, лендинги, мини-приложения внутри Telegram и VK, AI-помощников. Целевая аудитория — собственники готового бизнеса, премиум-сегмент. На «вы», тёплый премиум, без канцелярита. Сигнатура: «Слышу, какого человек цвета».

ЦЕЛЬ:
За 2 секунды показать «супер-дизайнер» визуально, до прочтения текста. Главная конверсия — нажатие на Telegram или VK.

ПАЛИТРА (только эти 8 цветов, ничего другого):
:root {
  --pine: #306654;       /* Сосновая хвоя — основной текст */
  --pine-light: #A1BF5B; /* Молодая зелень */
  --coral: #FF935E;      /* Тёплый коралл — главный акцент */
  --lemon: #F8EB00;      /* Лимонная цедра — точечный wow */
  --vanilla: #FCFAE1;    /* Ваниль — основной фон */
  --neon: #13F740;       /* Неоновый зелёный — wow-акцент */
  --acid: #4DAA02;       /* Кислотно-зелёный — CTA-вариант */
  --magenta: #FF2EC4;    /* Маджента — характер бренда */
}

Триада 60/30/10: Ваниль фон → Хвоя текст → Коралл акцент. Маджента и лимон — точечные wow-моменты, минимум 2-3 раза каждый. Тени ВСЕГДА цветные из палитры (rgba от coral/magenta/pine), никаких серых rgba(0,0,0,0.x).

ШРИФТЫ:
Playfair Display (display, 700-800) + Montserrat (body, 300-700) — Google Fonts.

7 СЕКЦИЙ:

1. HERO (полноэкранный, mesh-сильный)
   Фон: 4-5 radial-gradient на ванили — основной коралл opacity 0.95, точки лимона 0.40 и мадженты 0.45. Триада должна быть видна сразу.
   H1: «Слышу, какого человек цвета.» — Playfair Display 800, font-size: clamp(3rem, 1rem + 9vw, 9rem), letter-spacing -0.04em, drop-shadow 0 20px 60px rgba(255,147,94,0.55) (КОРАЛЛ, не серая).
   H2: «Создаю фирменный стиль и интерфейсы из разговора с человеком» — Montserrat 500, 22-32px.
   Статус: «Открыт 1 слот на май. Старт следующего — июнь.» — Montserrat 400, 14px, цвет хвоя 60%.
   Слева 60% — текст. Справа 40% — фото в КРУГЕ 480-560px с inset-glow коралл (фото загружу прикреплённым).
   ОДИН primary CTA «Написать в Telegram» — фон #229ED9 (фирменный голубой Telegram), иконка Telegram слева.
   Page-load orchestration: H1 60ms → H2 140ms → status 220ms → CTA 300ms → portrait 380ms.
   Custom cursor 24×24px коралловый кружок ТОЛЬКО в hero (на touch отключается через @media pointer:fine + JS).
   Magnetic effect на CTA: при курсоре в радиусе 120px кнопка притягивается на ±10px через requestAnimationFrame.

2. КТО Я (drop cap «И»)
   Drop cap «И» — Playfair Display Bold, 8em, цвет хвоя, float left.
   Эссе на Montserrat 400 18px, max-width 60ch: «Иногда дизайн рождается из одного разговора. Я слышу человека — его интонацию, темп, паузы — и за этим уже видна палитра. Превращаю «сделайте красиво» в чёткую визуальную систему, которая работает на бизнес-задачи.»
   Соц-доказательство: «5+ лет в дизайне. 8 ниш бизнеса.»

3. BREAK-OUT СЕКЦИЯ 100vw (между «Кто я» и услугами)
   Ширина 100vw (через width:100vw; margin-left:calc(50% - 50vw)).
   Фон: linear-gradient(135deg, var(--magenta), var(--coral)).
   В центре цитата ванилью, font-size: clamp(2rem, 1rem + 6vw, 8vw), Playfair Italic 700: «Палитра — это голос бренда.»
   Слово «голос» — градиент в lemon (background-clip: text).
   min-height: 60vh.

4. ЧТО Я ДЕЛАЮ (3 цветные карточки, БЕЗ ЦЕН)
   Сетка 3 в ряд десктоп, стек мобильный. Карточки одной высоты, скруглённые 16px.
   а) «Брендбуки и фирменный стиль» — иконка-палитра коралл, текст: «Палитра, типографика, логотип, гайдлайны. Чтобы все ваши соцсети, сайт и презентации выглядели одним брендом.» Срок: 1-3 дня.
   б) «Цифровые продукты» — иконка-окно браузера хвоя, текст: «Лендинги, мини-сайты, Telegram Mini App, VK Mini App, боты для записи, AI-помощники. Делаю под ключ — от макета до публикации.» Срок: 1-2 дня для лендинга, 2-5 дней для мини-аппа.
   в) «Упаковка соцсетей» — иконка-облако-сообщение маджента, текст: «Аватары, баннеры, обложки, шаблоны постов. Чтобы экспертный профиль выглядел собранно и узнавался с первого касания.» Срок: 1-2 дня.
   3 РАЗНЫЕ SVG-иконки (тонкие линии stroke 1.5-2px), не одинаковые.
   Hover: translateY(-2px) + усиление цветной тени. Маджента-карточка при hover заливается ВСЯ мадженой, текст становится ванильным.

5. ИНТЕРАКТИВНАЯ ДЕМКА «СЛЫШУ ЦВЕТ»
   H2: «Слышу цвет — попробуйте»
   Подсказка: «Выберите 3 ощущения — покажу палитру, которая вам отзовётся».
   12 чипов-эмоций (КАЖДЫЙ со своим цветом-glow в неактивном состоянии):
   - Тёплый (coral glow), Свежий (neon glow), Спокойный (pine glow), Дерзкий (magenta glow)
   - Лёгкий (lemon glow), Глубокий (pine-light glow), Точный (acid glow), Мягкий (vanilla-cream)
   - Природный (pine glow), Игривый (magenta glow), Системный (pine glow), Праздничный (coral glow)
   В неактивном: фон ваниль, бордер 2px цвета эмоции, box-shadow 8-12px цвета эмоции.
   Активный (выбран): фон цвета эмоции, текст ваниль.
   Лимит 3: при выборе 4-й деактивируется первая выбранная (FIFO).
   Кнопка «Показать палитру» активна при выборе 3.
   При нажатии — show результат: mesh-блок 100% ширины (4 radial-gradient), 5 точек с HEX-кодами палитры, название и описание палитры, объяснение «почему эта».
   После результата — primary CTA «Хочу свой бренд в этой палитре — написать», ведёт в Telegram с автотекстом «Здравствуйте, Ирина! Ваш сайт показал палитру [название]. Хочу обсудить бренд в этом настроении.»
   Можешь сам придумать 5 палитр-результатов с алгоритмом матчинга по выбранным эмоциям.

6. МАНИФЕСТ-ЦИТАТА (тёмная секция)
   Фон: linear-gradient(135deg, var(--pine) 0%, #1d4030 100%) + noise overlay 0.05.
   Цитата: «Я не работаю по референсам вслепую. Я слушаю — и за разговором уже видна палитра. Каждый бренд я собираю от человека, а не от «сделайте как у тех ребят».»
   Playfair Italic 700, font-size: clamp(2.5rem, 1rem + 6vw, 6.25rem), цвет ваниль.
   text-align: left, max-width: 16ch (off-grid — правые 40% пустые).
   Drop cap «Я» 8em с -webkit-text-stroke: 2px var(--magenta) и color: transparent (outline-маджентой).
   Слово «слушаю» — РЕАЛЬНЫЙ градиент coral → magenta (background-clip: text).

7. СТУДИЯ В VK
   H2: «Посмотреть, как я думаю».
   Текст: «В студии — посты, идеи, готовые работы. Заходите — там видно, как я разбираю задачи и собираю визуалы.»
   Большая кнопка «Студия в VK →» — фон #0077FF (фирменный VK), ведёт на https://vk.ru/stydiya.

8. СВЯЗАТЬСЯ
   H2: «Напишите. Куда удобнее.»
   Две одинаковые кнопки рядом:
   - Telegram (primary): фон #229ED9, иконка TG, ведёт на https://t.me/irina_onlane_gesing
   - VK (secondary): фон #0077FF, иконка VK, ведёт на https://vk.com/irina_1369
   Под кнопками: «Обычно отвечаю в течение дня. Будни 9:00–22:00 МСК.» (Montserrat 400, 13px, хвоя 60%).

ДОПОЛНИТЕЛЬНО:
— Sticky-bar внизу мобильного с иконками TG и VK на всех экранах кроме hero. С safe-area-inset-bottom для iOS.
— Scroll-snap proximity по крупным секциям на десктопе (на мобильном отключить, manifest-секцию исключить).
— Text-split reveal: каждое слово в H1/H2 разбивается на span, появляется через clip-path: inset(0 100% 0 0) → inset(0 0 0 0) 600ms ease-out, задержка 80-120ms между словами.
— prefers-reduced-motion: все анимации отключаются.
— color-scheme: light only (ванильный фон не должен ломаться тёмной системной темой).

ЧТО КАТЕГОРИЧЕСКИ НЕЛЬЗЯ:
× Шрифты Inter / Roboto / Arial / system-ui (только Playfair Display + Montserrat)
× Серые тени rgba(0,0,0,...) — все тени с цветом палитры
× Фиолетовые градиенты на белом (это AI slop)
× Шаблонная раскладка «3 одинаковые карточки в ряд + центральный CTA» — карточки должны быть разные
× Слова «провожу аудит / консультирую / сопровождаю» в любых текстах (она самозанятая, у неё нет ОКВЭД 70.22)
× Цены в карточках услуг (только сроки в днях)
× CAPS LOCK на длинных фразах (только короткие шильдики 1-3 слова)
× 3D-эффекты, Three.js, WebGL, геймификация (это не её бренд)
× Холодный минимализм Vogelaar-стиля «белый + чёрный + один акцент» — должна быть видна вся триада
× Header с навигацией сверху — одностраничник без шапки
× Pop-up «подпишись», auto-play видео

УРОВЕНЬ:
Ровняйся на awwwards-портфолио: art-yakushev.com (типографика-герой + sequential reveal), markvogelaar.com (один доминирующий цвет = идентичность), gsproductions.co.za (полноэкранный hero + асимметрия), elliott.mangham.dev (двойной CTA + статус доступности). Не «приличный лендинг», а «дизайнерский манифест».

H1 должен быть КРИКОМ (144px+ на десктопе), drop cap «И» 8em — драматичным, манифест-цитата 80-100px — off-grid (текст слева, правые 40% пустые), портрет 480-560px — большим. Подсажено по размеру = посредственность.

Формат вывода: один HTML-файл с CSS и JS внутри, готовый для скачивания и публикации.
````

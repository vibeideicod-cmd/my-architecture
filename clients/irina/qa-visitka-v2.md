### Правка 2 — Hero-фон, настоящая доминанта · ✅ реализовано (Approach B)

**Выбран Approach B (Mesh-сильный).**

**Обоснование выбора B vs A:**
- Архетип Ирины — «тёплое премиум, не цифровая стерильность» (`visual-language.md` §0). Approach A (весь hero коралл 80–90%) убивает ванильную тёплую базу, на которой стоит вся идентичность бренда.
- Бренд держится на ТРИАДЕ: ваниль 60–70% + хвоя 25–30% + коралл 10–15%. Approach A заливает всё одним цветом — это идентичность Vogelaar, не Ирины.
- Approach B даёт «триаду видна за 1 секунду» через мощный коралл-mesh + точки маджента 0.45 + лимон 0.40 — без жертвы ванильной основы.
- Маджента и лимон в v1 были ниже 0.20 / отсутствовали — теперь 0.45 / 0.40 видны явно.

**Hero-фон в коде (строки 264–271):**
```css
background:
  radial-gradient(at 20% 25%, rgba(255, 147, 94, 0.95) 0px, transparent 60%),  /* коралл 0.95 — доминанта */
  radial-gradient(at 78% 18%, rgba(255, 46, 196, 0.45) 0px, transparent 50%),   /* маджента 0.45 */
  radial-gradient(at 55% 88%, rgba(248, 235, 0, 0.40) 0px, transparent 48%),    /* лимон 0.40 */
  radial-gradient(at 92% 78%, rgba(48, 102, 84, 0.55) 0px, transparent 55%),    /* хвоя 0.55 */
  radial-gradient(at 5% 75%, rgba(255, 46, 196, 0.30) 0px, transparent 45%),    /* маджента-эхо */
  var(--vanilla);
```

Триада коралл/маджента/лимон видна «не угадывается».

### Правка 7 — Text-split reveal + clip-path stagger · ✅ реализовано

**JS (строки 1638–1646):** автоматически разбивает каждый `.split-words` (заголовки H2 в about, services, color-listen, studio-vk, contact) на `<span class="word">…</span>` после загрузки.

**CSS (строки 224–252):**
- Каждое слово стартует с `clip-path: inset(0 100% 0 0)` + `translateY(0.15em)`
- При `.in-view` (через IntersectionObserver) — `clip-path: inset(0 0 0 0)` за **700ms** ease-soft
- Stagger между словами **90ms** (через `nth-child` 1–12)

Также апгрейднут общий `.reveal` (строки 215–222):
- v1: `translateY(20px) → 0` за 250ms
- v2: `translateY(24px) + scale(0.98) + blur(6px) → none` за **600ms**

## Что не получилось / отложено

- **Palette-name в результате демки** — оставлена `font-style: italic` Playfair, но не апгрейднута до 80px (это блок результата под кнопкой, посетитель уже взаимодействует — не «вау»-точка). Сделана крупнее v1 (clamp 1.6 → 2.4rem max), но не «манифестно».
- **About-aside премиум-карточка** — упрощена визуально до 3 теней (pine + coral + magenta + inset). Не добавлен лимон как 4-й слой, чтобы не спорить с break-out секцией ниже (там лимон — главный акцент).
- **Лимон 5+ мест** перевыполнено (7 точек) — но ни одна не «лимонный CTA» или «лимонная плашка». Лимон остаётся вспомогательным glow-цветом, не доминантным. Это решение по brand-safe (`brandbook/dont.md` запрещает лимон как фон длинного текста).

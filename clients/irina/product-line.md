# ЧАСТЬ 1. ПРИНЦИПЫ И СТРАТЕГИЯ

## 1.1. Стратегия 80 / 19 / 1

| Слой | % фокуса | Кто | Что покупает |
|---|---|---|---|
| 🟢 80% | 80% времени Ирины | Самодостаточный бизнес | Крупные продукты 50к–1млн+ |

**Логика воронки:** массовая воронка через слой 1 → база клиентов → прогрев в слой 19 → часть слоя 19 в слой 80.

## 1.2. 5 болей клиента (для слоя 80%)

| Боль | Что Ирина создаёт |

## 1.3. 🌐 Принцип омни-канальности — формат как воронка удорожания

> **Правило (от Инны):** один и тот же продукт может жить в нескольких форматах (PDF / Веб / Telegram / VK), и **формат сам по себе — это уровень монетизации**. Чем технологичнее формат, тем дороже. Это даёт продукту собственную лестницу цен.

**Лестница форматов одного продукта:**

| Формат | Что это | Уровень цены | Кому |
|---|---|---|---|
| 📄 **PDF / документ** | Простой PDF, скачивает и пользуется | Бесплатно или 199-300₽ | Захват в базу, быстрый микро-доход |
| 🌐 **Веб-версия** | Интерактивная страница на сайте Ирины | 300-700₽ разово или 200-500₽/мес | Слой 1%, удобство «здесь и сейчас» |
| 📱 **Telegram (TMA / бот)** | Полноценный мини-апп или бот в TG | 500-1500₽ разово или 300-800₽/мес | Слой 1-19%, кто живёт в Telegram |
| 💙 **VK (VK Mini App / бот)** | Полноценное VK Mini App или бот | 500-1500₽ разово или 300-800₽/мес | Слой 1-19%, кто в VK |
| 🎯 **Подписка-платформа для эксперта (готовая)** | Все форматы вместе + кабинет эксперта + кабинеты его учеников + общая статистика | **1000-3000₽/мес** или 3-9к разово | Самозанятые эксперты для своих учеников (нутрициолог, психолог, коуч) |

> ⚠️ **Важно различать:** «Подписка-платформа для эксперта» = **готовый продукт массового тиражирования** (Ирина один раз сделала — продаёт многим экспертам по подписке). Не путать с **индивидуальной разработкой платформы под крупного клиента** (онлайн-школа, фитнес-клуб) — это уже средний/крупный сегмент 30-50к и выше, см. S-N7-S-N11.

**Почему это работает:**
- Один продукт = **5 ценовых ступеней** (PDF → Веб → TG → VK → Подписка для эксперта)
- Клиент сам выбирает уровень, до которого хочет дойти
- Самозанятый эксперт (нутрициолог, фитнес-тренер, коуч) выбирает «подписку-платформу» за 1-3к/мес — даёт своим ученикам как часть курса
- Пробовал PDF за 199₽ → понравилось → апгрейд до Веб → ещё дальше

**Пример развёртывания одного продукта по форматам:**

«Дневник питания с геймификацией» (M-N5):
- 📄 **PDF-дневник** (ручное заполнение, 299₽) — самозаполняющий клиент
- 🌐 **Веб-версия** (с подсчётом баллов, 500₽) — индивидуальный пользователь
- 📱 **Telegram-бот** (напоминания + игровая механика, 800₽) — для активного пользователя
- 💙 **VK-приложение** (то же + интеграция со страницей, 800₽) — кто в VK
- 🎯 **Подписка-платформа для нутрициолога** (Все форматы + кабинет нутрициолога + кабинеты его учеников + статистика по группе, **1500-3000₽/мес**) — готовый продукт для самозанятого эксперта

> Если нутрициологу нужна **индивидуальная** разработка с уникальным дизайном, своим брендом, кастомными функциями — это уже не подписка-шаблон, а средний сегмент (S-N10 Игровой дневник питания/тренировок индивидуально, 35-50к). Это два разных продукта в линейке.

**Где работает (можно развернуть в воронку форматов):**
- Все генераторы (палитра, шрифт, имя бренда) — PDF + Веб + TG-бот
- Все калькуляторы и дневники — все 5 ступеней
- Платформы геймификации — обязательно все форматы для премиум-варианта

**Где не работает (один формат):**
- Лендинги (только Web)
- Брендбуки (документ)
- Контент-сопровождение (услуга)
- Чистые гайды-самоучки (только PDF)

**Пометка в каталоге:** в столбце «Формат» у каждого продукта указано — `📄` / `🌐` / `📱` / `💙` / `🎯`. Большинство интерактивных = вся лестница `📄→🌐→📱→💙→🎯` с разными ценами на каждой ступени.

## 1.4. Голос бренда vs голос юридический

**Голос наружу (на лендингах, в соцсетях, в текстах):**
> «Слышу клиента → создаю под него → передаю готовый продукт.»

**Голос в чеках, оферте и договорах** — только разрешённые виды деятельности самозанятой:

| Юр.конструкция | Что покрывает |
|---|---|
| **«Изготовление дизайна»** | Брендбук, фирстиль, визитка, иллюстрации, макеты |
| **«Разработка веб-приложения»** | Лендинг, квиз, мини-сайт, мини-приложения внутри Telegram и VK |
| **«Разработка чат-бота»** | Бот для записи, бот-прогрев, AI-помощник под клиента |
| **«Вёрстка веб-страницы»** | HTML-вёрстка лендингов, цифровых визиток |
| **«Иллюстрация и художественное оформление»** | Иллюстрации, серии, баннеры |

**Запрещённые слова:** «провожу аудит», «консультирую», «разрабатываю стратегию», «управляю проектом», «сопровождаю внедрение». Подробно — `legal-okved.md`.

# ЧАСТЬ 3. КАТАЛОГ ПРОДУКТОВ — ПУЛ ИДЕЙ К ВЫБОРУ

## 🌟 Главный принцип всех продуктов — «вау, как круто»

> **Правило (от Инны):** любой продукт Ирины должен **кричать о технологичности**, а не быть будничным. Не просто бот / калькулятор / шаблон — а **игровой, увлекающий, с эффектом «вау»**. Клиент должен хотеть похвастаться: «смотри что я сделал за копейки», «смотри какой инструмент я даю своим ученикам».

**Что это значит на практике:**
- Любой калькулятор / трекер / дневник → **геймификация** (баллы, уровни, серии, бейджи)
- Визуальный результат — анимации, прогресс-бары, празднования достижений
- Не «введи цифры → получи цифры», а «введи → увидь визуальный результат → захоти показать друзьям»
- Все продукты — **с шильдиком «работает на нейросетях / ИИ»** (там где это правда), это маркер новизны
- Личный кабинет везде, где есть смысл — кабинет тренера + кабинет ученика, кабинет эксперта + кабинет его клиентов

**Что НЕ берём в линейку:**
- Будничные финансовые калькуляторы «для себя» (личный бюджет, налоги, бюджет свадьбы) — это считать цифры, скучно, не Иринина зона
- Скучные шаблоны без интерактива
- Боты, где «спросил — ответил» без эмоции

## ⚠️ Открытые вопросы перед запуском

### Экономика микро-продуктов 🔴

Прежде чем запускать любой микро-продукт за 100-500₽, **через Financial-агент посчитать юнит-экономику**. Вопросы:

- **Себестоимость операции:** комиссия платёжной системы (~3%), стоимость API-вызовов (для AI-генераторов), хостинг — сколько остаётся с одного оплаченного продукта в 200₽?
- **Нижний порог цены:** при какой цене продукт **уходит в минус**? (если AI-вызов стоит 50₽, а продукт за 200₽ → плохая экономика)
- **Минимальный тираж:** сколько штук нужно продать в месяц чтобы микро-направление окупало хостинг + поддержку?
- **Группировка:** возможно, выгоднее **подписка на доступ ко всем микро-инструментам сразу** (например, 999₽/мес = безлимит на все генераторы), чем разовые продажи

**Задача Financial:** просчитать модель для типового микро-продукта (палитра-генератор / гайд-самоучка / калькулятор) и дать «зелёный свет» / «красный» / «нужен рефакторинг цены».

### Технический формат реализации 🟡

Часть микро-продуктов проще как **отдельные мини-сайты** (один лендинг = один продукт), часть — как **разделы на одной платформе** «Сделать сейчас». Решение принимается через Systems-агент + DevOps/Infra при сборке Слоя 5.

### Юнит-экономика средних продуктов 🟡

Боты-администраторы 30-40к — нужно посчитать через Pricing-Specialist:
- сколько часов команды на типовой бот?
- какая маржа на одну продажу?
- какая модель сопровождения после сдачи (подписка / разовый / гибрид)?

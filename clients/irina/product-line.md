# ЧАСТЬ 1. ПРИНЦИПЫ И СТРАТЕГИЯ

## 1.1. Стратегия 80 / 19 / 1

| Слой | % фокуса | Кто | Что покупает |
|---|---|---|---|
| 🟢 80% | 80% времени Ирины | Самодостаточный бизнес | Крупные продукты 50к–1млн+ |
| 🟡 19% | 19% времени | Сомневающиеся, осторожные | Средние 5-40к + подписка на Платформу |
| 🟠 1% | 1% времени (автомат) | Массовая B2C-аудитория | Микро-продукты 100-1000₽ |

**Логика воронки:** массовая воронка через слой 1 → база клиентов → прогрев в слой 19 → часть слоя 19 в слой 80.

## 1.2. 5 болей клиента (для слоя 80%)

| Боль | Что Ирина создаёт |
|---|---|
| **«Меня не видят»** | Брендбук + упаковка соцсетей + визитка |
| **«Нет клиентов»** | Лендинг + квиз + бот для записи |
| **«Нет времени»** | AI-помощник под клиента + бот-прогрев + воронка |
| **«Нет структуры»** | Дизайн-система + гайдлайны + регламенты с инфографикой |
| **«Хочу всё»** (флагман) | Полная визуальная экосистема под ключ |

> 🔵 **Открытое:** на сверочной встрече решается расширение до 8 болей (+3 визуальных).

## 1.3. Голос бренда vs голос юридический

**Голос наружу (на лендингах, в соцсетях, в текстах):**
> «Слышу клиента → создаю под него → передаю готовый продукт.»

**Голос в чеках, оферте и договорах** — только разрешённые виды деятельности самозанятой:

| Юр.конструкция | Что покрывает |
|---|---|
| **«Изготовление дизайна»** | Брендбук, фирстиль, визитка, иллюстрации, макеты |
| **«Разработка веб-приложения»** | Лендинг, квиз, мини-сайт, мини-приложения внутри Telegram и VK |
| **«Разработка чат-бота»** | Бот для записи, бот-прогрев, AI-помощник под клиента |
| **«Вёрстка веб-страницы»** | HTML-вёрстка лендингов, цифровых визиток |
| **«Иллюстрация и художественное оформление»** | Иллюстрации, серии, баннеры |

**Запрещённые слова:** «провожу аудит», «консультирую», «разрабатываю стратегию», «управляю проектом», «сопровождаю внедрение». Подробно — `legal-okved.md`.

# ЧАСТЬ 2. ПУТЬ КЛИЕНТА И СТРУКТУРА ВЫРУЧКИ

## 2.1. Лестница 80 / 19 / 1

```
🟠 СЛОЙ 1 — массовый вход (раздел «Сделать сейчас» на сайте)
   ↓
   10-15 микро-продуктов автоматом: 100-1000₽
   Клиент попал в базу
   ↓ контент-прогрев + рассылки

🟡 СЛОЙ 19 — средний (мини-проекты + Платформа С.С.С.Р. в визуале)
   ↓
   Мини-визитка / лендинг / мини-приложение: 5-40к
   Подписка на Платформу: ежемесячный тариф
   Клуб: разборы, поддержка
   ↓ увидел качество, поверил процессу

🟢 СЛОЙ 80 — основной (крупные продукты по 5 болям)
   ↓
   Брендбук / сайт / автосистема / флагман: 50к-1млн+
   Кейс для портфолио → возврат рекомендаций
```

## 2.2. Структура выручки

| Тип | Слой ЦА | Цель |
|---|---|---|
| Микро-выручка (массовый автомат) | Слой 1 | Массовое наполнение базы + микро-доход без участия Ирины |
| Рекуррент (MRR) | Слой 19 + часть 80 | Стабильный доход (Платформа, контент-сопровождение, техподдержка) |
| Разовые проекты | Слой 19 (мини), Слой 80 (крупные) | Основная выручка по чеку |
| Бесплатный вход | Слой 19 → 80 | Прогрев в основной слой |
| Флагман | Слой 80 | Высокий чек, длинный проект, эталонный кейс |

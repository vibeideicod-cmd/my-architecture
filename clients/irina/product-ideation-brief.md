# ЗАДАНИЕ

## Контекст

**Ирина Цепаева** — Дизайнер, самозанятая. Виды деятельности: дизайн графический (74.10), иллюстрация (90.03), разработка ПО (62.01). Авторская методология «**С.С.С.Р. в визуале**» — Структура · Стиль · Свобода · Результат. Сигнатурная фраза: «Слышу, какого человек цвета».

## Стратегия трёх слоёв ЦА (80 / 19 / 1)

Целевая аудитория Ирины делится на три слоя по уровню готовности:

| Слой | % фокуса | Кто | Что покупает | Как работаем |
|---|---|---|---|---|
| **🟢 80%** | 80% времени | Самодостаточный бизнес — основной фокус. Готовы платить за качество | Крупные продукты: брендбук + сайт + AI + автосистемы (50к–1млн+) | Личное взаимодействие: встречи, разборы, длинные проекты |
| **🟡 19%** | 19% времени | Сомневающиеся, осторожные. Хотят попробовать перед крупным | Средние продукты (5–40к) + подписка на Платформу С.С.С.Р. в визуале | Лёгкое сопровождение: клуб, ежемесячные разборы, прогрев в основной слой |
| **🟠 1%** | 1% времени (автомат) | Массовая B2C-аудитория: родители, ученики, любые люди — «зашёл и ушёл» | **10–15 микро-продуктов на автомате** (100–1000₽) | Без личного контакта — клиент использует автомат и попадает в базу |

**Логика воронки:** массовая воронка через слой 1 наполняет базу клиентов → прогрев в средний слой 19 → часть среднего слоя растёт в основной 80.

## Что уже зафиксировано в архитектуре Ирины

### 5 болей клиента (для слоя 80%)

| Боль | Что Ирина создаёт |
|---|---|
| **«Меня не видят»** (нет упаковки) | Брендбук + упаковка соцсетей + визитка |
| **«Нет клиентов»** (нет потока) | Лендинг + квиз + бот для записи |
| **«Нет времени»** (живёт 24/7) | AI-помощник под клиента + бот-прогрев + воронка |
| **«Нет структуры»** (хаос) | Дизайн-система + гайдлайны + регламенты с инфографикой |
| **«Хочу всё»** (флагман) | Полная визуальная экосистема: сайт + AI + мини-аппы + автоматизации |

### Платформа С.С.С.Р. в визуале (для слоя 19%, Слой 6)

Закрытый подписной продукт. 4 модуля: Структура · Стиль · Свобода · Результат. 3 тарифа: База / Работа / VIP. AI-помощник в контексте клиента + клуб с разборами.

### 8 целевых ниш

1. Инфобизнес и онлайн-образование
2. Финансовые консультанты и брокеры
3. Бьюти (салоны, барбершопы, косметология, СПА)
4. Коучи, психологи, наставники
5. Фитнес-клубы и тренеры
6. Торговля — офлайн и онлайн
7. Риэлторы и агентства недвижимости
8. Санатории и отели

## ЗАДАЧА — три таблицы продуктов

### Таблица 1. 🟠 Микро-продукты для слоя 1% (10–15 штук) — ПРИОРИТЕТ

Это самая важная таблица. Сгенерируй **10–15 микро-инструментов на полном автомате** для раздела **«Сделать сейчас»** на сайте Ирины. Цена 100–1000₽. Без личного контакта Ирины. Цель — наполнение базы клиентов массово.

**Категории для поиска идей:**
- **Микро-визуал:** цифровые визитки, генераторы палитры, генераторы шрифтовых пар, генераторы аватаров, генераторы баннеров для соцсетей
- **Микро-документы:** шаблоны презентаций (для ребёнка / выпускника / коуча / стартапа), КП-шаблоны, сертификаты, открытки
- **Микро-калькуляторы:** калькулятор стоимости брендинга, калькулятор бюджета на упаковку, калькулятор количества постов в месяц, калькулятор размеров макетов
- **Микро-боты / AI-инструменты:** бот-проверка идеи бренда, бот-генератор слогана, AI-аудит визуала по фото, AI-генератор цветовой схемы по описанию
- **Микро-конструкторы:** конструктор визитки за 5 минут, конструктор простого лендинга, конструктор поста для соцсетей

**Критерии:**
- ✅ Полный автомат (без участия Ирины после оплаты)
- ✅ Результат за 5 минут или быстрее
- ✅ Универсально (подходит любому из 8 ниш + B2C-аудитории)
- ✅ Решает простую конкретную потребность (открытка для ребёнка, калькулятор бюджета, генератор палитры)
- ✅ Цена 100–1000₽

**Формат таблицы:**

| # | Название | Что делает (1 предложение) | Что получает клиент | Цена (₽) | Технически: как реализовать (1 фраза) | Целевая микро-аудитория |

### Таблица 2. 🟡 Средние продукты для слоя 19% (10–15 штук)

Недорогие проекты + подписки + дополнительные услуги. Цена 5–40к за разовый или подписка от 1000₽/мес. Здесь Ирина включается частично (онбординг, разбор, поддержка). **Должны иметь компонент сопровождения.**

**Категории:**
- Мини-проекты под одну задачу (визитка-проект, лендинг под продукт, мини-бот)
- Тарифы Платформы С.С.С.Р. в визуале (3 тарифа уже зафиксированы — нужны идеи дополнительных сервисов внутри)
- Дополнительные подписные сервисы (контент-сопровождение, библиотека шаблонов с обновлениями)
- Образовательные продукты (мини-курсы по визуалу, гайды, чек-листы за плату)

**Формат таблицы:**

| # | Название | Тип (разовый / подписка / гибрид) | Что делает | Боль (1–5) | Что получает клиент | Что Ирина делает (вкладка времени) | Цена разового (₽) | Цена подписки (₽/мес) |

### Таблица 3. 🟢 Крупные продукты для слоя 80% (15–20 штук)

Расширение основной линейки по 5 болям + флагманы. Цена 50к–1млн+. Это её любимая работа — личные встречи, длинные проекты, кейсы.

**Категории:**
- Расширение под каждую из 5 болей (минимум по 3 продукта на боль)
- Нишевые версии продуктов (брендбук для бьюти, лендинг для коуча, мини-апп для фитнес-клуба)
- Флагманские пакеты (брендбук + сайт + соцсети + AI; полная экосистема под ключ)
- Эксклюзивные форматы (дизайн под выступление, презентация для инвестора, дизайн-сопровождение запуска)

**Формат таблицы:**

| # | Название | Какая боль из 5 | Что входит | Подходит для нишы (одной или нескольких из 8) | Длительность проекта | Цена (₽) |

# Антипримеры — что НЕ Иринин бренд

> **Зона:** свод запрещённых приёмов в визуале, голосе, типографике, графике, продуктах и юридическом обороте. Накопительный документ — пополняется по мере встречи с реальными ошибками.
> **Принцип:** «нельзя» важнее «можно». Если бренд знает, чего избегает — он узнаваем. Если не знает — выглядит как все.
> **Версия:** MVP · 2026-05-01

---

## 1. Как пользоваться этим документом

1. **Перед публикацией** любой работы (пост, лендинг, PDF, дизайн) — пробежаться по разделам ниже.
2. **Если хотя бы 2 пункта совпадают** — не выпускаем, переделываем.
3. **Если встретили новую ошибку**, которой нет в списке — **добавляем сюда**. Документ растёт с каждой задачей.
4. **Каждое «нельзя» имеет пояснение «почему»** — иначе агент / новый человек не поймёт логику и нарушит.

## 2. Голос — что НЕ говорим

### 2.1. Корпоративный жаргон ❌

| Запрещено | Почему |
|---|---|
| «Мы предлагаем» | Ирина — самозанятая, не «мы». «Я предлагаю». |
| «Компания осуществляет» | Сразу «холодное» лицо. |
| «Реализуем», «обеспечиваем», «осуществляем» | Канцелярит, мёртвая речь. |
| «Эффективное решение» | Пустое слово, ничего не говорит. |
| «Комплексный подход» | Шаблон из всех маркетинговых текстов. |
| «Высококачественный сервис» | Дешёвая риторика. |
| «Профессионализм и индивидуальный подход» | Каждый второй так пишет. |

### 2.2. Агрессивный продажный душок ❌

| Запрещено | Почему |
|---|---|
| «Купите сейчас!» | Тёплый бренд Ирины так не звучит. |
| «Только сегодня!» | Создаёт фальшивую срочность. |
| «Не упустите шанс!» | Манипуляция. |
| «Уникальное предложение!» | Каждое предложение «уникальное» — слово обесценено. |
| «Звоните прямо сейчас!» | Холодный продажный приём. |
| «Скидка только до конца дня» | Манипуляция дедлайном. |

### 2.3. Технический жаргон, которого нет в её речи ❌

| Запрещено | Почему | Замена |
|---|---|---|
| «UI/UX-композиция» | Ирина так не говорит | «Дизайн интерфейса» |
| «Модульная сетка», «гриды» | Профессиональный жаргон | «Сетка», «структура» |
| «Конверсия», «click-through rate» | В публичных текстах — нет | «Сколько людей купят», «как много кликают» |
| «Стэк», «фреймворк» | Не её зона | «Инструменты», «технология» |
| «User flow» | Не её слово | «Путь клиента», «как клиент идёт» |

### 2.4. Юридически запрещённое (смертельно важно) ❌

> ⚠️ **Жёсткое правило:** Ирина — самозанятая. У неё **нет** видов деятельности «аудит», «консультация», «стратегия», «сопровождение». Использование этих слов в чеках = **штраф ФНС**.

| Запрещено | Замена |
|---|---|
| «Провожу аудит визуала» | «Изготавливаю разбор визуала» / «Делаю разбор» |
| «Консультирую по бренду» | «Изготавливаю брендбук» / «Делаю дизайн» |
| «Разрабатываю стратегию» | «Изготавливаю дизайн-систему» |
| «Сопровождаю внедрение» | «Изготавливаю и передаю готовый продукт» |
| «Управляю проектом» | «Делаю проект под ключ» |
| «Веду клиентов» | «Делаю проекты для клиентов» |

Подробно — `legal-okved.md`.

### 2.5. Корпоративные обращения к аудитории ❌

| Запрещено | Замена |
|---|---|
| «Уважаемые коллеги», «дорогие друзья» | «Привет», обращение по имени |
| «Дамы и господа» | Никогда (это театр) |
| «Будем рады видеть вас» | «Жду в гости» / «Заходи на сайт» |
| «Имеем честь представить» | «Сегодня показываю...» |

## 3. Цвета — что НЕ используем

### 3.1. Вне палитры ❌

| Цвет | Почему запрещён |
|---|---|
| Синий (любой) | Не из палитры. Если нужен «холодный» — берём Хвою. |
| Красный | Не из палитры. Если нужен срочный акцент (error) — Коралл `#FF935E`. |
| Фиолетовый | Не из палитры. Если нужна женственность — Маджента `#FF2EC4`. |
| Чёрный `#000000` | Слишком контрастный. Используем Хвою `#306654` вместо чёрного. |
| Чисто-белый `#FFFFFF` | Стерильно, «холодно». Используем Ваниль `#FCFAE1`. |
| Серые тона `#808080` | Скучно, бренд теряет тёплый характер. |
| Золото `#D4AF37` | Это палитра Инны. Ирина — Коралл вместо золота. |

### 3.2. Сочетания вне правил ❌

| Сочетание | Почему запрещено |
|---|---|
| Неоновый зелёный + Кислотно-зелёный | Спорят, шум для глаза |
| Маджента + Лимон большими массами | Перегруз — оба громкие, нужно разводить |
| Мадженти как фон длинного текста | Нечитаемо |
| Лимон как фон длинного текста | Нечитаемо |
| Тёмные цвета (Хвоя, Кислотный) на тёмном фоне | Контраст < 4.5:1, accessibility-проблема |
| Светлые цвета (Ваниль, Лимон) на светлом фоне | Текст не читается |

### 3.3. Эффекты, портящие палитру ❌

- ❌ Радужные градиенты (5+ цветов в одном переходе)
- ❌ Псевдо-3D с тенями неоновых цветов
- ❌ «Голограмма» (переливающиеся радужные эффекты)
- ❌ Перекраска фотографий в фиолетово-розовый, синий или другие чужие тона

## 4. Типографика — что НЕ используем

### 4.1. Шрифты-табу ❌

| Шрифт | Почему |
|---|---|
| Comic Sans | Сразу «непрофессионально». Минус 90% доверия. |
| Papyrus | Театрально-кичевый. |
| Times New Roman | Шрифт «по умолчанию», не дизайнерский выбор. |
| Verdana, Tahoma | Системные «канцелярские» шрифты. |
| Готические шрифты (Old English) | Не наша эстетика. |
| Шрифты без поддержки кириллицы | Если шрифт не поддерживает русский — ищем альтернативу. |
| Ультра-тонкие веса (Hairline, Thin) на body | Не читается в реальных условиях |

### 4.2. Типографические приёмы-табу ❌

| Приём | Почему |
|---|---|
| CAPS LOCK на длинных фразах | Кричит, не Иринин голос. |
| **Bold + *Italic* + подчёркнутый** одновременно | Перебор |
| 4-5 разных весов на одном носителе | Шум |
| Растягивание `letter-spacing > 0.1em` на body | Нечитаемо |
| Капитализация Каждого Слова Как В Английском | Англицизм, портит русский текст |
| Подчёркивание для акцента (не для ссылок) | Подчёркивание — только для ссылок |
| Маленький H1 (24px на десктопе) | Не «дышит», бренд теряет масштаб |
| Body меньше 14px на мобильном | Не читается, accessibility проблема |
| Body меньше 16px на десктопе | Глаз устаёт |

### 4.3. Что выглядит «не дизайнерски» ❌

- Заголовки и body в одном размере
- Идеальное центрирование всего текста (теряется иерархия)
- Текст в один сплошной блок без отступов
- Жёсткое выравнивание по ширине (`text-align: justify`) на узких колонках — рваные дыры
- Дефис `-` вместо длинного тире `—` в авторских мыслях

## 5. Логотип и графика — что НЕ делаем

### 5.1. С логотипом ❌

| Что нельзя | Почему |
|---|---|
| Растягивать непропорционально | Логотип искажается |
| Менять цвета на не-палитру | Теряется бренд |
| Поворачивать (на бок, наклоном) | Логотип всегда горизонтально |
| Добавлять обводки, тени, эффекты | Замусоривает чистый знак |
| Размещать на ярких неоновых фонах (Лимон, Маджента) | Нечитаемо |
| Накладывать на сложное фото без полупрозрачной плашки | Сливается |
| Делать «чёрно-белую версию» в чёрном `#000` | Используем Хвою или Ваниль |
| Разрезать пополам или обрезать | Логотип — целостный элемент |

### 5.2. Графика и иллюстрации ❌

| Что нельзя | Почему |
|---|---|
| Сток-картинки «дизайнер за компьютером» | Бренд Ирины — не сток |
| Перекрашенный для другого клиента дизайн | «Шаблонность» — главное, что Ирина не любит |
| Резкие острые треугольники, агрессивные зигзаги | Не её визуальный язык |
| 3D-иконки с тенями | Skeumorphism, устарело |
| Иконки от 5 разных дизайн-систем в одном проекте | Material + Bootstrap + Flaticon = шум |
| Радужные эффекты, голографика, неоновые свечения | Не наша эстетика |
| Картины с болью, ужасом, мрачным настроем | Ирина не любит |
| Запредельная абстракция без читаемого смысла | Раздражает её, см. tov-source |
| Хаос всех цветов и стилей в одном изображении | Антипод бренда |

## 6. UI и интерфейсы — что НЕ делаем

### 6.1. Кнопки ❌

- Кнопки без скругления (прямоугольные с прямыми углами)
- 3D-кнопки с большими тенями
- Главная CTA в чёрном или сером (всегда — Кислотный или Хвоя)
- Кнопка без текста (только иконка) на главной CTA — нарушает accessibility
- Hover-эффект с поворотом (`transform: rotate`) — выглядит дёшево
- Цвета вне палитры (синие ссылки в стиле Bootstrap)

### 6.2. Формы ❌

- Input без скругления
- Поля высотой меньше 40px (на мобильном — меньше 44px) — пальцу некомфортно
- Шрифт в input меньше 16px на мобильном — iOS зумирует страницу
- Красный для error (вне палитры) — используем `#C84B3A` или Коралл
- Подсказки внутри input исчезающие при фокусе (placeholder-as-label) — accessibility-проблема, пользователь забывает что вводить

### 6.3. Лендинги ❌

- Лендинг без primary CTA — пользователь не знает что делать
- Несколько Primary-CTA в одной секции — глаз не знает куда смотреть
- Огромные параллакс-эффекты на каждой секции — отвлекает
- Auto-play видео со звуком — пользователь сразу уходит
- Pop-up «Подпишись на рассылку» через 5 секунд после входа — раздражает
- Текст белым по белому на сложном фото без оверлея — не читается
- Кнопки в подвале внутри тёмного фона без хорошего контраста

### 6.4. Анимации ❌

- Анимации длительностью больше 300ms — тормозят восприятие
- Бесконечные пульсирующие элементы — отвлекают
- Анимация на главной CTA, которая скрывает кнопку — пользователь не успевает кликнуть

## 7. PDF-продукты — что НЕ делаем

### 7.1. Оформление ❌

- Дефолтные Helvetica/Arial без бренд-палитры
- Чёрный текст `#000000` на чисто-белом `#FFFFFF` — стерильно
- Times New Roman 12pt — выглядит как студенческий реферат
- Заголовок 14pt в углу обложки — «не дышит»
- Подвал отсутствует на страницах — теряется бренд
- Нет блока «Не хочешь сам — закажи у нас» в гайдах и чек-листах — упускаем продажу

### 7.2. Содержание ❌

- Чек-лист с 50+ пунктами — никто не прочтёт, теряется польза
- Гайд с водянистым «введением на 3 страницы» перед сутью — клиент уйдёт
- Скриншоты в плохом разрешении — критично для дизайнера
- Иллюстрации из Pinterest без разрешения — юридические риски

### 7.3. Технические ❌

- Шрифт не embedded в PDF — у клиента откроется кривым
- Цветовая модель RGB при отправке в типографию — печать будет «не такая»
- Иллюстрации меньше 300 dpi для печати — пиксели в PDF
- Гиперссылки не активны — пользователь не сможет кликнуть

## 8. Соцсети — что НЕ делаем

### 8.1. Контент ❌

- Один пост на 3 разные темы — теряется фокус
- Посты «10 советов как...» — формат всех остальных, без характера
- Только репосты чужого контента — нет авторского голоса
- Реклама в каждом 2-м посте — аудитория уходит
- Сток-фото вместо своих работ — бренд теряется

### 8.2. Визуал постов ❌

- Текст 60-70% площади картинки — лента в VK / Telegram такой текст не читает
- Watermark с другого ресурса (Pinterest, Behance) — палевно
- Длинный текст «листайте дальше →» в одном посте — это карусель, отдельный формат
- Скриншот переписки с клиентом — нарушение приватности
- Меми из других каналов с чужими шрифтами/палитрами — не Иринин стиль

### 8.3. Реакции ❌

- «Здравствуйте! Свяжитесь с нашими менеджерами» в комментариях — корпоратив
- Игнорирование комментариев — теряется доверие
- Перепалка с критиками — токсично
- «Спасибо за обратную связь, мы учтём» — формальный шаблон

## 9. Продукты в линейке — что НЕ делаем

### 9.1. Запрещённые типы продуктов ❌

| Тип | Почему |
|---|---|
| **Мини-курсы / обучающие программы** | Нет лицензии на образовательную деятельность. Всегда рискуем штрафом. |
| **«Запуск продаж за 1 день» и подобные** | Продажный душок. ФНС или Роспотребнадзор могут предъявить. |
| **Гарантированные результаты** («заработаешь 500к за месяц») | Юридический риск, маркетинговая претензия. |
| **Продукты с использованием персональных данных без согласия** | ФЗ-152, штрафы. |

### 9.2. Шаблонные «генераторы» ради генерации ❌

| Что НЕ делаем | Почему |
|---|---|
| Генератор имени бренда | Игрушка без результата для клиента |
| Генератор слогана | То же |
| Генератор архетипа | Абстракция |
| Генератор Tone of Voice | Невозможно сгенерировать |
| Бот, который «спросил-ответил» без эмоции | Не Иринин стиль (см. `voice.md`) |

См. историю правок в `product-line.md` — это уроки v1.3.

### 9.3. Цены ❌

- Цены «до 1 миллиона» — у нас нет миллионных клиентов
- Цены без «от X» (жёсткий ценник на услугу) — не позволяет варьировать под бюджет
- Просадки в лестнице цен (1000₽ → 5000₽ скачок без полки) — клиент теряется
- Подписки на платформы 30-40к/мес — нет такой ЦА

## 10. Юридические — что НЕ делаем

### 10.1. В чеках и оферте ❌

- Слова «аудит», «консультация», «стратегия», «сопровождение» в чеке — штраф ФНС
- Услуга вне видов деятельности самозанятой — штраф ФНС + риск перехода на ИП
- Чек без указания вида деятельности — штраф

### 10.2. В рекламе и публичных текстах ❌

- Реклама без маркировки «реклама» (с 2022 года в РФ) — штраф
- Гарантия результата («заработаешь миллион») — реклама с заведомо ложным обещанием
- Использование чужих фото и видео без разрешения — авторские права

### 10.3. С персональными данными ❌

- Сбор email/телефонов без согласия на ПД (ФЗ-152) — штрафы Роскомнадзора
- Хранение данных клиентов на иностранных серверах для российских лиц — штраф
- Передача данных третьим лицам без согласия — штраф

Подробно — `legal-okved.md` + у Yuridika и PD-Lawyer.

## 11. Чек-лист «не выпускать если...»

Перед публикацией любой работы — пробежаться:

- [ ] Нет корпоративного «мы» там где должно быть «я»
- [ ] Нет агрессивных CTA («купите сейчас», «только сегодня»)
- [ ] Нет запрещённых юридических слов («аудит», «консультация», «стратегия»)
- [ ] Цвета только из палитры
- [ ] Шрифт по выбранной типографической паре
- [ ] Логотип не растянут, не повёрнут, в палитре
- [ ] Иллюстрации — авторские (не сток без обработки)
- [ ] PDF-документ имеет шапку, подвал, контакты, блок «закажи у нас»
- [ ] Лендинг имеет один primary-CTA, не 3-4 равных
- [ ] Голос — Иринин (личностный, не корпоративный)

**Если хотя бы 2 пункта не сходятся — переделываем.**

## 12. Как пополнять этот файл

> **Принцип роста:** этот файл — **накопительный**. Каждый раз, когда что-то пошло «не так» и Ирина это поймала — добавляем сюда новый «нельзя».

### Шаблон записи нового антипримера

```markdown
### [Что произошло]

**Контекст:** где встретили (пост / лендинг / PDF / комментарий клиента)
**Что было плохо:** конкретно что увидели
**Почему это «не Ирина»:** объяснение через бренд-принципы
**Замена:** как должно быть правильно
**Дата фиксации:** YYYY-MM-DD
```

### Где новые «нельзя» появляются чаще всего

1. После сверочной встречи Инны и Ирины — что Ирина «увидела» в работе агентов
2. После клиентского фидбека — «не понравилось вот это»
3. После аудита Skill-Auditor / QA-агента
4. После наблюдения за конкурентами — «вот так точно не делаем»

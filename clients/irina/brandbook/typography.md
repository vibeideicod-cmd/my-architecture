# Типографика бренда Ирины

> **Зона:** все текстовые элементы — заголовки, подзаголовки, основной текст, подписи, цитаты, кнопки.
> **Принцип:** дизайнер без типографики — не дизайнер. Типографика выражает голос бренда так же, как палитра выражает эмоцию.
> **Версия:** MVP · 2026-05-01

> ⚠️ **Финальный выбор шрифтов — за Ириной.** Этот документ предлагает три рабочих варианта пары + правила применения. Ирина читает, выбирает, помечает ✅. После выбора файл фиксируется как утверждённый.

## 1. Шрифтовая пара — три варианта на выбор

### Вариант A — Cormorant Garamond + Inter ⏳

**Заголовки:** Cormorant Garamond (серифный, ренессансный, дизайнерский)
**Основной текст:** Inter (гротеск, чистый, читаемый)

**Характер:** элегантность · дизайнер старой школы · «у меня насмотренность»
**Подходит:** премиум-лендинги, презентации экспертам, обложки брендбуков для клиентов

```
Cormorant Garamond Bold — заголовок крупный
Cormorant Garamond Medium — подзаголовок
Inter Regular — основной текст, читается на длинных лендингах
Inter Medium — кнопки, акценты
```

### Вариант B — Playfair Display + Manrope ⏳

**Заголовки:** Playfair Display (серифный, контрастный, эффектный)
**Основной текст:** Manrope (гротеск, мягкий, тёплый)

**Характер:** журнальная подача · авторский · «каждая идея — новый визуал»
**Подходит:** соцсети, личный бренд Ирины, лендинги бьюти и коучей

```
Playfair Display Bold — wow-заголовки на лендинге
Playfair Display Regular Italic — цитаты, эпиграфы
Manrope Regular — основной текст, вкладки кабинетов
Manrope Bold — кнопки, ярлыки
```

### Вариант C — Onest + Caveat (акцент) ⏳

**Заголовки:** Onest (современный гротеск, нейтральный, тех)
**Основной текст:** Onest (тот же шрифт в разных весах)
**Акцент:** Caveat (рукописный) — для цитат-сигнатур, подписей, личных пометок

**Характер:** современный дизайнер · цифровой · «технологично + человечно»
**Подходит:** платформы геймификации, мини-аппы, конструкторы, AI-помощник

```
Onest Bold — заголовок
Onest Regular — основной текст
Onest Medium — кнопки
Caveat Bold — «Слышу, какого человек цвета» (сигнатура везде, где уместно)
```

## 2. Правила выбора одного шрифта на проект

После того как Ирина выберет основную пару — **остальные два варианта остаются как опциональные** под нишевые задачи:

| Контекст | Какая пара уместна |
|---|---|
| Премиум-клиент (финансы, премиум-эксперт, санаторий) | Вариант A — серифная классика |
| Личный бренд Ирины + бьюти + коучи | Вариант B — журнальная подача |
| Tech-продукты (платформы, мини-аппы, AI-помощник) | Вариант C — современный гротеск |

**Не миксовать варианты внутри одного носителя.** Например, на лендинге фитнес-клуба = только один из вариантов, не «заголовок Playfair, кнопки Onest».

## 3. Иерархия размеров

> **Принцип:** масштаб должен **читаться сразу**. Заголовок крупнее основного текста минимум в 2 раза.

### Веб (px) — для лендингов и мини-аппов

| Уровень | Размер | Вес | Межстрочный | Использование |
|---|---|---|---|---|
| **H1** (главный заголовок) | 56-72px | Bold (700) | 1.1 | Один на странице — главный смысл |
| **H2** (раздел) | 40-48px | Bold (700) | 1.15 | Заголовки крупных блоков |
| **H3** (подраздел) | 28-32px | Bold (700) | 1.2 | Подразделы внутри блока |
| **H4** (мини-заголовок) | 22-24px | Bold (700) | 1.25 | Карточки, мини-блоки |
| **Body** (основной) | 16-18px | Regular (400) | 1.5-1.6 | Длинный текст |
| **Body Small** | 14px | Regular (400) | 1.5 | Подписи, метаданные |
| **Caption** | 12-13px | Regular (400) | 1.4 | Сноски, копирайты |
| **Button** | 16-18px | Medium (500) или Bold (700) | 1 | Кнопки CTA |

### Печать (pt) — для PDF-продуктов и презентаций

| Уровень | Размер | Использование |
|---|---|---|
| **H1** | 28-36pt | Обложка, главный заголовок |
| **H2** | 18-22pt | Раздел в PDF |
| **H3** | 14-16pt | Подраздел |
| **Body** | 10.5-11pt | Основной текст в гайде |
| **Caption** | 8-9pt | Сноски, копирайты |

> 📄 PDF-продукты Ирины (визитки, чек-листы, гайды) — все через `_md2html.py` (см. `clients/irina/deliverables/`). Размеры в шаблоне CSS уже учтены.

## 4. Веса и начертания

**Используем максимум 3 веса** на одном носителе — иначе типографика «шумит».

| Вес | Когда |
|---|---|
| **Regular (400)** | Основа — длинный текст, описания, body |
| **Medium (500)** | Промежуточные акценты, навигация, ярлыки |
| **Bold (700)** | Заголовки, кнопки, главные акценты |

**Italic** — только для цитат, эпиграфов, выделения иностранных слов. Не использовать как замену Bold.

## 5. Кириллица + латиница

> **Особенность:** Ирина пишет по-русски, но иногда вставляет английские термины («Brandbook», «AI», «vibe coding»). Шрифт должен поддерживать оба алфавита **без переключения**.

**Все три варианта (Cormorant, Playfair, Onest, Manrope, Inter, Caveat) поддерживают кириллицу + латиницу с одинаковой стилистикой.** Это критично — иначе на странице получится «винегрет» из двух разных характеров.

**Правило:** если шрифт не поддерживает кириллицу — НЕ используем, ищем альтернативу. Не делаем «русская версия одним шрифтом, английская другим».

---

## 6. Кернинг и leading

**Кернинг** (расстояние между буквами):
- Заголовки H1-H2 → tight (-0.02em до -0.01em) — крупный шрифт «дышит» сам
- Body → normal (0)
- Caps (заглавные) → wide (+0.05em до +0.1em) — иначе «сливается»

**Leading** (межстрочный):
- Заголовки → 1.1-1.25 (плотно)
- Body → 1.5-1.6 (комфортно для чтения длинного текста)
- Caption → 1.4

## 7. Типографические приёмы

### Что использовать ✅

- **Висячая пунктуация** на печати (русские «ёлочки» « » за пределы блока)
- **Длинное тире —** вместо дефиса в авторских мыслях («дизайн — это не логотип побольше»)
- **Неразрывные пробелы** между числом и единицей («50 к», «3 дня»)
- **Цитаты-сигнатуры в Italic или Caveat** для эмоционального выделения
- **Капитализация по правилам русского языка** — не «Все Слова С Большой» (это англицизм)

### Что НЕ использовать ❌

- **CAPS LOCK на длинных фразах** — кричит, не Иринин голос
- **Подчёркивание** для выделения — это для ссылок, не для акцента
- **Курсив + Bold одновременно** — перебор
- **Разные начертания в одном предложении** — «давайте поговорим **сегодня** *в 5 вечера*» = плохо
- **Растягивание текста** (letter-spacing > 0.1em на длинных фразах) — нечитаемо

## 8. Применение по носителям

| Носитель | Заголовок | Основной | Особенности |
|---|---|---|---|
| **Лендинг (десктоп)** | 56-72px H1 | 18px body | Цифры жирнее обычного — для CTA «от 5к» |
| **Лендинг (мобильный)** | 36-44px H1 | 16px body | Заголовки переносятся в 2-3 строки |
| **TMA / Telegram Mini App** | 24-28px H1 | 15-16px | Ограничено WebView, экономим высоту |
| **VK Mini App** | 22-26px H1 | 14-15px | Похоже на TMA |
| **PDF-продукты (визитки, гайды)** | 22-28pt | 10.5-11pt | По шаблону `_md2html.py` |
| **Презентации (slide)** | 36-48pt | 18-22pt | Большие шрифты — слайд читается из зала |
| **Соцсети (пост-картинка)** | 32-40px (на 1080px холст) | 18-22px | Текст занимает максимум 30% площади |
| **Сторис VK / Telegram** | 48-72px (на 1080×1920) | 24-32px | Один заголовок + 1-2 строки тезиса |
| **Email-подпись** | 16px | 14px | Минимальный набор: имя, роль, контакты |
| **Бот / AI-помощник** | — | Системный шрифт мессенджера | Шрифты контролировать нельзя — управляем эмодзи и форматированием |

## 10. Антипримеры

### ❌ Что выглядит «не Ирина»

- **Слишком много весов** на одном лендинге — Light + Regular + Medium + SemiBold + Bold + ExtraBold = шум
- **Маленький H1** — если главный заголовок 24px, страница «не дышит»
- **Body 12-14px на десктопе** — устаёт глаз, бренд-впечатление портится
- **Жёсткий Verdana / Arial / Times New Roman** — это шрифты «по умолчанию», не дизайнерские
- **Comic Sans, Papyrus** — мгновенный сигнал «непрофессионально»
- **Резкие цветные шрифты** (бирюзовый текст, фиолетовый текст) на ярких фонах — нечитаемо
- **Заголовок занимает 5 строк** — нужно сократить смысл, а не уменьшать шрифт

### ✅ Что выглядит «Ирина»

- **2 шрифта максимум** на проекте (заголовочный + основной), 3-й только как опциональный акцент
- **Крупный H1 56-72px** на десктопе — «есть что сказать, и сказано громко»
- **Body 16-18px** с межстрочным 1.5-1.6 — чтение комфортное
- **Кириллица + латиница в одном шрифте** — без переключений
- **Цитаты-сигнатуры в Italic / Caveat** — личный голос
- **Висячая пунктуация на печати** — мелочь, которую видят дизайнеры

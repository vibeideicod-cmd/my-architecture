# Портфолио Ирины Цепаевой

Источник работ для брендбука v1.1, кейсов, цифровой визитки, лендингов.

## Структура

| Папка | Что внутри | Сколько |
|---|---|---|
| `works/` | Готовые работы Ирины (`1.jpg`–`12.jpg`) — для галереи в брендбуке v1.1 и кейс-блоков | 12 jpg, ~6 МБ |
| `references/` | Визуальные референсы Ирины (`1.jpg`–`7.jpg`) — что её вдохновляет, помогает калибровать архетип в её проектах | 7 jpg, ~1.5 МБ |

## Как использовать

- **Брендбук v1.1 — галерея 10 работ** (задача из OPEN.md): отбираем 10 лучших из `works/`, в брендбук как раздел «Примеры работ».
- **Цифровая визитка/портфолио-страница**: блок `#cases` или `#works` после hero — асимметричная сетка из 4-6 работ с якорными URL `#case-001`, `#case-002`.
- **Кейсы для клиентов**: каждая работа = потенциальный кейс. Нужно подписать «что за задача / кто клиент / что было до».
- **Калибровка архетипа** в новых проектах Ирины: смотрим `references/` чтобы понять её визуальный язык до нового брифа.

## Открытое — что докинуть

- **Подписи к работам**: для каждой `1.jpg`–`12.jpg` нужны 1-2 строки от Ирины: «Что за проект, для кого, какая была задача».
- **Сортировка**: какие работы Ирина считает топ-3 / топ-5 / топ-10 для брендбука. Решение за Ириной.

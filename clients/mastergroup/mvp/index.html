Нейро Бабки СССР · Платформа мастер-групп

Твоя мастер-группа живёт вне Telegram

Браузерная платформа для ведения мастер-групп. Витрина для привлечения, кабинет для работы с группой, единая база участников. Работает везде, где есть интернет. Не зависит от блокировок мессенджеров.

Три интерфейса — одна система

🎯
Витрина мастера
Красивая публичная страница с программой. Участник переходит по ссылке, читает, записывается в один клик.
🎓
Кабинет мастера
Список участников в реальном времени, материалы, задания, чат с группой. Всё в одном месте, без Telegram.
📊
Админ-панель
Все мастера, все участники, вся статистика. Экспорт в CSV. Общая база для дальнейшей работы.

Что дальше

Этого MVP достаточно чтобы показать продукт живому мастеру и собрать обратную связь. После валидации добавим: загрузку PDF/видео в Storage, отметки выполнения заданий, форму сбора кейсов от участников, Telegram Mini App, платёжку.

Бэкенд — Supabase, фронт — чистый HTML без фреймворков. Деплой — Beget.

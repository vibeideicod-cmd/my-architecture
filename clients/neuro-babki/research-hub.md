# Research: Лендинг HUB — Нейро Бабки СССР

Глубокое исследование для разработки продающего лендинга маркетинговых услуг.
Запущено 6 агентов-исследователей параллельно. Результаты скомпилированы в единый документ.

## Часть 1. Конкуренты и примеры

*Источник: анализ маркетинговых агентств с Awwwards, Dribbble, Behance*

### 9 лучших примеров для изучения

#### 1. Huge Inc. (hugeinc.com)
**Первый экран:** полноэкранный манифест — одна фраза позиции. Ничего лишнего.
**Секции:** Hero-манифест → Кейсы (крупные карточки) → Философия → Услуги строками → Контакт
**Стиль:** Монохром, чёрный фон, белый текст. Один акцент в hover.
**Что взять:** Манифест-заголовок на первом экране. Для НБ СССР: *"Мы не ведём соцсети — мы строим системы продаж."*

#### 2. Pentagram (pentagram.com)
**Первый экран:** Сетка работ — без предисловий. Портфолио сразу.
**Секции:** Кейсы → Фильтрация → Партнёры
**Стиль:** Белый фон, всё внимание на работы.
**Что взять:** Позиционировать Инну и Ирину как партнёров-владельцев, а не сотрудников — это строит доверие.

#### 3. Instrument (instrument.com)
**Первый экран:** Крупный заголовок + анимированный видеоряд.
**Секции:** Hero → Кейсы → Capabilities → Команда
**Стиль:** Тёмный/нейтральный фон, яркие акценты в работах.
**Что взять:** Секция "Как мы работаем" — не список услуг, а шаги методологии. Идеально для системы СССР.

#### 4. Wolff Olins (wolffolins.com)
**Первый экран:** Провокационный заголовок, ломающий ожидание.
**Секции:** Манифест → Кейсы с "Impact" → Принципы → CTA "Let's talk"
**Стиль:** Смелые цветовые блоки, засечковые шрифты.
**Что взять:** Акцент на результат ("Impact"), а не процесс. Цифры и истории — не перечисление услуг.

#### 5. Ueno (ueno.co)
**Первый экран:** Личность агентства на первом месте — "кто мы", а не "что делаем".
**Секции:** Hero → Кейсы → Культура → Контакт
**Стиль:** Чистый, тёплые акценты. Живые фото людей.
**Что взять:** Живые фото Инны и Ирины с характером — конкурентное преимущество перед безликими агентствами.

#### 6. Ragged Edge (raggededge.com)
**Первый экран:** Провокационный текст, цепляющий "уставших от обычных агентств".
**Секции:** Манифест → Нестандартные кейсы → "Почему мы" → Честные отзывы
**Стиль:** Чёрный + белый + один яркий акцент. Засечковые шрифты.
**Что взять:** Раздел "Почему именно мы" — конкретный ответ, чем отличаются от других.

#### 7. Work & Co (work.co)
**Первый экран:** Максимальный минимализм — название, одна строка, список проектов.
**Стиль:** Белый фон, геометрическая типографика. Ничего лишнего.
**Что взять:** Принцип "сдержанность = уверенность". Аудитория 55+ хорошо читает спокойствие.

#### 8. DesignStudio (design.studio)
**Секции:** Брендированный Hero → Кейсы с видео → Манифест → Список клиентов
**Стиль:** Тёмный фон, кинематографичность.
**Что взять:** Список клиентских объектов по названиям — реальные названия работают лучше, чем абстрактные отзывы.

#### 9. Moving Brands (movingbrands.com)
**Первый экран:** Анимированный кинематографический вход без торопливости.
**Что взять:** Описывать результат как трансформацию: не "мы ведём соцсети", а "ваш бизнес начинает говорить с клиентами иначе".

### Ключевые выводы по конкурентам

**Дизайн-принципы для НБ СССР:**
- Горчичный `#c99700` — точечно: подчёркивание ключевого слова, иконки этапов, hover. Не заливка блоков.
- Playfair Display на заголовках — элегантность + зрелость + доверие
- Живые фото команды (не стоки) — критически важно для этой аудитории
- Тёмный фон чередовать: основной тёмный, кейсы чуть светлее (`#1a1a1a`)
- Щедрые отступы, воздух между блоками = признак уверенности

**Чего избегать:**
- Анимаций ради анимаций — замедляют и раздражают на мобильном
- Сленга (таргет, лид-магнит, воронка) без объяснения
- Бесконечных списков услуг — лучше три ёмких направления
- Форм с десятью полями — одно поле + кнопка
- Значков несуществующих наград

## Часть 2. Идеальная структура лендинга (10 блоков)

*Источник: Unbounce, ConversionXL, Tilda Publishing, One Page Love*

### Логика построения

Лендинг работает как воронка осознанности. Клиент НБ СССР (владелец санатория, магазина, сервиса) скептичен, имел негативный опыт с подрядчиками, не понимает "цифровой" язык. Лендинг должен снять страх → объяснить просто → показать результат → дать понятный первый шаг.

### Блок 1. HERO — Первый экран
**Цель:** За 3–5 секунд ответить "я попал туда, куда надо?"

Что внутри:
- Заголовок: конкретный результат, не название компании
- Подзаголовок: для кого и что конкретно
- Главный CTA: одна кнопка, глагол + выгода
- Визуал: живые фото команды (не стоки)
- Опционально: 2–3 цифры под CTA ("7 лет · 40+ проектов · 5 городов")

Ошибки: заголовок = название бренда / два CTA / стоковая фотография

### Блок 2. ПРОБЛЕМА — Попадание в боль
**Цель:** Показать, что вы понимаете ситуацию клиента лучше, чем он сам.

Что внутри:
- Заголовок-вопрос: "Узнаёте себя?"
- 4–6 конкретных болей языком клиента:
  - "Соцсети есть, ведёте от случая к случаю — когда есть время"
  - "Были подрядчики по SMM, деньги взяли, результат непонятен"
  - "Сарафан работает, но нестабильно — хочется системы"
  - "Не хватает времени разбираться во всех этих Инстаграмах и ботах"
- Переходная фраза: "Мы не обещаем магии. Мы строим систему — шаг за шагом."

Ошибки: язык маркетолога вместо языка клиента / пропустить блок совсем

### Блок 3. РЕШЕНИЕ — Что предлагаете
**Цель:** Объяснить продукт просто, показать логику.

Что внутри (две карточки):
- **MINI-модули** — начать с одного направления (лиды / коммуникация / контент)
- **SYSTEM** — маркетинг под ключ

Формат: заголовок + 2–3 строки + "для кого подходит"

Ошибки: список услуг без ценности / профессиональный жаргон / нет разницы между продуктами

### Блок 4. КАК РАБОТАЕМ — Процесс
**Цель:** Снять страх "непонятно, что происходит после оплаты".

3–5 шагов с номером и коротким пояснением:
1. Диагностика — разбираемся в вашем бизнесе
2. Стратегия — составляем план
3. Запуск — начинаем с первого модуля
4. Аналитика — смотрим что работает
5. Рост — масштабируем результат

Важно: первый шаг = бесплатная диагностика, не покупка.

Ошибки: абстрактные названия шагов / 7+ шагов / не указано что первый шаг бесплатный

### Блок 5. ДЛЯ КОГО — Портрет клиента
**Цель:** Помочь посетителю сказать "это точно про меня".

Три карточки:
- Санаторий / оздоровительный комплекс
- Магазин / розница
- Сервисный бизнес

Плюс: "Вам подойдёт, если..." и честно "Вам не подойдёт, если..." — честность строит доверие.

### Блок 6. РЕЗУЛЬТАТЫ / КЕЙСЫ — Доказательства
**Цель:** Социальное доказательство. Кейс с похожей компанией сильнее любого аргумента.

Формат мини-кейса:
```
Кто: Санаторий "Берёзки", Тверь
До: не было онлайн-присутствия, все заявки по телефону
Что сделали: лендинг + Telegram-бот + контент
Результат: +40 заявок в первый месяц, 30% онлайн
Срок: 3 месяца
```

Если кейсов нет — отзывы с конкретикой: кто клиент, что изменилось, цифра.

Ошибки: "Всё отлично!" без деталей / кейсы без цифр / только восторг без фактов

### Блок 7. СТОИМОСТЬ — Прозрачность
**Цель:** Снять тревогу "сколько это стоит". Страница без цен теряет до 30% конверсии.

Два направления: MINI (от X руб./мес.) и SYSTEM (от Y руб./мес.)
CTA: "Узнать стоимость для своего бизнеса"

Если цена зависит от масштаба — честно писать "от... — точная стоимость после диагностики".

Ошибки: скрывать цены / сложная таблица / кнопка "Купить" (агрессивно)

### Блок 8. КОМАНДА — Живые люди
**Цель:** Клиент платит не компании, он платит людям.

Две карточки с живыми фото: Инна + Ирина.
Под карточками: "Вместе мы закрываем весь цикл: от стратегии до визуала."
1–2 личных факта — показывают человека за профессией.

Ошибки: стоковые фото / список регалий без контекста / официальный тон биографии

### Блок 9. FAQ — Ответы на возражения
**Цель:** Снять оставшиеся сомнения до нажатия кнопки.

5–7 вопросов:
1. Как быстро будет результат? — честно: 1–2 мес. первые данные, 3–6 мес. устойчивый результат
2. Мы небольшой бизнес — нам подойдёт? — именно для вас
3. Уже пробовали SMM — не сработало — расскажите на диагностике, чаще дело в стратегии
4. Можно только один модуль? — да, для этого есть MINI
5. Как отчитываетесь? — ежемесячный отчёт с цифрами
6. Можно остановить? — да, помесячно без штрафов
7. Работаете дистанционно? — да, со всеми клиентами

Ошибки: вопросы-саморекламы / уклончивые ответы на неудобное / меньше 5 вопросов

### Блок 10. ФИНАЛЬНЫЙ CTA — Первый шаг
**Цель:** Сделать последний шаг максимально простым и безопасным.

Что внутри:
- Заголовок: "Начнём с диагностики — бесплатно и без обязательств"
- 1–2 строки: "30 минут — и вы поймёте, что мешает расти и что с этим делать"
- Форма: имя + телефон (не больше)
- Кнопка: "Записаться на диагностику" (не "Отправить")
- Альтернатива: "Или напишите в Telegram" — ссылка
- Микротекст: "Не перезваниваем 100 раз. Напишем один раз — договоримся о времени."

Ошибки: CTA "Купить" / форма с 6+ полями / нет альтернативного канала

### Рекомендуемый порядок блоков

```
1. Hero              — захватить внимание
2. Проблема          — создать резонанс
3. Решение           — объяснить продукт
4. Как работаем      — снять страх неизвестности
5. Для кого          — персонализировать
6. Кейсы             — доказать словами других
7. Стоимость         — снять тревогу о цене
8. Команда           — добавить доверие
9. FAQ               — убрать последние сомнения
10. Финальный CTA    — сделать первый шаг лёгким
```

### 1. Кинетическая типографика
Каждое слово заголовка появляется с анимацией. Без JS.

```css
@keyframes reveal-word {
  from { opacity: 0; transform: translateY(40px) skewY(4deg); filter: blur(4px); }
  to   { opacity: 1; transform: translateY(0) skewY(0); filter: blur(0); }
}
.hero-title span {
  display: inline-block;
  animation: reveal-word 0.7s cubic-bezier(0.22, 1, 0.36, 1) forwards;
  opacity: 0;
}
.hero-title span:nth-child(1) { animation-delay: 0.1s; }
.hero-title span:nth-child(2) { animation-delay: 0.25s; }
.hero-title span:nth-child(3) { animation-delay: 0.4s; }
```

### 2. Bento Grid Layout
Карточки разного размера в плотной асимметричной сетке. Стиль Apple.

```css
.bento-grid {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 16px;
}
.card-wide { grid-column: span 2; }
.card-tall { grid-row: span 2; }
@media (max-width: 768px) {
  .bento-grid { grid-template-columns: 1fr; }
  .card-wide, .card-tall { grid-column: span 1; grid-row: span 1; }
}
```

### 3. Fluid Typography
Размер шрифта плавно масштабируется без медиазапросов.

```css
:root {
  --text-hero: clamp(3rem, 10vw, 7rem);
  --text-h1:   clamp(2rem, 5vw, 4rem);
  --text-h2:   clamp(1.5rem, 3vw, 2.5rem);
  --text-body: clamp(1rem, 2vw, 1.125rem);
  --text-sm:   clamp(0.875rem, 1.5vw, 1rem);
  --text-xs:   clamp(0.7rem, 1.2vw, 0.8rem);
}
```

### 4. Micro-анимации состояний
Каждая кнопка "дышит" — hover, active, focus.

```css
.btn-primary {
  transition: transform 0.2s cubic-bezier(0.34, 1.56, 0.64, 1), box-shadow 0.2s ease;
}
.btn-primary:hover  { transform: translateY(-2px) scale(1.02); box-shadow: 0 8px 32px rgba(201,151,0,0.35); }
.btn-primary:active { transform: translateY(0) scale(0.98); }
```

### 5. Grain / Noise текстура
Зернистость убирает "пластмассовость" тёмного фона.

```css
body::after {
  content: '';
  position: fixed;
  inset: 0;
  pointer-events: none;
  z-index: 9999;
  opacity: 0.04;

  background-size: 256px 256px;
}
```

### 6. Горизонтальный марквиз
Бегущая строка с именами клиентов или ключевыми словами.

```css

  to   { transform: translateX(-50%); }

```

### 7. Glassmorphism 2.0 (утончённый)

### 8. Outline Ghost-текст
Контурный текст рядом с заливным — глубина без изображений.

```css

```

### 9. Модульная сетка на фоне
Тонкие линии разграфки создают структуру и глубину.

## Часть 4. Копирайтинг и CTA

*Источник: Copyhackers (Joanna Wiebe), ConversionXL, Andy Maslen, David Ogilvy*

### Заголовки Hero — 4 рабочих формулы

**1. Зеркало болей** — говоришь словами клиента:
> "Клиентов стало меньше, а реклама съедает бюджет?"

**2. Неожиданный разворот** — боль → выход:
> "Устал объяснять таргетологам, что такое твой бизнес — и снова переделывать?"

**3. Конкретное обещание:**
> "Выстраиваем поток клиентов для оффлайн-бизнеса за 60 дней"

**4. Провокация через иронию** (тон бренда):
> "Маркетинг без воды, дашбордов и умных слов — зато с результатом"

### 5 вариантов заголовка для НБ СССР

> **A:** "Реклама есть, клиентов нет. Звучит знакомо?"

> **B:** "Делаем так, чтобы твой магазин / кафе / санаторий был забит — без TikTok-плясок и умных слов"

> **C:** "Маркетинг для оффлайн-бизнеса: от «нас не знают» до «запись на месяц вперёд»"

> **D:** "Ты открыл бизнес, чтобы работать — а не сидеть в рекламных кабинетах. Мы берём это на себя."

> **E:** "Выстраиваем маркетинг для оффлайн-бизнеса. Системно. Без воды."

### Формула PAS (Problem → Agitate → Solution)

Пример для кафе:
> Реклама в соцсетях стоит денег, но столики всё равно пустуют в будни. *(P)*
> Ты уже пробовал таргет, листовки, акции — каждый раз всплеск на неделю, потом тишина. *(A)*
> Мы выстраиваем систему, которая приводит людей регулярно — не разово. *(S)*

### Формула Before → After → Bridge

Пример для мастера-одиночки:
> **До:** Клиенты приходят по сарафанному радио — нестабильно, непредсказуемо.
> **После:** Запись на 3 недели вперёд, люди находят тебя сами.
> **Мост:** Мы настраиваем присутствие в интернете так, чтобы нужные люди приходили к тебе — а не ты к ним.

## Часть 5. Визуальный дизайн тёмной темы

*Источник: Dribbble, Behance, Muzli, design.google*

### Цветовая схема рядом с горчичным #c99700

**Рекомендуемая: Тёплая монохромная**
```css
:root {
  --bg:            #0d0b08;   /* тёплый чёрный, не технический */
  --bg-card:       #1a1610;   /* тёплый коричневатый */
  --bg-elevated:   #221e14;
  --text-primary:  #f0e6cc;   /* кремовый, не чистый белый */
  --text-secondary:#c8b896;
  --text-body:     #9e8e76;
  --text-muted:    #6e5e4a;
  --accent:        #c99700;
  --accent-hover:  #e6b800;
  --border:        rgba(201, 151, 0, 0.15);
  --border-hover:  rgba(201, 151, 0, 0.4);
}
```

Чего избегать рядом с горчичным: чистый белый (убивает теплоту), синий и фиолетовый (холодный диссонанс), чистый чёрный #000000 (слишком резко).

### Градиенты на тёмном фоне

Принцип 2025: градиент = источник света, не украшение.

**Радиальный glow (самый популярный):**
```css
.hero {
  background:
    radial-gradient(ellipse 80% 60% at 20% 50%, rgba(201,151,0,0.12) 0%, transparent 70%),
    #0d0b08;
}
```

**Gradient text:**
```css
.gradient-text {
  background: linear-gradient(135deg, #e6b800 0%, #c99700 40%, #b87333 100%);
  -webkit-background-clip: text;
  -webkit-text-fill-color: transparent;
  background-clip: text;
}
```

**Gradient border (тренд 2025):**
```css
.card-gradient-border::before {
  content: '';
  position: absolute;
  inset: 0;
  border-radius: 16px;
  padding: 1px;
  background: linear-gradient(135deg, rgba(201,151,0,0.6), rgba(201,151,0,0.1), transparent);
  -webkit-mask: linear-gradient(#fff 0 0) content-box, linear-gradient(#fff 0 0);
  -webkit-mask-composite: xor;
  mask-composite: exclude;
}
```

### Карточки

```css
.card {
  background: rgba(255, 255, 255, 0.03);
  border: 1px solid rgba(201, 151, 0, 0.15);
  border-radius: 16px;
```

### Иерархия яркости на тёмном фоне

На тёмном — элементы "всплывают" светом, не тенью. Важнее = светлее.

```css
:root {
  --text-h1:       #f0e6cc;  /* главный заголовок */
  --text-h2:       #c8b896;  /* подзаголовки */
  --text-body:     #9e8e76;  /* основной текст */
  --text-muted:    #6e5e4a;  /* вторичный */
  --text-ghost:    #3e302a;  /* декоративный */
}
```

### Визуальный язык "уверенность зрелого человека"

Ощущение: пришёл к опытному специалисту. Здесь не пытаются произвести впечатление — здесь работают.

- **Неидеальная симметрия:** `grid-template-columns: 1.2fr 0.8fr` вместо 50/50
- **Тёплые оттенки вместо нейтральных:** `#1a1610` вместо `#1a1a1a`
- **Щедрое пространство:** `min-height: 90vh`, `max-width: 14ch` для заголовка
- **Serif в заголовках:** Playfair Display — зрелость, надёжность, история
- **Зерно:** добавляет тактильность (opacity: 0.03–0.05)

Итоговая таблица рекомендаций:

| Параметр | Рекомендация |

| Карточки | `rgba(255,255,255,0.03)` + горчичная рамка |

## Часть 6. UX и конверсия — мобильный приоритет

*Источник: Nielsen Norman Group, Baymard Institute, Luke Wroblewski, Google Mobile UX*

### Что обязательно на мобильном

- Минимальный шрифт: 16px основной текст, 22px заголовки
- Зоны касания: минимум 48×48px, для аудитории 40+ — 56–64px
- Одна колонка, вертикальная прокрутка
- Первый экран: "Я попал куда надо?" за 3–5 секунд

### Что убивает конверсию

- Всплывающие окна в первые 5 секунд
- Мелкий шрифт + серый на сером (для людей 40+ с пресбиопией — физическая боль)
- Горизонтальный скролл (воспринимается как сломанная страница)
- Более одного CTA на экране (закон Хика — паралич выбора)
- Скрытые контакты в футере (аудитория 40+ предпочитает позвонить)

### Навигация

**Рекомендация для НБ СССР:**
- Sticky header 48px: логотип + кнопка "Написать в Telegram"
- Fixed bottom bar на мобиле: кнопка звонка
- Закрывает обе модели поведения: мессенджер и телефон

### Кнопки CTA

- Размер: 56–64px высота, полная ширина (или 90%)
- Повторение: минимум 3 раза — Hero → середина → финал
- Позиция: нижняя треть экрана — thumb zone по Wroblewski

### Форма

**Ультраминимум:**
- Имя
- Телефон (`type="tel"` → цифровая клавиатура)
- Кнопка "Записаться на диагностику"

**Детали для аудитории 40+:**
- Метки `<label>` над полями (не placeholder — исчезает при вводе)
- `autocomplete="name"` и `autocomplete="tel"` — браузер подставит сам
- После отправки: явное подтверждение с именем ("Спасибо, [Имя]!")
- `font-size: 16px` в полях — блокирует автозум в Safari

**Альтернатива форме:** прямая ссылка `<a href="https://t.me/username">` — нет формы, нет барьера.

### Скорость загрузки

- Изображения: WebP + `loading="lazy"` + указывать `width` и `height`
- Шрифты: `preconnect` + `display=swap` + только нужные начертания (`wght@400;600;700`)
- Анимации только через `transform` и `opacity` — не вызывают перерасчёт layout
- Цель: LCP за 2.5 секунды на 4G

### Удержание внимания

**Структура убеждения:**
```
Hero → Боль → Решение → Доказательства → CTA
```
Средняя глубина прокрутки на мобиле — 50–60%. Социальное доказательство ставить в середину, не только в конец.

**Паттерн F-сканирования:** начало каждой строки читается, конец — нет. Важные слова — влево. Цифры и жирный текст притягивают взгляд.

**Аудитория 40+:**
- Читают медленнее, но внимательнее
- Больше доверяют тексту, чем иконкам
- Ищут контакты (телефон, адрес) в первую очередь
- Телефон должен быть в шапке или первом экране, не только в футере

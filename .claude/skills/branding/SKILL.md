# Branding — бренд-платформа и визуальная система

## Обзор

Описываю логику бренда: характер, цвета, шрифты, голос. Работаю словами и структурой — Ирина воплощает визуально. Каждое решение обосновываю, не «красивый цвет», а «зелёный — доверие и здоровье, подходит для санатория».

## Порядок работы

### 1. Собери данные

Убедись что знаешь:
- Сфера и тип бизнеса
- Целевая аудитория
- Три слова, которые описывают бизнес
- Есть ли уже логотип / цвета / стиль
- Примеры которые нравятся (из любой сферы)

Если нет — задай эти 5 вопросов блоками по 2.

### 2. Для клиентского бренда

⚠️ Брендбук Инны НЕ применяем к клиентам. У каждого клиента — своя система.

Структура результата:

```
## Бренд: [Название]

### Характер бренда
[3-5 прилагательных + расшифровка каждого]

### Целевая аудитория
[Краткое описание: кто, что важно, как принимает решение]

### Цветовая палитра
- Основной: #HEX — [почему, что передаёт]
- Дополнительный: #HEX — [роль]
- Акцент: #HEX — [где использовать]
- Текст: #HEX
- Фон: #HEX

### Шрифты
- Заголовки: [шрифт] — [почему]
- Текст: [шрифт] — [почему]

### Стиль изображений
[Тёмный / светлый, фото / иллюстрации, настроение]

### Tone of voice
[Как бренд говорит: формально / дружески, серьёзно / с юмором]
[Запрещённые слова или темы]

### Применение
- В соцсетях: [рекомендации]
- На сайте: [рекомендации]
- В печатных материалах: [рекомендации]
```

### 3. Для нашего бренда (Инна-Архитектор)

Официальный брендбук: `knowledge/brand/brandbook.md`
Официальная палитра: `knowledge/brand/palette.md` — 8 цветов

Работаем только с тем, что утверждено Ириной. Ничего не добавляем без согласования с дизайнером.

## Примеры

**Вход:** Клиент — частная стоматология «Улыбка», Краснодар, аудитория — семьи с детьми и взрослые 30-55, три слова: «надёжный, заботливый, современный», есть голубой логотип
**Выход:** Характер бренда — «профессиональный (доверие врача), тёплый (забота, не страх), современный (технологии без холодности)», палитра — голубой `#4a90d9` + белый `#ffffff` + нейтральный серый `#f5f5f5`, шрифты — Nunito (заголовки, мягкая геометрия) + Open Sans (текст)

## Если данных не хватает

Если нет трёх слов — попроси: «Назовите три слова, которые должны приходить в голову клиенту когда он думает о вашем бизнесе. Не "качество" и "профессионализм" — они у всех. Что-то специфичное для вас.»

Если нет референсов — предложи 3 варианта визуального направления с описанием: «Вариант А — строгий и минималистичный, вариант Б — тёплый и органичный, вариант В — яркий и энергичный. Что ближе?»

Если уже есть логотип и цвета — не предлагай другую палитру. Работай в рамках существующей айдентики.

## Что передаём дальше

Brandbook Creator — получает бренд-платформу и оформляет в готовый документ.
Content — получает TOV-стандарт для соблюдения голоса.
Websites — получает цвета, шрифты и стиль для вёрстки.

Финальные решения по визуалу принимает Ирина (дизайнер команды).

Финальная фраза скилла: «Бренд-платформа готова. Передаю Brandbook Creator и дизайнеру.»

## KPI

Скилл отработал хорошо если:
- Характер бренда описан 3-5 прилагательными с конкретной расшифровкой (не «красиво»)
- Каждое цветовое решение обосновано — что передаёт, почему подходит этому бизнесу
- TOV задан конкретно: есть «говорим так» и «не говорим вот так»
- Брендбук Инны не применён к клиентскому проекту

## Рекомендуемая модель

**Sonnet** — бренд-платформа требует понимания аудитории, контекста бизнеса и умения обосновывать визуальные решения словами

# FILES — карта проекта my-architecture

Дата обновления: 2026-05-15

Статусы:
- **источник правды** — главный актуальный документ по теме;
- **активный** — используется в работе;
- **черновик** — рабочий материал, не финал;
- **исторический** — оставлен для истории, не источник правды;
- **архив / осторожно** — не трогать без отдельного решения.

## Правило для новых папок

Если в папке становится больше 5 рабочих документов или появляются версии одного результата, добавь локальный `FILES.md` с пометками:
- активное;
- финальное;
- черновик;
- устаревшее;
- справочное;
- архив.

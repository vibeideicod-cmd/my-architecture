## Цепочка как пришли к продукту

1. **Telegram + сервер vs локальная сборка.** Инна спросила — почему её архитектура локально, а не на сервере как у Димы? Объяснила: сервер vs ноутбук — это просто два разных компьютера, разница только в интерфейсе (Telegram vs VS Code). У неё «автомат сохранения» **уже работает** через правила в [CLAUDE.md](../../CLAUDE.md) и хук на коммит. Перенос на сервер нужен **только** ради Telegram. Без Telegram — бессмысленно.

2. **Идея — Telegram для my-architecture тоже бы пригодился.** Возникла мысль протестировать.

3. **Поворот — а что если такого же ассистента для Ирины?** Ирина = «расширенный пользователь», заходит в свой бот, делает там свою работу.

4. **Большой поворот — это новый продукт.** Инна сформулировала: «если потом по аналогии отдаём этого бота клиенту — на подписку с ограничениями». **Это коммерческий продукт, не личный инструмент.**

5. **Точное наблюдение:** **Дима строит ровно эту бизнес-модель.** Каждый клиент = свой VPS + бот + workspace + подписка. У него уже спроектированный коммерческий шаблон. Не изобретаем — **переиспользуем**.

### Технический стек
- **Дима 1-в-1.** Скиллы не трогаем, бренд не меняем, сценарии копируем как есть. Меняем только данные про Ирину в DNA-файлах (SOUL/USER/MEMORY/MISSION/GOALS/PROJECTS/PREFERENCES)
- Сервер — **отдельный Beget Cloud VPS Ирины** (Single-tenant как у Димы)
- LLM-провайдер — **ProxyAPI** (российский прокси к Claude, чек, оплата с российской карты)
- Биллинг — **ЮKassa** с тарифом «Для самозанятых» (авто-выписка чеков в «Мой налог»)

### Продуктовая модель
- **Дорога А — универсальный ассистент** (Дима 1-в-1). Дорога Б (контент-конструктор) отложена как «следующий продукт #2», когда отладим инфраструктуру
- **Тарифы — гипотеза:**
  - Демо (50 сообщений/7 дней) — 0₽
  - Старт (500/мес) — ~1500₽/мес + овердрафт +1000₽ за 100 сверху
  - Стандарт (2000/мес) — ~3500₽/мес + овердрафт +500₽ за 100 сверху
  - Индивидуальный бот — отдельный продукт, не подписка: настройка от 50к₽ + 5-10к/мес
- **Кто платит за токены — Вариант 1:** Ирина (общий ключ ProxyAPI), включает в подписку, лимиты в тарифах
- **Воронка:** Демо → Старт → Стандарт → перерос → Индивидуальный

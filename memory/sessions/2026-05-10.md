## Что я делал плохо в течение сессии (заметки на будущее)

1. **Сначала повёл в Cloudflare** не объяснив зачем — пришлось разворачивать на 180°. Урок: прежде чем рекомендовать иностранный сервис, объяснить что мы НЕ уходим с Beget, и какие именно технические нужды это закрывает.
2. **Брал палитру из CLAUDE.md** не сверившись с `knowledge/brand/palette.md`. Урок: при создании любых визуалов сначала читать `knowledge/brand/palette.md` (это источник правды) и только потом CLAUDE.md (он может быть устаревшим). Теперь CLAUDE.md выровнен.
3. **Метания между путями** — Cloudflare → Beget API → multi-SAN HTTP-01. Урок: до прямого «делай» вызвать команду (DevOps/Infra + QA + Yuridika), решить вопрос один раз, потом не менять. Команда подключилась только когда Инна попросила прямо.
4. **Suggested checking SSL каждые 5 минут** — лишняя суета, Beget может выпускать до часа. Урок: для процессов с непредсказуемой длительностью — заводить heartbeat 1200-1800s, не 270s.
5. **Убирал demo из плана** прочитав «отложен» как «убираем». Урок: «отложен» = поддомен в архитектуре остаётся, просто конкретное действие (переезд) откладывается.

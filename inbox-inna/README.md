# inbox-inna/ — Входящие от Инны

Сюда Инна кладёт файлы вручную для обработки Директором (Claude 100).

## Как пользоваться

1. Положи MD или PDF файл в эту папку
2. Скажи Директору: **«обработай входящие»**
3. Директор прочитает, вытащит знания, обновит knowledge/
4. Обработанный файл переедет в `archive/` с датой в имени

## Что класть

- MD/PDF — конспекты, статьи, заметки, идеи
- Именуй описательно: `урок-15-бот-копирайтер.md`, `идея-квиз.md`

## Что НЕ класть

- `.env`, токены, пароли
- Файлы которые уже обработаны и лежат в knowledge/

## Что использовать дальше

Для нового продукта Ирины: `architecture-sync.md` + `product-line.md` + `brandbook/palette.md` + `tov-source.md`.

Для дизайнерской задачи: `brandbook/` целиком + `deliverables/` для финальных версий.

Для визитки/портфолио: `visitka-v3/` + `portfolio/`.

## Связь с верхней папкой

`irina/` (один уровень выше) хранит стратегический PLAN направления (партнёрство, метрики, фаза). Эта папка `irina/` — реализация: брендбук, визитка, продукты, deliverables. Не дубль, разные уровни.

# Архив — возможно к удалению

> **Назначение папки.** Сюда переезжают документы, которые перестали быть источником правды (заменены новыми мастер-документами, дубли, устаревшие черновики), но удалять их сразу нельзя — мог остаться важный контекст, на который ещё захочется сослаться.
>
> **Правило:** ничего не удаляем без явного решения Инны. Когда она проходит ревизию архива — тогда и решает: окончательно удалить или вернуть в работу.

## Что делать при ревизии

1. Открыть файл из таблицы выше
2. Открыть «чем заменён» — мастер-файл
3. Сверить: точно ли всё из старого попало в мастер
4. Если да — окончательное решение Инны: удалить файл или вернуть в работу
5. Если нет — вернуть упущенное в мастер, потом архивный файл удалить

**Удаление выполняется только Инной**, не агентами и не автоматически.

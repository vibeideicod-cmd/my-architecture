# Группа VK «Идеи для бизнеса»

---

## Решение по роли группы

**Группа = сообщество + контентный движок + воронка.**
Личная страница Инны ≠ дубль группы. Они работают в паре:

| Личная страница | Группа |
|---|---|
| Я — человек, эксперт, история | Контент, польза, кейсы, движение |
| Личное мнение, «за кулисами» | Методология, практики, идеи для бизнеса |
| Ведёт в группу через закреп | Ведёт на сайт / в продукты |

---

## Что делаем с аудиторией

**Задача 1: перегнать подписчиков личной страницы в группу**
- В VK есть функция «Пригласить друзей в группу» — до 40 приглашений в день
- В закрепе личной страницы → ссылка на группу с объяснением зачем идти туда
- Первый пост после реактивации — анонс группы

**Задача 2: реанимировать текущих 100+ участников группы**
- Закрепить приветственный пост с позиционированием
- Серия из 3–5 постов с реальной пользой (не «мы вернулись», а сразу ценность)

## Автоматизация группы

**Что можно подключить в VK-группу:**
- **Senler** — рассылки по подписчикам группы (аналог email-рассылки в VK)
- **VK Mini Apps** — мини-приложение прямо внутри группы (Beauty Визитка — уже делали)
- **Чат-бот** — автоответ на ключевые слова в комментариях/личке группы

**Приоритет подключения:**
1. Senler — чтобы собирать базу и делать рассылки (бесплатно до 150 подписчиков)
2. Бот с автоответом — квалифицировать входящие сообщения
3. Мини-апп — когда будет готов соответствующий продукт

## Воронка из группы

```
Пост в группе
  → CTA: «Получи чек-лист / запишись / напиши»
  → Личные сообщения группы / Senler-рассылка
  → Переход на сайт → AI-консультант → продукт
```

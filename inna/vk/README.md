## Логика связки

```
Личная страница Инны
  → люди видят личность, доверяют
  → закреп ведёт в группу
  → в группе: контент, движение, бот, воронка в продукты
```

Личная страница = **лицо**.
Группа = **двигатель**.

## Приоритет задач

**🔴 Срочно:**
1. Юридически почистить профиль → `vk/legal-check.md`
2. Заполнить профиль правильными текстами → `vk/personal-page-texts.md`
3. Определить роль группы и сделать первый закреп → `vk/group.md`
4. Перегнать аудиторию с личной страницы в группу

**🟡 Среднесрочно:**
5. Подключить бота/мини-апп в группу
6. Выстроить регулярный контент (делегировать SMM-Manager)
7. Запустить VK Реклама на реактивацию

Первая воронка Инны. Все материалы готовятся здесь — в публичные каналы НЕ выкладываем по частям. Когда пакет готов — деплоим одним днём.

Дата старта пакета: 2026-05-07.

## Концепция воронки

Все каналы (VK, TG, сайт, визитка, мессенджеры) ведут в **одну точку входа** — квиз «Карта точек роста твоего бизнеса» на HUB. Целевой маршрут: после прохождения контакт попадает в Мою Базу Клиентов (CRM), дальше движется по продуктовой лестнице. В первой HTML-версии CRM ещё не подключена: результат показывается на экране, follow-up идёт вручную через Telegram.

```
VK / TG / сайт / визитка
        ↓
Карта точек роста (квиз) — karta-rosta.html
        ↓
Результат на экране + ручной Telegram follow-up
        ↓
AI-диагностика (бесплатно, выдача результата квиза)
        ↓
AI-аудит (платно) → Сопровождение внедрения
```

# Моя воронка 1

Первая воронка Инны. Все материалы готовятся здесь — в публичные каналы НЕ выкладываем по частям. Когда пакет готов — деплоим одним днём.

Дата старта пакета: 2026-05-07.

## Концепция воронки

Все каналы (VK, TG, сайт, визитка, мессенджеры) ведут в **одну точку входа** — квиз «Карта точек роста твоего бизнеса» на HUB. После прохождения — контакт попадает в Мою Базу Клиентов (CRM), дальше движется по продуктовой лестнице.

```
VK / TG / сайт / визитка
        ↓
Карта точек роста (квиз) — hub.../karta-rosta
        ↓
Захват в Мою Базу Клиентов (CRM)
        ↓
AI-диагностика (бесплатно, выдача результата квиза)
        ↓
AI-аудит (платно) → Сопровождение внедрения
```

# Инна Андрейченко — Контур 1

Личный бизнес-контур Инны: бренд, продукты, каналы, HUB, визитка, воронки, AI-помощник, CRM и операционная инфраструктура.

Стратегический принцип: **живая витрина** — сначала система работает у Инны, потом зеркалится на Ирину, потом продаётся клиентам.

## Текущий запуск

Первый рабочий путь:

```
HUB → Карта точек роста → результат → контакт → ручной/CRM follow-up → предложение следующего шага
```

VK и Telegram не запускаются до готового входа на HUB.

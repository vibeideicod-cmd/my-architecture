# Архитектура Инны — Идея v3 (финальная, 09.05.2026)

> Это окончательная интегральная формулировка архитектуры. Учитывает: Research v1 (8 РФ-экспертов), Research v2 (TG-боты + VK Mini App + tripwire + email + Ледовских), Critique v2 (24+ правок), полную продуктовую лестницу из 5 уровней под каждую боль, и три критичных уточнения Инны вечером 09.05: НЕТ лицензии на обучение, НЕТ платной рекламы, тёплый круг = тестировщики и платящие одновременно.

## 1. Краткое резюме идеи

Инна Андрейченко (Архитектор автосистем и коммуникаций, ИП ОКВЭД 70.22) собирает **трёхслойную инфраструктурную экосистему** на домене `ideidlyabiznesa1913.ru`: HUB на корне Beget shared + 8 поддоменов на VPS Cheerful Marik + изолированный Vultr London VPN. Внутри HUB — 5 лендингов под 5 болей × **полная продуктовая лестница из 5 уровней под каждую боль** (вход → дёшево → основное → подписка → флагман), AI-помощник Инны на всех страницах, /catalog, /cases, /blog, /platforma, /podarok. Воронка строится на модели **«Сарафан → автоматизация»** (без платной рекламы, бюджет 0₽), точка сборки — HUB + AI-помощник, мост к продаже — бесплатная диагностика по образцу Ледовских. ЦА — эксперты в 8 нишах в состоянии «24/7, тяну на себе», 9 портретов A/B/C. Вся продуктовая линейка под ОКВЭД 70.22 (никаких «курсов / обучения / школы / уроков»). Тёплый круг (≈7000 VK + мастера МГ + ученики мастеров + Zoom-контакты + сарафан) — это одновременно тестировщики новых продуктов и часть платящих клиентов: они бесплатно дают обратную связь по новому, покупают маленькое и среднее по доверию, рекомендуют другим. Запуск — поэтапный по триггерам, темп — дни и часы через vibe coding. Принцип «Расширяем не урезаем»: tripwire-лестница, платный таргет, MAX/YouTube/Дзен — не отрезаются, переходят в очередь по триггерам.

**Юр.лексика на всех публичных текстах** (см. `inna/website.md` + `feedback_no_education_license.md`):
- Только «консультация / аудит / разбор / сопровождение / методические материалы / экспертная сессия / диагностика».
- Запрещены: курс / обучение / школа / тренинг / урок / преподаватель / ученик / академия / мини-курс / диплом.
- AI-помощник = инструмент квалификации запросов, не самостоятельный сервис.
- Технологии и vibe coding — показываем цветом и визуалом, не словами.

## 6. Воронка с учётом тёплого круга и без рекламы

Не «холодный таргет → лид-магнит → email → tripwire → mid → флагман» (модель РФ-конкурентов на готовой холодной базе). У Инны — тёплый круг (≈7000 VK личная + мастера МГ + ученики мастеров + Zoom-контакты + сарафан + ЛС VK) и НЕТ холодной email-базы и текущего контент-ритма.

### 6.2 Тёплый круг = тестировщики + платящие одновременно (уточнение Инны 09.05)

```
ТЁПЛЫЙ КРУГ (≈7000 VK + МГ + ученики мастеров + Zoom + сарафан)
   │
   ├── Тестируют новые продукты бесплатно → дают обратную связь
   │   (это полезно команде — видим, как продукт работает на живых людях)
   │
   ├── Покупают маленькое (3-15к: брендбук, визитка, аудит, квиз)
   │   и среднее (25-80к: упаковка, бот, лендинг + квиз)
   │   по доверию, без длинного прогрева
   │
   ├── Часть со временем покупает флагман (Платформа от 5к/мес или 300к+ под ключ)
   │
   └── Рекомендуют другим → сарафан → платящие холодные

ПЛАТЯЩИЕ ХОЛОДНЫЕ
   ← по сарафану от тёплого круга
   ← через органический контент VK/TG (без рекламных вставок)
   ← через SEO и блог HUB (когда блог запущен)
   ← через HUB → AI-помощник Инны квалифицирует
   ↓
   Им продаём mid (50-200к) и флагман (от 5к/мес или 300к+)
   через доверие и кейсы тёплого круга
```

Тёплый круг — это не «бесплатная массовка», это **полноценный сегмент клиентов с большим стартовым доверием**. Они и тестируют, и покупают, и рекомендуют.

### 6.3 Каналы входа

| Канал | Что от него ожидаем | Куда ведёт |
|---|---|---|
| №1 — личное общение Инны (Zoom, ЛС VK, мероприятия) | Контакт сразу попадает в `crm.*` через n8n, источник «личное» | Прямой разговор, без HUB |
| №2 — посты VK личная + VK группа + TG-канал (запуск с тёплого круга) | Точка сборки — HUB | HUB → AI-помощник |
| №3 — рекомендации (мастера МГ, ученики, существующие клиенты) | Ссылка на HUB / визитку | HUB → AI-помощник или напрямую mid-продукт |
| №4 — лид-магниты `/podarok/<имя>/` после квиза или поста | Email-цепочка 7 писем | HUB-лендинг боли |
| №5 — TG-бот @innabot и VK Mini App «AI-диагностика» | Магнит + AI-чат + запись на бесплатную диагностику | HUB или ЛС менеджеру |
| Очередь | Yandex Карты, бизнес-клубы, оффлайн-мероприятия, MAX, YouTube, Дзен, Авито | По дорожной карте Слоя 4 architecture.md v7.0 |

### 6.4 Точка сборки и продуктовая лестница

```
Канал входа №1-5
     ↓
HUB (главная или лендинг боли) ИЛИ TG-бот ИЛИ VK Mini App
     ↓
AI-помощник Инны (4-6 вопросов: ниша → боль → уточнение → этап C/B/A → предложение)
     ↓
Маршрутизация по этапу C/B/A:
   ├─ C (греется) → лид-магнит /podarok/ → email-цепочка 7 писем за 1-2 нед
   ├─ B (на этапе развития) → лендинг боли + бесплатная диагностика 30-40 мин с Инной (по образцу Ледовских aibasis.ru/consultai)
   ├─ A (готов к росту) → Платформа С.С.С.Р. ИЛИ Нейро-архитектура под ключ
   └─ Любой этап + горячий лид из TG-бота → quick-response менеджер (Инна сама на старте)
     ↓
Контакт → crm.* (Моя база клиентов) с пометкой источника, через n8n
     ↓
Дальше — кейс / рекомендация / повторная продажа
```

### 6.5 Бесплатная диагностика как мост (по образцу Ледовских)

Отдельный шаг между прогревом и продажей флагмана. Лендинг бесплатной диагностики (можно `/diagnostika/` внутри HUB или отдельная посадка через `crm.*`-форму). 30-40 минут разговора Инны с клиентом по Zoom. Цель — не продать сразу, а разобрать ситуацию и дать рекомендацию (что делать дальше, какой продукт подходит). Конверсия в платного — выше, чем у tripwire (Ледовских продаёт ~100k через эту воронку без tripwire).

### 6.6 Email-цепочка 7 писем за 1-2 недели + реактивация

Вместо «5 за 5 дней» (частит, по Research v2). Стандарт рынка — 5-8 писем за 1-2 недели, первые 3 ежедневно, потом интервалы. Контентных писем больше, чем продающих.

| День | Письмо | Тип |
|---|---|---|
| 1 (сразу) | Магнит + спасибо + 1 совет по использованию | Welcome + Доставка |
| 2-3 | Углубление магнита + лайфхак | Контент |
| 4 | Кейс конкретного клиента | Контент + Социальное доказательство |
| 5 | Soft sales (но БЕЗ tripwire-блока инфоцыганской эстетики, без зачёркнутых цен и таймеров) | Soft Sales |
| 7 | Разбор частой ошибки в нише | Контент + Работа с возражением |
| 9 | Предложение бесплатной диагностики 30-40 мин | Sales |
| 12-14 | Реактивация — предложение Платформы или нового магнита | Closing |

Сервис: на старте Unisender (запуск за 1-3 дня, 100+ шаблонов) → долгосрочно n8n + SMTP на VPS. Email-инфраструктура (DKIM/SPF/DMARC + IP-репутация + доставляемость в Mail.ru/Yandex) — Systems оценивает в Шаге 4 (Critique v2 п.4.6).

### 6.7 Tripwire — в очередь, не на старте

Решение зафиксировано: tripwire-лестница 990 / 3990 / 9990 ₽ — **не строится на старте**. Причины:
- Платной рекламы нет → холодного потока нет → tripwire не для кого.
- Ледовских продаёт ~100k без tripwire через бесплатную диагностику.
- 990 ₽ как анкер — устаревший (2020-22), для премиум-консалтинга не работает.
- ЦА Инны (эксперты «24/7») не та аудитория, у которой спор «990 vs 1490 ₽».

Tripwire остаётся в архитектуре **как опция на будущее** (правило «расширяем не урезаем»). Триггер разворачивания — появление холодного потока (платный таргет, гостевые посты, новый домен). Pricing-Specialist + Marketing-Strategist уточняют формат и цены к моменту разворачивания.

## 11. Принципы (must-keep, не урезаем)

1. **AI-помощник на корне HUB = главная фишка.** Виджетом + hero-блок на главной (>50% экрана), проактивный первый вопрос. Юр.чисто. Маршрутизация по 5 болям × 8 нишам × A/B/C.
2. **Один поддомен = одна функция.** Не дублируем (грабли Гандапаса).
3. **Свой VPS, не GetCourse / Bizon365.** Соответствие позиционированию и ОКВЭД 70.22.
4. **Поэтапный запуск экосистемы**, не одновременный. По этап-карте п.12.
5. **`mini.*` под ОКВЭД 70.22** — обязательная плашка «демо, не продукт» + noindex + лимит 5 вопросов / Email-gate.
6. **`bz.*` = один источник, два интерфейса** (read-only зеркало `knowledge/`).
7. **Vultr London VPN изолирован** от продуктовой инфры (ФЗ-152).
8. **Принцип живой витрины** — Слой 1 п.1.7. Сначала строю себе → потом предлагаю клиентам. 3 ступени обкатки параллельно.
9. **Технологии показываем цветом, не словами** — главное правило бренда.
10. **Расширяем, не урезаем** — все идеи сохраняются в архитектуре, переносятся в очередь, не отрезаются (tripwire, платный таргет, MAX/YouTube/Дзен/Авито, Подбренд Нейро Бабки, MINI-6 SaaS).
11. **Фокусные документы обновляем одновременно** с обсуждением (новое правило 09.05).
12. **Темп — дни и часы, не недели** через vibe coding и параллельную работу агентов.
13. **«Начинаем с этапа клиента»** через лестницу C/B/A — каждое предложение под этап.
14. **🔴 НЕТ лицензии на обучение — все продукты только под ОКВЭД 70.22.** Запрещены: курс / обучение / школа / тренинг / урок / преподаватель / ученик / академия / мини-курс / диплом. Применяется ко ВСЕЙ продуктовой линейке + лид-магнитам + Платформе + tripwire-форматам (когда подключим).
15. **🟢 НЕТ платной рекламы, бюджет 0 ₽.** VK Таргет / Яндекс.Директ / TG Ads / закупка у блогеров — нет. Даже посты без CTA «купи / акция / скидка». Только органика + сарафан + автоматизация прогрева.
16. **Тёплый круг = тестировщики + платящие одновременно.** Они тестируют новые продукты бесплатно (обратная связь), покупают маленькое и среднее по доверию, рекомендуют другим, часть со временем покупает флагман.

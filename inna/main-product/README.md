# Главный продукт — «Бизнес-архитектура под ключ»

Дата старта продуктового контура: 2026-05-17  
Статус: идея продукта переведена в реализацию через Client Zero  
Client Zero: Контур Инны (`inna/`)

## Что это

«Бизнес-архитектура под ключ» — флагманский продукт Инны.

Клиент приходит с хаосом, разрозненными активами, непонятной воронкой, перегрузом собственника или желанием “собрать всё под ключ”. На выходе получает:

- понятную архитектуру бизнеса;
- карту текущих активов;
- продуктовую линейку;
- маршрут клиента;
- план внедрения;
- первый рабочий вход;
- понимание, какие компоненты строить первыми.

## Почему этот контур нужен

Мы начали делать `karta-rosta.html`, потом увидели, что нужен CRM. Это правильно как симптом, но опасно как порядок: можно начать строить компоненты, не зная полного состава продукта.

Поэтому фиксируем главный продукт как отдельную идею и ведём его по шагам:

```text
Идея
→ составляющие продукта
→ Client Zero на Инне
→ план реализации
→ сборка компонентов
→ QA
→ упаковка для клиентов
```

## Ключевое решение

CRM — не отдельный стартовый проект. CRM — один из компонентов продукта. Её строим тогда, когда карта компонентов показывает, что без неё ломается первый рабочий маршрут.

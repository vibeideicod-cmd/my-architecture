# Платформа С.С.С.Р. — концепция и план

> Последнее обновление: 2026-04-30
> Связь с архитектурой: `inna/architecture.md` → Слой 5, п.5.5

---

## Что это

**Закрытая подписная платформа для сегмента A** — экспертов, которые уже ведут свой бизнес и хотят выстроить систему.

Клиент покупает подписку → получает личный кабинет → работает по методологии С.С.С.Р. с поддержкой AI и Инны. Инна не ведёт каждый кабинет вручную — система работает через автоматизации.

---

## Для кого

**Сегмент A:** эксперт, инфобизнесмен, наставник, коуч, финансовый консультант.

Фильтр:
- Уже есть продукт или услуга, есть клиенты
- Нет системы — всё держится на руках, нет автоматизаций
- Хочет вырасти без роста усилий
- Готов платить за результат, не ищет «самое дешёвое»

**Откуда приходит:**
```
Каналы (VK/TG) → сайт → AI-консультант → "ты эксперт" → лендинг Платформы → подписка
```

## Что получает клиент (4 модуля)

| Модуль | Что внутри | Боль |
|---|---|---|
| **Структура** | Аудит бизнеса, схема процессов, план внедрения | «Нет структуры, всё в голове» |
| **Стратегия** | Личная стратегия роста, позиционирование, продуктовая линейка | «Не вижу картину целиком» |
| **Свобода** | AI-инструменты, автоматизации, делегирование | «Нет жизни вне бизнеса» |
| **Результат** | Трекинг прогресса, AI-отчёты, разборы | «Работаю много — системы нет» |

---

## Модель монетизации

**3 тарифа (черновик):**

| Тариф | Что входит | Цена (ориентир) |
|---|---|---|
| **База** | Доступ к 4 модулям + материалы + сообщество | 5 000–9 000 ₽/мес |
| **Работа** | База + AI-консультант в контексте клиента + еженедельный разбор | 15 000–25 000 ₽/мес |
| **VIP** | Работа + личные сессии с Инной + приоритет | от 50 000 ₽/мес |

> ⚠️ Цены предварительные — финальные определяет Pricing-Specialist через юнит-экономику.

## Путь клиента внутри платформы

```
1. ВХОД
   Покупает подписку на сайте
   → автоматизация создаёт профиль клиента в МБК
   → отправляет приветствие + доступ в кабинет

2. ОНБОРДИНГ
   Анкета (8-10 вопросов о бизнесе)
   → AI читает ответы → знает контекст этого клиента
   → персональный план работы по 4 модулям

3. РАБОТА
   Клиент в кабинете: модули, материалы, задания
   AI-консультант знает его бизнес → отвечает в контексте
   Прогресс записывается в МБК

4. РАЗБОРЫ
   Еженедельный разбор (Zoom / запись) — Инна смотрит на весь поток
   AI готовит отчёт по прогрессу каждого клиента перед разбором

5. РЕЗУЛЬТАТ / ПРОЛОНГАЦИЯ
   AI-отчёт за период → клиент видит прогресс
   → естественная пролонгация или переход на следующий тариф
```

---

## Как Инна не ведёт каждый кабинет вручную

```
N8N — автоматизирует всё рутинное:
- онбординг (приветствие, доступы, анкета)
- напоминания о заданиях
- AI-отчёты по прогрессу
- уведомления Инне: «клиент застрял на Структуре уже 3 недели»

Инна включается точечно:
- разборы (раз в неделю на весь поток)
- VIP-сессии (индивидуально, не у всех)
- стратегические решения по платформе
```

## Технический стек (MVP)

| Компонент | Что | Где |
|---|---|---|
| **Личный кабинет** | HTML/CSS (vibe coding), уникальный токен в URL | Beget или VPS |
| **МБК (Моя база клиентов)** | Своя таблица клиентов на сервере (выглядит как Google Таблицы). Подробнее: Слой 7 архитектуры | Beget VPS (рядом с N8N) |
| **Автоматизации** | N8N | VPS (уже работает) |
| **AI в контексте клиента** | AI-консультант с подгрузкой профиля из МБК | VPS |
| **Оплата** | ЮKassa или ручное выставление счёта (MVP) | — |
| **Контент модулей** | MD → HTML страницы | Beget |

## MVP — что строим первым

**Фаза 0 (сейчас): концепция и архитектура** ← мы здесь

**Фаза 1 (MVP, vibe coding):**
- [ ] Лендинг Платформы на сайте (что это, тарифы, CTA)
- [ ] Простой личный кабинет — статичные HTML-страницы с токеном в URL
- [ ] 4 раздела по модулям с базовыми материалами
- [ ] МБК: таблица с профилями клиентов (поднимаем в Слое 7 — инфраструктура)
- [ ] N8N: онбординг-флоу (анкета → профиль → доступ)
- [ ] AI-консультант с загрузкой контекста клиента

**Фаза 2 (рост):**
- [ ] Трекинг прогресса по заданиям
- [ ] AI-отчёты за неделю и за месяц
- [ ] Автоматические напоминания
- [ ] Полноценная авторизация (логин/пароль)

**Фаза 3 (масштаб):**
- [ ] Тариф-система с автоматическим управлением доступом
- [ ] Партнёрская программа (куратор приводит клиента → получает %)
- [ ] Аналитика по платформе (сколько клиентов активны, где застревают)

## Связь с экосистемой

```
Сайт-экосистема (публичный)
  → Главная → AI-консультант → сегмент A → лендинг Платформы

Платформа (закрытая зона)
  → Личный кабинет → модули → AI в контексте → разборы

Инфраструктура (невидимая)
  → МБК (Моя база клиентов) → автоматизации (N8N) → VPS
```

---

## Связь с Mastergroup (Контур 2)

Mastergroup = **бесплатная тест-среда** для методологии.
Платформа С.С.С.Р. = **коммерческий продукт** той же методологии.

Путь: куратор проходит через МГ → видит как работает система → покупает подписку на Платформу.
Это самый органический конверсионный путь.

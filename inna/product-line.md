# Продуктовая линейка 11.05 v2 — Инна Андрейченко

> **Все продукты — через ОКВЭД 70.22:** консультация, аудит, сопровождение внедрения. **Запрещены** слова: курс, обучение, школа, тренинг, урок, преподаватель, ученик, академия, мини-курс (нет лицензии на обучение, см. auto-memory `feedback_no_education_license`).

> **Инструменты реализации:** vibe coding (Claude Code в VS Code) + Claude Design + нейросети.

> **Парный документ:** `inna/architecture-2026-05-11-v1.md` — Слой 2 «Продуктовая линейка» (5 болей × 5 уровней) + Слой 6 «Платформа С.С.С.Р.».

> **Версия:** 11.05 v2 (предыдущая 09.05 — после решения по Open Point 5.5).

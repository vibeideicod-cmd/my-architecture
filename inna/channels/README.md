## Стратегическая оценка (апрель 2026)

### Проблема: дробление без фокуса
Сейчас: 3 VK-аккаунта + 3 TG-канала + 2 TG-чата + 3 личных TG-аккаунта = **11 точек присутствия**, все полумёртвые.

Причина не в том, что каналов много — причина в том, что **каждый канал без чёткой роли** требует отдельной стратегии, а ресурса нет на все одновременно.

### Принцип активации: фокус → переток

```
Фокус сейчас (активировать):
  VK личная страница Инны  →  VK группа «Идеи для бизнеса»
  TG канал Нейропрокачка   →  TG НейроКлуб (сообщество)

Заморозить (не удалять, перенаправлять):
  TG Практикумы → ссылка на основной канал
  TG Бизнес клуб → архив или слияние с НейроКлубом
  TG Взаимопиар → оставить как есть (пассивно)

Отдельная линия (запустить после основных):
  НейроБабки (Инна + Ирина) — Фаза 2
```

### Точка входа «одна ссылка»
Единая ссылка-агрегатор → страница на сайте (`/links` или `/start`) с кнопками на все активные каналы. Это же страница для bio в VK, TG, Авито.

## ПЕРВЫЙ ШАГ ДЛЯ НОВОГО ЧАТА

При входе в задачу:
1. **Читаешь источники правды** (список выше)
2. **Подключаешь DevOps/Infra методологию** — читаешь `agents/devops-infra.md`
3. **Текущее состояние сервера проверяешь сам** через SSH (`ssh model-zarek`, GREEN-уровень аудит)
4. **Собираешь план Этапа 1** (nginx + wildcard SSL) — что конкретно ставим, какие шаги, какие риски
5. **Показываешь план Инне** — она подтверждает или корректирует
6. **Только после ✔** — выполняешь Этап 1 по методологии (YELLOW-уровень: план → ✔ → действие)

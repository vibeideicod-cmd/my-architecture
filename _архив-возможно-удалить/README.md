# Архив — возможно удалить

> Эта папка — **буфер между «есть в репо» и «удалить»**. Сюда уезжает то, что **спорно или устарело**, но не удаляется автоматически. Финальное удаление делает только Инна, отдельным заходом, осознанно.
>
> Правило: `feedback_no_deletion_archive_first` в auto-memory.

## Как пользоваться

Когда какой-то агент команды или Director считает, что файл/папку нужно удалить:

1. **НЕ удалять автоматически.** Перенести через `mv` в эту папку с понятным именем (формат `<имя>-YYYY-MM-DD/`).
2. **Добавить запись в раздел «История операций» этого README** — что, откуда, почему, как восстановить.
3. **Инна сама** периодически открывает эту папку, проверяет содержимое, и удаляет вручную через Finder если действительно не нужно.

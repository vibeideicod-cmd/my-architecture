## Утверждённые правки к карте `project_domain_map.md`

### P0 — блокеры запуска

1. **Поэтапный запуск.** Не строим 8 поддоменов одновременно. Очередь: HUB+1 лендинг → `inna.*` → `crm.*` минимум → `ai.*`+AI на HUB → `mini.*` 1 демо → `demo.*` по факту → `platforma.*` после 3-5 подписчиков → `bz.*`/`ain8n.*` внутренние под BasicAuth.
2. **`bz.*` остаётся в карте**, но как «один источник, два интерфейса» — read-only веб-морда `knowledge/` через MkDocs/Docusaurus. Не отдельная база.
3. **AI-помощник на корне HUB** — главная фишка, не прятать. Без email-gate, без лимита вопросов на старте.
4. **Лендинг подписки `/platforma`** на корне HUB **до** запуска платформы. Тизер 4 модулей + форма «ранний доступ». Закрывает дырку «нечего продавать пока платформа не построена».
5. **Fallback при недоступности AI** — форма «AI занят, оставьте контакт» вместо «загрузка».

### P1 — для конверсии и юр.чистоты

6. **`demo.*`** доступен только по прямому URL клиента, без публичной навигации с HUB (чтобы не сливался с `mini.*` в голове клиента).
7. **Цены «от X»** на главной и в каталоге — фильтр лидов. Конкретику ставит Pricing-Specialist + Инна.
8. **Переформулировать 5 болей** на живой язык: «Тащу всё на себе» вместо «Нет структуры».
9. **Кейсы по нише — на лендинге боли** (1 клик, не 2). На каждом из 5 лендингов внизу блок «Как это сработало в твоей нише».
10. **`mini.*` под 70.22 + noindex.** Плашка «Демонстрационная среда» + `<meta name="robots" content="noindex">` + `X-Robots-Tag: noindex` в nginx.
11. **Защита `crm.*`/`ain8n.*`/`bz.*`** — BasicAuth + IP-whitelist российских IP. **Без Vultr London VPN** (он изолирован от продуктовой инфры — `key-decisions.md` 09.05).

### P2 — после MVP

12. Зарезервировать `partners.*` (партнёрская программа Слой 4 → SaaS-перспектива).
13. Зарезервировать место для подбренда «Нейро Бабки» (Фаза 2): `mini.*/babki/` или будущий `babki.*`.
14. Один AI = один контекст в MVP (HUB-универсал). Multi-tenant — итеративно.
15. План миграции Cheerful Marik → Model Zarek с объявленным окном.
16. Мониторинг SSL и uptime по всем 8 поддоменам.
17. Повторный точечный Research через 1-2 месяца — не появился ли конкурент с AI на главной.

## Financial — задачи

### Текущие расходы
- Vultr London VPN: ~550 ₽/мес — изолирован, не растёт.
- Beget shared: ~200-500 ₽/мес.
- Cheerful Marik: ~600-1500 ₽/мес (требует уточнения по счетам).
- Model Zarek: ~960 ₽/мес (тариф ~32 ₽/день с публичным IPv4). **Кто оплачивает сейчас — требует уточнения у Инны.**
- **Итого сейчас: ~1500-2500 ₽/мес.**

### Что добавится
- Поддомены и SSL: 0 ₽.
- Claude API при публичном AI: **$20-500/мес = 1500-40000 ₽/мес** — узкое место.
- DeepGram при клиентском аудио: ~100-500 ₽/мес.
- Апгрейд VPS под Платформу: +1500-3000 ₽/мес.
- Backup Object Storage: ~50-200 ₽/мес.

### Митигации по AI-расходам
1. nginx rate-limit (10 req/min на IP).
2. Лимит токенов на ответ (макс 500).
3. Дневной лимит в Anthropic Console.
4. Fallback «AI занят» при превышении.

### Цель экономики
- До запуска Платформы — 2000-5000 ₽/мес.
- После 5 подписчиков — 3000-7000 ₽/мес.
- AI = воронка, окупается с первой продажи 50-200к.

## QA — Definition of Done на каждый этап

### Этап 1 — HUB корень + 1 лендинг
- index.php фикс работает
- Wildcard SSL валиден
- Главная и 1 лендинг адаптивны
- Политика и оферта в футере
- Аналитика подключена
- Lighthouse > 80

### Этап 2 — `inna.*` визитка
- QR ведёт на `/vizitka` без ошибок
- Адаптивно на мобиле
- Ссылка с VK-обложки работает

### Этап 3 — `crm.*` минимум
- BasicAuth + IP-whitelist российских IP
- Бэкап БД в Object Storage
- HTTPS only + 2FA Beget
- Логи изменений включены

### Этап 4 — `ai.*` + AI на HUB
- Маркировка AI в первом сообщении
- Чекбокс согласия ПД перед первым ответом
- nginx rate-limit (тест на 11-й запрос/мин = 429)
- Дневной лимит Anthropic Console установлен
- Fallback «AI занят» работает (тест-симуляция отключенного API)
- Контакты из AI → `crm.*` через n8n (e2e тест)

### Этап 5 — `mini.*` 1 демо
- Плашка под 70.22 на странице
- `noindex` в meta + `X-Robots-Tag` в nginx
- Лимит демо (5 вопросов / email-gate / 10 мин — одно из)
- Адаптивно на мобиле

### Чек-лист «зелёного света» перед публичным анонсом HUB
- [ ] Этапы 1-4 закрыты по DoD
- [ ] Политика + оферта + согласия на месте

- [ ] Договор поручения с Ириной подписан (если работает с ПД)
